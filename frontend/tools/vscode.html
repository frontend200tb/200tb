Удаление VS Code (не портативной версии)

  1. Удалить через Панель управления - Программы и компонентыMicrosoft Visual Studio Code (User)
  2. Удалить папкиc:\Users\1\.vscode\
    c:\Users\1\AppData\Roaming\Code\

Установка VS Code (портативной версии)

  1. Заходим на сайт code.visualstudio.com

  2. Скачиваем 64битную zip (портативную) версию VSCode-win32-x64-1.63.1.zip
  3. Распаковываем ее в каталог c:\Portable\VSCode\
  4. В этом каталоге создать папку data (здесь будет храниться пользовательский профиль настроек и расширения):c:\Portable\VSCode\data
  5. Далее, внутри data создаём 2 обязательныеc:\Portable\VSCode\data\extensions
    c:\Portable\VSCode\data\user-data    и 1 опциональную папкиc:\Portable\VSCode\data\tmpextensions — здесь будут храниться все установленные расширения;
    user-data — это, собственно, пользовательский профиль настроек;

    tmp — опциональная папка, если вы хотите, чтобы Visual Studio Code хранил все временные файлы локально, если эту папку не создавать, то все временные файлы будут храниться в %TEMP%.

  6. Вот и всё, можно запускать Code.exe и настраивать, работать. Если вы уже работали на ПК с Visual Studio Code и хотите скопировать или иммигрировать с обычной версии на портативную, то повторяем все шаги выше, и дополнительно для переноса расширений и настроек необходимо сделать следующее:data\extensions — копируем сюда всё из %USERPROFILE%\.vscode\extensions
    data\user-data — копируем сюда всё из %APPDATA%\Code

Как добавить VS Code в меню пуск

  1. Выбираем файл (клик правой кнопкой мыши) c:\Portable\VSCode\Code.exe
  2. Выбираем Закрепить на начальном экране

Как добавить программу VS Code в контекстное меню

  1. Открыть Редактор реестра, для этого выбрать Пуск - Выполнить или нажатьwin + Rи ввести команду regedit
  2. В редакторе реестра переходим к ветке HKEY_CLASSES_ROOT\*\shell
  3. Клик правой кнопкой мыши на папке shell и выберите команду Создать > Раздел
  4. Назовите его так, как должна называться команда в контекстном меню. Скажем, Открыть с помощью VS Code
  5. Клик правой кнопкой мыши на созданной папке и снова выбираем команду Создать > Раздел
  6. Новый вложенный раздел назовите command
  7. Щелкните на значке раздела command и дважды в окне справа на ключе (По умолчанию) REG_SZ
  8. В открывшемся окне введите следующую команду "c:\Portable\VSCode\Code.exe" "%1"

Как добавить иконку VS Code в контекстное меню

  1. Открыть Редактор реестра, для этого выбрать Пуск - Выполнить или нажатьwin + Rи ввести команду regedit
  2. В редакторе реестра переходим к ветке HKEY_CLASSES_ROOT\*\shell\Открыть с помощью VS Code
  3. Клик правой кнопкой мыши и выбираем команду Создать > Строковый параметр
  4. в поле Имя укажите значение Icon
  5. а в поле Значение укажите путь к exe-файлу, как показано ниже (кавычки обязательны): "c:\Portable\VSCode\Code.exe"

Расширения (extentions)

Visual extentions

  1. Material Theme (Mattia Astorino) Цветовая тема
  2. Material Icon Theme (Philipp Kief) Цветовая тема значков
  3. Ayu (teabyii) Рамки панелей
  4. Prettier - Code formatter (Esben Petersen) Выравнивание кода
  5. Bracket Pair Colorizer 2 (CoenraadS) Цветные кавычки
  6. indent-rainbow (oderwat) Цветные отступы
  7. Image preview (Kiss Tamas) Просмотр превью картинки в html коде
  8. Polacode (P & P) Делает красивые скриншоты кода
  9. Debugger for Chrome (Microsoft) Отладка javascript кода

Functional extentions

  1. Auto Rename Tag (Jun Han) При изменении открывающего тега изменяет закрывающий
  2. CSS Peek (Pranay Prakash) Переход из html в css к выбранному классу
  3. HTML CSS Support (ecmel) Дописывает название класса в html из css
  4. Live Sass Compiler (Ritwick Dey) Вместе сним уже идет Live Server
  5. Sass/Less/Stylus/Pug/Jade/Typescript/Javascript Compile Hero Pro
  6. Beautify (HookyQR)
  7. HTML Snippets (Mohamed Abusaid)

JavaScript extentions

  1. Quokka.js (Wallaby.js) Песочница js внутри редактора
  2. Babel JavaScript (Michael McDermott) Подсветка нового синтаксиса
  3. JavaScript (ES6) code snippets (charalampos karypidis)
