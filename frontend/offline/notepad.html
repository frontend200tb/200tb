Notepad++

Документация Notepad++

  1. npp-user-manual.org

Установка Notepad++

  1. Заходим на сайт notepad-plus-plus.org
  2. Скачиваем 32битную портативную версию npp.8.1.2.portable.zip
  3. Распаковываем ее в каталог c:\Portable\Notepad++\
  4. Выбираем файл (клик правой кнопкой мыши) c:\Portable\Notepad++\notepad++.exe
  5. Выбираем Закрепить на начальном экране

Как добавить программу Notepad++ в контекстное меню

  1. Открыть Редактор реестра, для этого нажать Пуск - Выполнить и ввести команду regedit
  2. В редакторе реестра переходим к ветке HKEY_CLASSES_ROOT\*\shell
  3. Клик правой кнопкой мыши на папке shell и выберите команду Создать > Раздел
  4. Назовите его так, как должна называться команда в контекстном меню. Скажем, Открыть с помощью Notepad++
  5. Клик правой кнопкой мыши на созданной папке и снова выбираем команду Создать > Раздел
  6. Новый вложенный раздел назовите command
  7. Щелкните на значке раздела command и дважды в окне справа на ключе (По умолчанию) REG_SZ
  8. В открывшемся окне введите следующую команду "c:\Portable\Notepad++\notepad++.exe" "%1"

Как добавить иконку Notepad++ в контекстное меню

  1. Открыть Редактор реестра, для этого нажать Пуск - Выполнить и ввести команду regedit
  2. В редакторе реестра переходим к ветке HKEY_CLASSES_ROOT\*\shell\Открыть с помощью Notepad++
  3. Клик правой кнопкой мыши и выбираем команду Создать > Строковый параметр
  4. в поле Имя укажите значение Icon
  5. а в поле Значение укажите путь к exe-файлу, как показано ниже (кавычки обязательны): "c:\Portable\Notepad++\notepad++.exe"

Установка плагина Python Script

Установить pythonscript и emmet можно только на 32битную версию notepad++
Сначала надо установить Python Script. После него установать Emmet

  1. Заходим на сайт sourceforge.net/projects/npppythonscript
  2. Скачиваем установщик PythonScript_1.0.8.0.msi
  3. Запускаем его и он установит в папку c:\Program Files (x86)\Notepad++\ файлы:c:\Program Files (x86)\Notepad++\plugins\PythonScript\
    c:\Program Files (x86)\Notepad++\plugins\PythonScript.dll
    c:\Program Files (x86)\Notepad++\python27.dll
  4. Запускаем notepad++ от имени Администратора
  5. Выбираем меню Опции - Импортировать - Импортировать плагины
  6. В открывшемся окне выбираем файл c:\Program Files (x86)\Notepad++\plugins\PythonScript.dll
  7. После успешного импортирования перезапускаем notepad++
  8. Смотрим меню Плагины и видим Python Script

Установка плагина Emmet

  1. Заходим на сайт emmet.io
  2. Выбираем плагин Emmet для notepad++
  3. Переходим на сайт https://github.com/emmetio/ и скачиваем архив emmet-npp.zip
  4. Распаковываем архив emmet-npp.zip в папку c:\Portable\Notepad++\plugins\ появится папка и в ней файл c:\Portable\Notepad++\plugins\EmmetNPP\
    c:\Portable\Notepad++\plugins\EmmetNPP\EmmetNPP.dll
  5. Запускаем notepad++ от имени Администратора
  6. Выбираем меню Опции - Импортировать - Импортировать плагины
  7. В открывшемся окне выбираем файл c:\Portable\Notepad++\plugins\EmmetNPP\EmmetNPP.dll
  8. После успешного импортирования перезапускаем notepad++
  9. Смотрим меню Плагины и видим Emmet

Полезные плагины для Notepad++ которые надо сразу установить

  1. Python Script (нужен для Emmet)
  2. Emmet (ускоряет ввод кода)
  3. Explorer показывает слева файлы и папки
  4. Auto Indent отступы в коде
  5. MultiClipboard копирует в много буферов обмена
  6. NppFTP
  7. Compare сравнивает два текста
  8. Snippets
  9. TextFX Characters подставляет закрывающий тег
  10. Webedit

Настройка Notepad++

  1. Цветовая схема Опции - Определение стилей - Выбрать стиль - Plastic Code Wrap
  2. Вид - Отображение символов - Отображать пробелы и табуляции
  3. Вид - Отображение символов - Отображать направляющие отступов
  4. меню Запуск - Запуск чтобы запустить файл из нотепада надо написать путь к программе и "$(FULL_CURRENT_PATH)"
  5. меню Опции - Настройки - Правка - Мультиредактирование ctrl + shift + t открыть последнюю закрытую вкладку
