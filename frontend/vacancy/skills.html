Требования в вакансиях

Вакансия HTML верстальщик, JavaScript разработчик

Обязанности

  1. Вёрстка веб-страниц, лендингов, корпоративных сайтов, интернет-магазинов, рассылок
  2. Вёрстка по дизайн-макетам Photoshop, Figma
  3. Поддержка вёрстки существующих крупных проектов
  4. Проведение проекта с нуля до продакшена
  5. Разработка клиентских приложений на основе JavaScript
  6. Участие в code-review внутри команды и вне её
  7. Написание документации по коду

Требования

  1. Вёрстка
    адаптивная, кроссбраузерная, отзывчивая (резиновая, responsive), pixel perfect
  2. Верстка по дизайн-макетам
    Figma, Photoshop
  3. Верстка по методологии БЭМ
  4. Чистая вёрстка, без библиотек и фреймворков
  5. Знание
    HTML5, CSS3, JavaScript, TypeScript
  6. CSS препроцессоры
    (SASS, SCSS, LESS, Stylus, PostCSS)
  7. Знание CSS фреймворков
    Bootstrap, Foundation
  8. Знание JS фреймворков
    Angular, React, Vue.js
  9. Система контроля версий GIT
  10. Работа с инструментами сборки
    менеджеры пакетов (NPM, YARN), исполнители задач (Grunt, Gulp), сборщики модулей (Browseryfy, Webpack, Parcel)
  11. Работа с CMS системами
    Wordpress, 1C Bitrix, Opencart
  12. Уметь покрывать код тестами (unit tests, integration tests)
  13. Способность читать и понимать чужой код
  14. Понимание
    принципов работы SPA, основ UI/UX
  15. Умение работать с Rest API
  16. Понимание принципов ООП, MVC и других JS-паттернов
  17. Знание английского языка на уровне Intermediate

Soft Skills

  1. Активная жизненная позиция
  2. Способность быстро обучаться новому
  3. Желание учиться и развиваться в IT
  4. Неконфликтность
  5. Ответственность
  6. Аналитический склад ума
  7. Стрессоустойчивость
  8. Грамотная устная и письменная речь
  9. Самостоятельность
