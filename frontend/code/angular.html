Angular

https://angular.io/
https://metanit.com/web/angular2/

Для установки Angular нужно ввести команду

npm i -g @angular/cli

Перейдем в папку, где создадим новый проект командой

ng new nameOfProject

Зайдем в папку созданного проекта и запустим сервер командой

ng serve

Перейдем в браузере на localhost:4200 и увидим шаблон нашего нового приложения

Или можно запустить сервер и он сам откроется в браузере командой

ng serve -o , сокращение от ng serve --open

Откроем наш проект в VS Code. В папке src в файле index.html находится только один элемент , в котором уже будет находиться все наше SPA (одностраничное приложение) работающее без перезагрузки браузера

В файле style.css можно писать глобальные стили которые применятся ко всему нашему сайту

Создадим новый компонент для нашего сайта

ng g component components/car
