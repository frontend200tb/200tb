Задача A. Плюс или XOR

Ограничение по времени 1с
Ограничение по памяти 256мб

Вам даны два целых неотрицательных числа a,b. Вы можете применять два типа операций к a любое количество раз и в любом порядке:

  • a←a+1. Стоимость этой операции составляет x;
  • a←a⊕1, где ⊕ обозначает побитовую операцию XOR. Стоимость этой операции составляет y.

В результате применения операций вам нужно получить a = b. Если это возможно, выведите минимальную стоимость; в противном случае сообщите о невозможности.

Входные данные

Каждый тест состоит из нескольких наборов входных данных. В первой строке находится одно целое число t (1 ≤ t ≤ 10⁴) — количество наборов входных данных. Далее следует описание наборов входных данных.

Единственная строка каждого набора входных данных содержит четыре целых числа a, b, x, y (1 ≤ a, b ≤ 100, 1 ≤ x, y ≤ 10⁷) — два данных вам числа и стоимости двух типов операций.

Выходные данные

Для каждого набора входных данных выведите целое число — минимальную стоимость, чтобы сделать a = b, или -1, если это невозможно.

Если a = b, то операции не нужны и стоимость будет равна 0.

Если a > b , то решение невозможно, кроме случая, когда a = b + 1 и b четное, в этом случае решение возможно уменьшением a на 1 операцией (a XOR 1) стоимостью y.

Если b > a и y >= x, то можно применить b - a операций (a + 1) стоимостью x.

Если b > a и x > y, то четные числа между a и b надо увеличивать на 1 более дешевой операцией (a XOR 1), а нечетные операцией (a + 1). Осталось только посчитать количество четных и нечетных чисел от a до b.

#include <iostream>

using namespace std;

int main() {
  // число тестов
  int t;
  cin >> t;

  while (t--) {
    // ввод данных
    int a; // исходное число
    int b; // конечное число
    int x; // стоимость операции a + 1
    int y; // стоимость опекрации a xor 1
    cin >> a >> b >> x >> y;

    // решение
    int res = 0;

    // решение невозможно
    // если a % 2 = 0 и a > b
    // или если a > 2 и a % 2 = 1 и a > b + 1
    if (a % 2 == 0 && a > b) {
      res = -1;
    } else if (a % 2 == 1 && a > b + 1) {
      res = -1;

    } else if (a % 2 == 1 && a == b + 1) {
      // решение возможно уменьшением a на 1
      // операцией y (a xor 1)
      res = y;
    } else if (y >= x) {
      // то делаем только операцию x (a+1)
      res = (b - a) * x;
    } else if (x > y) {
      // к нечетным числам делаем операцию x (a + 1)
      // и чтобы уменьшить стоимость
      // к четным числам делаем операцию y (a xor 1)
      int dif = b - a;
      if ( dif % 2 == 0) {
        res = (dif / 2) * x + (dif / 2) * y;
      } else if (dif % 2 == 1) {
        if (a % 2 == 0) {
          res = (dif / 2) * x + ((dif + 1) / 2) * y;
        } else if (a % 2 == 1) {
          res = ((dif + 1) / 2) * x + (dif / 2) * y;
        }
      }
    }

    // вывод результата
    cout << res << '\n';
  }
}
