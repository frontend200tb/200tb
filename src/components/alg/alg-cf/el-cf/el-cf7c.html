Ограничение по времени 1с
Ограничение по памяти 256мб

Своим уравнением Ax + By + C = 0 задана прямая на плоскости. Требуется найти любую принадлежащую этой прямой точку, координаты которой — целые числа от -5·10¹⁸ до 5·10¹⁸ включительно, или выяснить что таких точек нет.

Входные данные

В первой строке содержатся три целых числа A, B и C (-2·10⁹ ≤ A, B, C ≤ 2·10⁹) — соответствующие коэффициенты уравнения прямой. Гарантируется, что A² + B² > 0.

Выходные данные

Если искомая точка существует, выведите ее координаты, иначе выведите единственное число -1.

Примеры

Пример 1

2 5 3 6 -3
