Задача A. Сбор урожая

Ограничение по времени 1с
Ограничение по памяти 256мб

Игорь и Ира решили поехать на дачу, ведь там куча дел: прополоть и полить грядки, собрать урожай, окучить картошку, проверить сохранность забора и много-много всего.

На огороде у Игоря и Иры растет множество разных культур — n штук. Пришло время уборки урожая. Игорь будет хранить урожай в одинаковых ящиках. Игорь посчитал, сколько получилось собрать килограмм каждой культуры - aᵢ. Кроме того в один ящик помещается bᵢ килограмм каждой культуры.

Игорь не хочет складывать в один ящик разные овощи, ведь так не получится их сохранить за зиму. Помогите Игорю посчитать, сколько потребуется ящиков, чтобы сохранить весь урожай.

В каждой следующей строке записано два целых числа aᵢ и bᵢ (1 ≤ aᵢ, bᵢ ≤ 10⁵) — количество урожая в килограммах i культуры и то, сколько килограммов можно хранить в одном ящике.

#include <iostream>
#include <vector>
#include <math.h> // ceil

using namespace std;

int main() {
  // ввод данных
  int n;     // число культур
  cin >> n;
  vector<pair<double, int> > A(n);
  for (int i = 0; i < n; i++) {
    cin >> A[i].first >> A[i].second;
  }

  // решение
  int res = 0;

  for (int i = 0; i < n; i++) {
    res += ceil(A[i].first / A[i].second);
  }

  // вывод результата
  cout << res;

  return 0;
}

Тестировщик Кеша работает в IT-компании "Числа". Сегодня Кеша тестировал программу случайные числа, программа работает не очень быстро. Она выбирает сначала какое-то случайное число n (2 ≤ n ≤ 10⁵) делает перестановку из n чисел и выводит ее на экран.

Решение

Сумму чисел от 1 до n можно быстро найти по формуле Гауса

sum = n * (n + 1) / 2; 
#include <iostream>
#include <vector>

using namespace std;

int main() {
  // ввод данных
  vector<int> A;
  int tmp;
  while (cin >> tmp) {
    A.push_back(tmp);
  }

  // решение
  int max_num = A.size() + 1;
  long long sum_num = 0;

  for (int i = 0; i < A.size(); i++) {
    sum_num += A[i];
  }

  long long sum = (long long)max_num * (max_num + 1) / 2;
  int res = sum - sum_num;

  // вывод результата
  cout << res;

  return 0;
}

Задача C. Сольмир

Ограничение по времени 1с
Ограничение по памяти 256мб

В мире «Героев Меча и Магии 3» есть огромное количество интересных персонажей: Лорд Хаарт, Уфретин, Шакти, Ксерон, Сандро, Луна, Тазар, Крэг Хак, Кассиопея. Эта история будет посвящена могущественному джинну Сольмиру.

Сольмир – настоящий джинн и специализируется на «цепных молниях». Заклинание «цепная молния» – одно из сильнейших атакующих заклинаний, оно наносит неимоверный урон x по отряду противника и бьет в следующий отряд с уроном ⌊x/2⌋ вдвое меньше с округлением в меньшую сторону. «Цепная молния» не может бить дважды по одному отряду, поэтому после каждого пораженного отряда молния ударяет в новый непораженный отряд. Сольмир настоящий специалист по этому заклинанию и способен нанести по первому отряду противника урон равный z, а также Сольмир может управлять этой «цепной молнией», выбирая, в какой последовательности молния должна ударять по отрядам.

Разведчики Сольмира узнали, что Крэг Хак планирует на него напасть на 121 и отвоевать его капитолий. Крэг ведет за собой n отрядов, в каждом из которых суммарно aᵢ количество очков здоровья. Помогите Сольмиру определить, сможет ли он за одно заклинание уничтожить все отряды Крэг Хака.

Входные данные

В первой строке задано два целых числа n, z (1 ≤ n ≤ 100,1 ≤ z ≤ 10¹⁸) — количество отрядов в армии Крэг Хака и урон который нанесет «цепная молния» по первому отряду.

Во второй строке записано n целых чисел aᵢ (1 ≤ aᵢ ≤ 10¹⁸,1 ≤ i ≤ n) — количество очков здоровья в i-м отряде Крэг Хака.

Выходные данные

Выведите «YES», если Сольмир сможет уничтожить все отряды Крэг Хака за одно заклинание «цепная молния», иначе выведите «NO».

Задача D. Кубики

Ограничение по времени 1с
Ограничение по памяти 256мб

Игорь и Ира позвали к себе в гости Сашу и Лешу, чтобы поиграть в настольные игры. В прошлый раз Игорь проиграл в игре «Brass: Бирмингем», теперь Игорь серьезно настроен победить в игре «Покорение марса».

В начале игры игрокам раздается множество карточек, монет, ресурсов и кубиков действий. Игорь убежден в том, что для того чтобы победить в игре необходимо использовать все n кубиков действий и выложить их в виде прямоугольника a*b. Это безусловно принесет удачу в игре.

Помогите Игорю определить, сколько существует таких прямоугольников.

Входные данные

В единственной строке содержится целое число n (1 ≤ n ≤ 10¹²) — количество кубиков действий.

Выходные данные

Примеры

Пример 1

7 2

Пример 2

50 6

Пример 3

Пример 4

Решение 
#include <iostream>
#include <cmath>

using namespace std;

int main() {
  // ввод данных
  long long n; // число кубиков
  cin >> n;

  // решение
  long long res = 0;
  long long sq = sqrt(n);

  // находим целочисленные делители
  for (long long i = 1; i < sq; i++) {
    if (n % i == 0) {
      res += 2;
    }
  }

  // если n это квадрат числа
  if (sq == sqrt(n)) {
    res++;
  }

  // вывод результата
  cout << res;
}

Задача E. Поступление в аспирантуру

Ограничение по времени 1с
Ограничение по памяти 256мб

Игорь закончил магистратуру и решил подать документы в аспирантуру в свой любимый вуз, чтобы стать классным молодым учёным.

Всего в вузе есть n бюджетных мест в аспирантуре. Вуз отбирает только лучших из лучших студентов, учитывая их достижениям за время обучения. При подаче документов учитываются все грамоты, публикации и гранты студента. Если в результате учета достижений, у студентов окажется равное количество баллов, тогда этим студентам придется написать вступительные экзамены, чтобы поступить на бюджет.

Эта новость очень обрадовала Игоря, ведь у Игоря было накоплено много грамот, он участвовал в олимпиадах и писал небольшие статьи в сборники. Собрав все свои достижения и посчитав свои баллы (их получилось k), Игорь подал документы. Теперь он ждет не дождется рейтинговых таблиц.

#include <iostream>

  if (k == proh) {
    res = "entrance exams";
  }
  if (k < proh) {
    res = "no chance";
  }

  // вывод результата
  cout << res;
}

Задача F. Двери

Ограничение по времени 1с
Ограничение по памяти 256мб

Игорь и Ира решили поменять двери в квартире. Сделав все необходимые замеры ширины дверных проемов и записав их на бумажке, они поехали в магазин. По пути они обсудили, что самое важное, чтобы двери были одного цвета, иначе будет не гармонично смотреться.

В магазине перед Игорем и Ирой положили огромный каталог со всеми дверьми, имеющимися в наличии. Помогите Игорю и Ире определить, какие цвета они могут использовать.

Входные данные

В первой строке входных данных натуральное число n (1 ≤ n ≤ 10⁵) — количество дверей в квартире Игоря и Иры.

Во второй строке входных данных задана последовательность из n целых чисел aᵢ (1 ≤ aᵢ ≤ 10⁹). Ширина дверного проема с номером i равна aᵢ.

В третьей строке входных данных указано натуральное число m (1 ≤ m ≤ 10⁵) — количество цветов дверей, доступных в каталоге.

Каждая из следующих m строк описывает наличие дверей цвета с номером j. Первое число в ней kj (1 ≤ j ≤ m) указывает общее количество дверей этого цвета, далее через пробел записаны kj целых чисел — размеры дверей bᵢ (1 ≤ bᵢ ≤ 10⁹, 1 ≤ i ≤ kj). Гарантируется что количество всех дверей на складе не превосходит 10⁵.

Выходные данные

Выведите в порядке возрастания номера цветов, которые могут выбрать Игорь и Ира. Если подходящих цветов нет, то нужно вывести «-1» (без кавычек).

Решение 
#include <iostream>
#include <vector>
#include <algorithm>
#include <map>

using namespace std;

int main() {
  // ускорение ввода-вывода
  ios::sync_with_stdio(0);
  cin.tie(0);
  cout.tie(0);

  // условная компиляция
#ifdef _DEBUG
  freopen("input.txt", "rt", stdin);
  freopen("output.txt", "wt", stdout);
#endif

  // ввод данных
  int n; // число дверей
  cin >> n;
  vector<int> A(n+1);
  for (int i = 1; i <= n; i++) {
    cin >> A[i];
    // i-ая дверь имеет ширину A[i]
  }
  int m; // число цветов
  cin >> m;
  vector<vector<int> > B(m + 1);
  for (int color = 1; color <= m; color++) {
    int len;
    cin >> len;
    for (int i = 0; i < len; i++) {
      int w;
      cin >> w;
      B[color].push_back(w);
      // B[color] массив ширины дверей цветом color
    }
  }

  // решение
  // соберем частотный словарь ширины дверей которые нужны
  map<int, int> Freq;
  for (int i = 1; i <= n; i++) {
    Freq[A[i]]++;
    // A[i] - ширина двери, Freq[A[i]] - их количество
  }
  vector<int> Res;
  for (int color = 1; color <= m; color++) {

    // для каждого цвета двери соберем частостный словарь ширины дверей
    map<int, int> FreqColor;
    for (int i = 0; i < B[color].size(); i++) {
      FreqColor[B[color][i]]++;
    // B[i] - ширина двери, FreqColor[B[i]] - их количество
    }

    // если есть все двери этого цвета, то цвет добавляем в результат
    bool flag = true;
    for (pair<int, int> wid: Freq) {
      if (wid.second > FreqColor[wid.first]) {
        flag = false;
      }
    }
    if (flag) {
      Res.push_back(color);
    }
  }

  // вывод результата
  if (Res.size() == 0) {
    cout << -1;
  } else {
    for (int i = 0; i < Res.size(); i++) {
      cout << Res[i] << ' ';
    }
  }
}

    cin >> tmp;
    B.insert(tmp);
    Name.insert(tmp);
  }

  set<string> C;
  for (int i = 0; i < m1; i++) {
    cin >> tmp;
    C.insert(tmp);
  }

  set<string> D;
  for (int i = 0; i < m2; i++) {
    cin >> tmp;
    D.insert(tmp);
  }

  // решение
  // выберем все имена которые есть в обоих списках A и B
  // и которых нет ни в одном списке C или D
  set<string> Res;
  for (auto name : Name) {
    if (A.find(name) != A.end() &&
      B.find(name) != B.end() &&
      C.find(name) == C.end() &&
      D.find(name) == D.end()) {
      Res.insert(name);
    }
  }

  // вывод результата
  if (Res.size() == 0) {
    cout << 0;
  } else {
    for (auto name: Res) {
      cout << name << '\n';
    }
  }
}
