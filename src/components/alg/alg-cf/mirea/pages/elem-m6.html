Задача A. Сложение

Ограничение по времени 1с
Ограничение по памяти 256мб

Сережа пришел на олимпиаду и с ужасом обнаружил, что во всех задачах, где ответ слишком большой и не вмещается в стандартные типы данных, требуется найти ответ по простому модулю, используя арифметику остатков.

Помогите Сереже реализовать функцию сложения двух чисел по простому модулю.

Входные данные

В первой строке записаны 2 целых числа a и b (0 ≤ a, b < 10⁹+7)

Выходные данные

Выведите остаток от деления суммы s (0 ≤ s < 10⁹+7) чисел a и b по простому модулю 10⁹+7.

Примеры

Пример 1

2 5 7

Пример 2

1 1000000006 0

Пример 3

1000000006 1000000006 1000000005

Убедитесь, что Вы всегда выводите неотрицательное число.

Решение 
#include <iostream>

using namespace std;

int main() {
  // ввод данных
  long long a, b; // два числа
  cin >> a >> b;

  // решение
  long long mod = 1e9 + 7;
  // необходимо вывести сумму по модулю mod
  long long res = (a + b) % mod;

  // вывод результата
  cout << res;
}

Выходные данные

Примеры

Пример 1

Пример 2

2 5 1000000004

Пример 3

1000000006 1 1000000005

Пример 4

1 1000000006 2

Пример 5

1000000006 1000000006 0

Убедитесь, что Вы всегда выводите неотрицательное число.

Решение 
#include <iostream>

using namespace std;

int main() {
  // ввод данных
  long long a, b; // два числа
  cin >> a >> b;

  // решение
  long long mod = 1e9 + 7;
  // необходимо вывести сумму по модулю mod

  // вывод результата
