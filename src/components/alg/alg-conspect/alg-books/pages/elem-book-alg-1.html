Урок 1 Простейшая программа на языке Pascal (19)

Программа на языке Pascal (тело программы) должна начинаться со слова begin, а заканчиваться словом end и знаком точки. Простейшая программа больше ничего не содержит и ничего не делает.

begin
end.

Для того чтобы программа вывела на экран фразу Hello World, добавим оператор вывода write

begin
  write('Hello World')
end.

Заметим, что перед словом end точка с запятой не ставится, потому что она нужна для разделения операторов между собой. Эту программу можно записать и в одну строку, если заменить символы перевода строки на пробелы.

begin write('Hello World') end.

Чтобы окно консоли сразу не закрывалось, нужно добавить в конце программы оператор readln

begin
  write('Hello World');
  readln
end.

Вывод данных

Для вывода информации служат стандартные процедуры write() и writeln(). В скобках через запятую перечисляются параметры, которые нужно вывести. Число параметров не ограничено. В качестве параметров могут быть константы, переменные, выражения.

Единственное отличие между процедурами write и writeln: после вызова процедуры write курсор остается на той же строке, после вызова процедуры writeln курсор переходит на новую строку.

Строковая константа пишется в одинарных кавычках. Если нужно напечатать кавычку, то нужно написать две кавычки подряд

write('Hello ', 'World ', 'O''key') Hello World O'key

Числовые константы пишутся без кавычек. При печати параметров, пробелы между ними автоматически не вставляются

write(1, 2, 3) 123

Чтобы они разделялись пробелом можно добавить символы пробела или отформатировать печать.

write(1, ' ', 2, ' ', 3) 1 2 3

Для форматирования печати после каждого элемента списка вывода ставится двоеточие и целое число, которое указывает, сколько позиций должен занимать элемент. Если элемент меньше этого числа то добавляются пробелы слева до необходимого значения. Если больше то элементв выводится полностью и пробелы не добавляются

write(1:2, 2:3, 3:3, 77777:3) 1 2 377777

При форматировании вывода вещественных чисел, после второго двоеточия указывается число цифр после десятичной точки. При этом вещественное число округляется по правилам математики.

write(pi:0:0, pi:5:2, pi:8:5, pi/2:2:0) 3 3.14 3.14159 2

Если в качестве параметра записано выражение, то выведется результат его вычисления.

write(7 + 5) 12

Урок 2 Целые и вещественные числовые типы данных (22)

Целые числа

Тип integer в Borland Pascal от -32768 до 32767

Тип longint в Borland Pascal от -2147483648 до 2147483647

Тип integer в Delphi от -2147483648 до 2147483647

Арифметические операции с целыми числами : +, -, *, div, mod.

div - деление нацело, дробная чать просто отбрасывается

mod - остаток от деления нацело

a mod b = a - ((a div b) * b)

Вещественные чила

Тип real

Функции для работы с числами

abs(x) sqr(x) sqrt(x) pi sin(x) cos(x) arctan(x) exp(x) ln(x) round(x) trunc(x) int(x) frac(x)

Ввод данных

Для ввода информации служат стандартные процедуры read() и readln(). В скобках через запятую перечисляются имена переменных, которые нужно ввести. Число переменных не ограничено.

Выполнение программы будет приостановлено, до тех пор пока пользователь не введет соответствующее количество значений. Чтобы ввести несколько переменных, они должны отделяться пробелом или переводом строки. После ввода последнего значения нужно нажать Enter, чтобы все значения записались в переменные.

Единственное отличие между процедурами read и readln: после вызова процедуры read курсор остается на той же строке, после вызова процедуры readln курсор переходит на новую строку.

После нажатия клавиши Enter курсор в любом случае переходит на новую строку, поэтому разницу между операторами read и readln заметить невозможно.

Урок 3 Оператор присваивания (26)

Пример 1. Пример программы

var a, b, c: real;
    i, j, k: integer;
begin
  a := 1;
  b := 2.0;
  write('Enter i = ');
  readln(i);
  j := sqr(i);
  i := i + 1;
  c := sin(a + b);
  writeln('a = ', a:2:2, ' b = ', b:2:2, ' c = ', c:2:2);
  writeln('i = ', i, ' j = ', j:2);
  readln
end.

Пример 2. Способ выделения цифр из трехзначного числа в переменной a

var a, a1, a2, a3: integer;
begin
  a := 765;
  a1 := a div 100;
  a2 := a div 10 mod 10;
  a3 := a mod 10;
  writeln('a = ', a);
  writeln('a1 = ', a1, ' a2 = ', a2, ' a3 = ', a3);
  readln
end.
