Занятие 7. Условный оператор (2023-10-08)

Как проверить что введенное число n равно 0?

int res = 1 / (abs(n) + 1);

Для n = 0, целочисленное деление 1 / 1 = 1. Для всех остальных чисел знаменатель будет больше 1. Значит результат целочисленного деления 1 на число больше 1 равен 0.

В этом идея математического решения задач, без использования логических операторов.

Задача 907 Мышка acmp.ru

Решение

Если радиус круглого коврика r * 2 меньше чем высота и ширина прямоугольного коврика, то круглый коврик можно спрятать под прямоугольный.

#include <iostream>

using namespace std;

int main() {
  int w, h, r;
  cin >> w >> h >> r;
  if (w >= r * 2 && h >= r * 2) {
    cout << "YES";
  } else {
    cout << "NO";
  }
}

Задача 539 Торт acmp.ru

Решение

Если требуется один кусочек, то это весь торт и его резать не надо. Если нужно четное число кусочков, то режем торт по диаметру и с один надрез дает два кусочка. Если нужно нечетное число кусочков, то режем по радиусу и один надрез дает один кусочек.

#include <iostream>

using namespace std;

int main() {
  int n;
  cin >> n;
  if (n == 1) {
    cout << 0;
  } else if(n % 2 == 0) {
    cout << n / 2;
  } else {
    cout << n;
  }
}

Задача 16 из 17. Контрольная работа

Решение 
#include <iostream>

using namespace std;

int main() {
  int a, b, c, d;
  cin >> a >> b >> c >> d;
  double sr = 1.0 * (a * 5 + b * 4 + c * 3 + d * 2) / (a + b + c + d);
  int res = 0;
  if (sr < 5) {
    res += a;
  }
  if (sr < 4) {
    res += b;
  }
  if (sr < 3) {
    res += c;
  }

  cout << res;

  return 0;
}
