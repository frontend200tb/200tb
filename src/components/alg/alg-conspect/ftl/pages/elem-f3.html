Занятие 3. Целочисленное деление

Занятие 3. Целочисленное деление (2023-09-24)

Операция деления / отличается для целых и вещественных чисел. С целыми числама она дает целое число, а остаток отбрасывает. С вещественными числами, то есть если одно из чисел вещественное, она дает вещественный результат от деления.

По возможности, все операции нужно проводить с целыми числами. Например, если надо работать с ценами, которые в рублях и копейках, то надо сначала все перевести в копейки и работать уже с целыми числами. Если расстояния в километрах и метрах, то надо сначала все перевести в метры и работать с целыми числами.

Почему работать с целыми числами лучше чем с действительными? на 64-разрядной архитектуре процессора, процессор за один такт работает с 64 рязрядами. Бесконечные дроби невозможно записать в память, так как бесконечной памяти не бывает.

Поменять числа переменные a и b местами можно используя переменную tmp. После изменения переменных нам переменная tmp не нужна, а она остается и занимает память. Будем использовать фигурные скобки, внутри которых создается блочная область видимости. Переменная tmp создается только в области видимости в фигурных скобках и уничтожается при выходе из них.

{
  int tmp = a;
  a = b;
  b = tmp
}

Задача разделить два целых числа

Есть два целых числа a и b. Найти результат целочисленного деления и остаток.

#include <iostream>

using namespace std;

int main() {
  int a, b;
  cin >> a >> b;

  cout << a << '/'' << b
       << '=' << a / b << '\n';
  cout << a << '%'' << b
       << '=' << a % b << '\n';
}

Если два числа положительные

При положительных a и b эта программа даст правильный ответ.

5/3=1
5%3=2

Если делимое число отрицательное

Если a отрицательное, то эта программа выдает неправильный ответ.

-5/3=-1
-5%3=-2

Это связазно с операцией деления реализованной в самых первых процессорах, которая не совпадала с математическим определением целочисленного деления. В следующих процессорах логику оставили такой же для совместимости уже написанных программ.

В языке Python операции деления и остаток от деления реализованы правильно, независимо от того как это выполняется в процессоре. Python сам исправляет эту ошибку и о ней думать не надо. На C++ и других языках, нужно самому писать реализацию правильного деления и правильного остатка от деления.

Математическое деление

Разделить число a на число b это значит найти такие остаток r и частное q, что

a = b * q + r 0 <= r < b

При положительных числах 49 / 14. Здесь a = 49, b = 14. Подставим в формулу деления

49 = 14 * q + r q = 3, r = 7

При отрицательном делимом -5 / 3. Программа посчитает так

-5 = 3 * q + r -5 = 3 * (-1) - 2

Но в математике r не может быть меньше нуля.

Правильная программа должна выдавать q = -2, r = 1

-5 = 3 * (-2) + 1
