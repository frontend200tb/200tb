Здесь мы будем писать первые программы на языке C++. Причем первая программа за нас уже написана на этом сайте

#include <iostream>

using namespace std;

int main() {
  cout << "Hello World";

  return 0;
}

Компьютер не понимает язык C++. Компилятор переводит программу с языка C++ на машинный код. Компилятор запускается кнопкой Run.

В олимпиадном программировании принято один раз писать

using namespace std;

В промышленном программировании принято каждый раз писать

std::cin... std::cout...

В каждой программе мы будем подключать библиотеку ввода-вывода в консоль

#include <iostream>

Почему эта библиотека не встроена в язык C++ по умолчанию? Потому что не каждая программа использует потоки ввода-вывода в консоль. Эту библиотеку можно не подключать если она не нужна

int main() {
  return 0;
}

Объявление переменной. Выделяется память для хранения значения типа int

int a;

Инициализация переменной. В выделенную память записывается значение типа int

a = 1;

Объявление с инициализацией. Выделяется память и сразу в нее записывается значение

int a = 1;

Что будет в переменной до ее инициализации зависит от компилятора. Компилятор может записать туда 0, а может и оставить мусор - те данные что остались от других программ.

Дома можно получать 0, а на олимпиаде - мусор из-за того что используется разная среда программирования и разные компиляторы. Нужно самому всегда инициализировать переменные.

Ввод данных с консоли состоит из двух частей cin это поток, а >> это направление потока

cin >>

Вывод данных на консоль состоит из двух частей cout это поток, а << это направление потока

cout <<
