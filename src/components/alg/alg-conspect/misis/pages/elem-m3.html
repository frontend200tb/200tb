Занятие 3. Целочисленная арифметика. Операция деления

Задача A. Дележ яблок

https://contest.misis.ru/contest/1081/problems/A

Время 1с. Память 256мб.

n школьников делят k яблок поровну, неделящийся остаток остается в корзине. Сколько яблок достанется каждому школьнику? Сколько яблок останется в корзине?

Входные данные

Программа получает на вход числа n и k. n - количество школьников, k - количество яблок. n, k - целые положительные числа, не превосходящие 10000.

Выходные данные

Программа должна вывести два числа через пробел: количество яблок у каждого школьника и количество яблок, оставшихся в корзине.

Примеры

Пример 1

3 14 4 2
Решение 
#include <iostream>

using namespace std;

int main() {
  int n, k;
  cin >> n >> k;

  cout << k / n << ' ' << k % n;
}

Задача D. Электронные часы

https://contest.misis.ru/contest/1081/problems/D

Время 1с. Память 256мб.

С начала суток прошло n минут. Определите, сколько часов и минут будут показывать электронные часы в этот момент. Программа должна вывести два числа: количество часов (от 0 до 23) и количество минут (от 0 до 59).

Входные данные

Вводится число n - целое, положительное, не превышает 10⁷

Выходные данные

Программа должна вывести два числа: количество часов (от 0 до 23) и количество минут (от 0 до 59).

Примеры

Пример 1

150 2 30

Пример 2

1441 0 1
Решение 
#include <iostream>

using namespace std;

int main() {
  int n;
  cin >> n;

  int t = n % (24 * 60);
  int h = t / 60;
  int m = t % 60;
  cout << h << ' ' << m;
}

Задача E. Стоимость покупки

https://contest.misis.ru/contest/1081/problems/E

Время 1с. Память 256мб.

Пирожок в столовой стоит a рублей и b копеек. Определите, сколько рублей и копеек нужно заплатить за n пирожков.

Входные данные

Программа получает на вход три числа a, b, n (0 ≤ a, n ≤ 10000; 0 ≤ b < 100).

Выходные данные

Программа должна вывести два числа: стоимость покупки в рублях и копейках.

Примеры

Пример 1

10 15 2 20 30

Пример 2

2 50 4 10 0
Решение 
#include <iostream>

using namespace std;

int main() {
  int a, b, n;
  cin >> a >> b >> n;

  int r = a * n + (b * n) / 100;
  int k = (b * n) % 100;
  cout << r << ' ' << k;
}

Даны два натуральных числа n и m. Если одно из них делится на другое нацело, выведите 1, иначе выведите любое другое целое число. Требуется решить задачу, не используя оператор if.

Задача G. Максимум

https://contest.misis.ru/contest/1081/problems/G

Время 1с. Память 256мб.

Напишите программу, которая считывает два целых числа a и b и выводит наибольшее значение из них. Числа - целые от 1 до 1000.

При решении задачи можно пользоваться только целочисленными арифметическими операциями +, -, *, /, %, =. Нельзя пользоваться условным оператором if.

Входные данные

Вводятся 2 числа

Выходные данные

Выведите ответ на задачу.

Примеры

Пример 1

Решение 
#include <iostream>

using namespace std;

int main() {
  int a, b;
  cin >> a >> b;

  int a1 = (a / b) * a;
  int b1 = (b / a) * b;
  int c = (a1 + b1) / (a/b + b/a);

  cout << c;
}
