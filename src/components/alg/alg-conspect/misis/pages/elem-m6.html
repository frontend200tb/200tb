Задача D. Наименьший делитель

https://contest.misis.ru/contest/1155/problems/D

(Время 1с. Память 64мб.)

Мальчик Петя собрался на День Рождения к своей подруге Ане. У Пети есть число n, а так же Кодекс Чести его Страны, руководствоваться которым должен каждый мальчик Страны, когда хочет сделать подарок своей подруге. Кодекс Чести говорит о том, что если у мальчика есть число n тогда он должен подарить своей подруге его минимальный делитель. Однако, Кодекс Чести также говорит о том, что лучше не дарить ничего, чем подарить число 1 или число n. Помогите Пете узнать, какое число он должен подарить Ане, чтобы он успел сделать для него красивую упаковку.

Входные данные

На вход подается число N (1 ≤ n ≤ 1000).

Выходные данные

Выведите наименьший делитель числа n, который не равен 1 и не равен n. Если такого делителя нет, выведите -1.

Примеры

Пример 1

6 2

Пример 2

357 3

Пример 3

2 -1
Решение 
#include <iostream>

using namespace std;

int main() {
  int n;
  cin >> n;

  int res = -1;
  for (int i = 2; i < n; i++) {
    if (n % i == 0) {
      res = i;
      break;
    }
  }

  cout << res;
}

"Я пригласил вас, господа, с тем, чтобы сообщить вам пренепреятное известие: к нам едет число N".

До последнего времени жизнь в уездном городе K протекала спокойно. Однако скорое прибытие числа N заставило жителей города подумать о будущем. Чтобы их город не стал называться уездным городом KN жителям необходимо выполнить заданиче числа N. После того, как число N прибудет жителям потребуется вывести все числа от 1 до N, кроме тех, при делении N на которые получаются целые числа. Помогите жителям уездного города K сохранить название своего города!

Входные данные

Вводится натуральное число N: 1 ≤ N ≤ 1000.

Выходные данные

Выведите все числа от 1 до N, пропуская те, на которые N делится без остатка. Используйте оператор continue.

Примеры

Пример 1

3 2

Пример 2

4 3
Решение 
#include <iostream>

using namespace std;

int main() {
  int n;
  cin >> n;

  for (int i = 1; i <= n; i++) {
    if (n % i == 0) {
      continue;
    }
    cout << i << ' ';
  }
}

Задача D. Сапоги скороходы

https://contest.misis.ru/contest/1155/problems/D

(Время 1с. Память 256мб.)

Девочке Алене ее двоюродная бабушка-ведьма подарила сапоги скороходы. Сапоги устроены следующим образом: перед тем, как надеть их, надо загадать число a и прыгнуть на один метр в ту сторону, куда собираешься отправиться. После этого каждый i-ый шаг получается длиной ai метров. Алена загадала число a и отправилась гулять. Определите, на каком расстоянии от дома она окажется через n шагов.

Входные данные

Даны два числа a и n (1 ≤ a ≤ 5, 1 ≤ n ≤ 12).

Выходные данные

Выведите одно число: расстояние на которое Алена ушла от дома.

Примеры

Пример 1

2 5 63
Решение 
#include <iostream>

using namespace std;

int main() {
  int a, n;
  cin >> a >> n;

  int cnt = 1;
  int step = a;
  for (int i = 0; i < n; i++) {
    cnt += step;
    step *= a;
  }

  cout << cnt;
}

Задача E. Остатки

https://contest.misis.ru/contest/1155/problems/E

(Время 1с. Память 256мб.)

Даны целые неотрицательные числа a, b, c, d, при этом 0 ≤ c < d. Выведите в порядке возрастания все числа от a до b, которые дают остаток c при делении на d.

Входные данные

Вводятся четыре целых числа a, b, c, d.

Выходные данные

Выведите ответ на задачу. Если таких чисел в указанном интервале нет, то ничего выводить не нужно.

В этой задаче нельзя использовать if и операторы сравнения. Должен быть только один цикл.

Примеры

Пример 1

2 5 0 2 2 4
Решение

В цикле первое число это ближайшее к a, которое делится на d с остатком c.

#include <iostream>

using namespace std;

int main() {
  int a, b, c, d;
  cin >> a >> b >> c >> d;

  int start = (a - c + d - 1) / d * d + c;
  for (int i = start; i <= b; i += d) {
    cout << i << ' ';
  }
}
