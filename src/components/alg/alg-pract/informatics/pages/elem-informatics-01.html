Задача 1476 informatics.msk.ru

Условие

Часовая стрелка повернулась с начала суток на d градусов. Определите, сколько сейчас целых часов h и целых минут m.

Входные данные

На вход программе подается целове число d (0 ≤ d < 360)

Выходные данные

Выведите на экран фразу

It is ... hours ... minutes.

Вместо многоточий программа должна выводить значения h и m, отделяя их от слов ровно одним пробелом.

Пример

90 It is 3 hours 0 minutes.
Решение

При полном обороте часовой стрелки от 0 до 360 градусов пройдет 12 * 60 = 720 минут. Получается, что поворот часовой стрелки на 1 градус равен 2 минутам. При повороте на d градусов, пройдет d * 2 минут.

#include <iostream>

using namespace std;

int main() {
  int d;
  cin >> d;
  int h, m;
  h = d / 30;
  m = (d * 2) % 60;
  cout << "It is " << h << " hours "
       << m << " minutes.";
}

Задача 1484 informatics.msk.ru

Условие

На вход программе подаются два целых числа m и n, по модулю не превосходящие 10⁶. Если m делится на n или n делится на m, то требуется вывести 1, в противном случае - любое другое число.

Условный оператор использовать нельзя.

Входные данные

Целые числа m, n

Выходные данные

1 или другое целове число

Пример 1

2
8
1

Пример 2

0
0
100

Пример 3

5
0
1
Решение с логическими операторами 
#include <iostream>

using namespace std;

int main() {
  int m, n;
  cin >> m >> n;
  int res = (n != 0) && (m % n == 0)
         || (m != 0) && (n % m == 0);
  cout << res;
}
Решение математическое

Переход к абсолютным значениям не изменит делимости чисел, но упростит работу с остатками при отрицательных числах. Введем два числа m1 и n1 которые всегда равны нулю, кроме случаев, когда m и n равны нулю. В этих случаях m1 и n1 равны 1. Добавляя m1 и n1 мы избавимся от деления на 0. Если хотябы одно число делится на другое, то произведение равно нулю. Прибавив к этому 1 мы получим ответ 1. Если m и n равны нулю, то ответ должен быть не 1. Добавим к ответу m1 * n1.

#include <iostream>

using namespace std;

int main() {
  int m, n;
  cin >> m >> n;
  m = abs(m);
  n = abs(n);
  int m1 = 1 / (m + 1);
  int n1 = 1 / (n + 1);
  int res = (m % (n + n1)) * (n % (m + m1)) + 1 + m1 * n1;
  cout << res;
}

Задача 1486 informatics.msk.ru

Условие

Определите, верно ли, что в заданном четырехзначном числе ровно две одинаковые цифры.

Условный оператор, операторы сравнения, булевский тип данных, строка и множество использовать нельзя.

Входные данные

Целое число k (1000 ≤ k ≤ 9999)

Выходные данные

1 если да, или любое число в противном случае

Пример 1

2008 1

Пример 2

2002 0
Решение

Найдем все цифры k1, k2, k3, k4 четырехзначного числа k. Разность двух одинаковых цифр равна нулю. Проверим разность всех пар цифр на равенство нулю.

int a1 = 1 / (1 - abs(k1 - k2)); // 1 если цифры равны 
#include <iostream>

using namespace std;

int main() {
  int k;
  cin >> k;
  int k1 = k % 10;
  k /= 10;
  int k2 = k % 10;
  k /= 10;
  int k3 = k % 10;
  k /= 10;
  int k4 = k;
  int r1 = 1 / (1 - abs(k1 - k2));
  int r2 = 1 / (1 - abs(k1 - k3));
  int r3 = 1 / (1 - abs(k1 - k4));
  int r4 = 1 / (1 - abs(k2 - k3));
  int r5 = 1 / (1 - abs(k2 - k4));
  int r6 = 1 / (1 - abs(k3 - k4));
  int res = r1 + r2 + r3 + r4 + r5 + r6;
  cout << res;
}

Задача 1487 informatics.msk.ru

Условие

На вход программе подаются 4 целых числа, по модулю не превосходящие 10⁶: m, n, k, l. Если остаток от деления m на n равен k или l, то выведите 1, в противном случае - любое другое число.

Условный оператор использовать нельзя.

Входные данные

Целые числа m, n, k, l

Выходные данные

1 или другое целове число

Пример 1

12
8
3
4
1

Пример 2

0
5
1
2
0
Решение математическое

Чтобы избежать деления на 0, введем новую переменную n1, которая будет равна 1 если n = 0 и равна 0 если n != 0

int n1 = 1 / (abs(n) + 1);

Тогда можно записать остаток от деления и не бояться что может быть деление на 0

int r = m % (n + n1);

Если остаток от деления равен k или l, то произведение равно нулю

(r - k) * (r - l) = 0

Если число n = 0 то ответ не должен быть равен 1, тогда к ответу прибавим n1.

#include <iostream>

using namespace std;

int main() {
  int m, n, k, l;
  cin >> m >> n >> k >> l;
  int n1 = 1 / (abs(n) + 1);
  int r = m % (n + n1);
  int res = (r - k) * (r - l) + 1 + n1;
  cout << res;
}
