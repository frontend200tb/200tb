Markdown

  • Заголовоки h1-h16 выделяются с помощью решетки # 
    # Заголовок 1
## Заголовок 2
### Заголовок 3
#### Заголовок 4
##### Заголовок 5
###### Заголовок 6
    Решетку # можно поставить с двух сторон 
    # Заголовок 1 #
## Заголовок 2 ##
### Заголовок 3 ###
#### Заголовок 4 ####
##### Заголовок 5 #####
###### Заголовок 6 ######
    Заголовок h1 можно выделить если на следующей строке указать любое количество знаков «равно» = 
    Заголовок 1
===
    Заголовок h2 можно выделить если на следующей строке указать любое количество знаков «минус» - 
    Заголовок 2
--
  • Абзац создается при помощи пустой строки. Если вокруг текста сверху и снизу есть пустые строки, то текст превращается в абзац.
  • Перенос строки. Чтобы сделать перенос строки вместо абзаца, нужно поставить два пробела в конце предыдущей строки.
  • Горизонтальная черта 
    ***
    или 
    ---
    или 
    ___
  • Курсивный текст 
    *курсивный текст*
    или 
    _курсивный текст_
  • Жирный текст 
    **жирный текст**
    или 
    __жирный текст__
  • Жирный курсивный текст 
    ***жирный курсивный текст***
    или 
    __жирный курсивный текст___
  • Зачеркнутый текст ~~зачеркнутый текст~~
  • Картинка ![текст тега alt](https://pictures.com/foto.jpg "текст тега title")
  • Ссылка 
    [ссылка на гугл](https://google.com)
  • Картинка-ссылка [![текст тега alt](https://pictures.com/foto.jpg "текст тега title")](https://pictures.com)

  • Цитаты

  • Исходный код пишут с помощью обратных кавычек. Инлайн код выделяют одиночными обратными кавычками. Блок кода - тройными 
    \`let a = 5;\`
    \`\`\`let a = 5;
let b = a +5;\`\`\`
  • Сноски 
    Сноска первая [^1] и сноска вторая [^2]
[^1]: Текст сноски 1
[^2]: Текст сноски 2
  • Таблица. Для таблиц используют вертикальную черту и дефис 
    цвет | rgb | hex
:----|:----:|------:
красный | 255,0,0 | ff0000
зеленый | 0,255,0 | 00ff00
синий   | 0,0,255 | 0000ff
  • Экранирование. Чтобы спецсимволы не исчезали и не влияли на оформление, нужно использовать экранирование. Как и во многих других языках программирования, этим целям служит обратная косая черта (бэкслеш).
