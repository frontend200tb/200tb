Заголовки (Headings)

Заголовоки h1-h6 выделяются с помощью решетки #

# Заголовок 1
## Заголовок 2
### Заголовок 3
#### Заголовок 4
##### Заголовок 5
###### Заголовок 6

Решетку # можно поставить с двух сторон

# Заголовок 1 #
## Заголовок 2 ##
### Заголовок 3 ###
#### Заголовок 4 ####
##### Заголовок 5 #####
###### Заголовок 6 ######

Заголовок h1 можно выделить если на следующей строке указать любое количество знаков «равно» =

Заголовок 1
===

Заголовок h2 можно выделить если на следующей строке указать любое количество знаков «минус» -

Заголовок 2
--

Параграфы (Paragraphs)

Абзац создается при помощи пустой строки. Если вокруг текста сверху и снизу есть пустые строки, то текст превращается в абзац.

Paragraph1

Paragraph2

Перенос строки (Line Breaks)

Перенос строки. Чтобы сделать перенос строки вместо абзаца, нужно поставить два пробела в конце предыдущей строки.

String1  
string2

Горизонтальная черта (Horizontal Rules)

Для горизонтальной черты используются три или более звездочки * или тире - или подчеркивания _.

*** --- ___

Жирный текст (Bold)

Для выделения жирным текстом надо добавить две звездочки * или два подчеркивания _ перед и после фразы, слова или буквы.

**жирный текст** __жирный текст__

Курсивный текст (Italic)

Для выделения курсивным текстом надо добавить одну звездочку * или одно подчеркивание _ перед и после фразы, слова или буквы.

*курсивный текст* _курсивный текст_

Жирный и курсивный текст (Bold and Italic)

Для выделения жирным и курсивным текстом надо добавить три звездочки * или три подчеркивания _ перед и после фразы, слова или буквы.

***жирный и курсивный текст*** ___жирный и курсивный текст___

Зачеркнутый текст (Strikethrough)

Для выделения зачеркнутым текстом надо добавить две тильды ~ перед и после фразы, слова или буквы.

~~зачеркнутый текст~~

Подчёркнутый текст (Underline)

В синтаксисе Markdown нет встроенного способа подчеркнуть текст. Но если ваш редактор поддерживает HTML, то можно использовать теги u:

<u>подчёркнутый текст</u>

Ссылки (Links)

Самый лёгкий способ поместить ссылку в Markdown — заключить её в угловые скобки.

<https://frontend200tb.ru/>

Чтобы оформить ссылкой часть текста, используется такой синтаксис: [текст](ссылка). Можно сделать всплывающую подсказку при наведении курсора. Для этого в круглых скобках после ссылки нужно поставить пробел и написать текст подсказки в кавычках.

[google.com](https://google.com) без подсказки [google.com](https://google.com "Всплывающая подсказка") с подсказкой

Картинки (Images)

Для картинки используется такой синтаксис: ![текст](путь к изображению). Можно сделать всплывающую подсказку при наведении курсора. Для этого в круглых скобках после ссылки на картинку нужно поставить пробел и написать текст подсказки в кавычках.

![Изображение](https://www.google.com/images/google.png) без подсказки ![Изображение](https://www.google.com/images/google.png "Всплывающая подсказка") с подсказкой

Картинка-ссылка (Image-Link)

[![текст тега alt](https://pictures.com/foto.jpg "текст тега title")](https://pictures.com)

Нумерованный список (Ordered List)

Для создания нумерованного списка перед пунктами нужно поставить число с точкой и пробел. При этом нумерация в разметке ленивая. Неважно, какие именно числа вы напишете: Markdown пронумерует список автоматически.

1. Первый пункт
2. Второй пункт
3. Третий пункт

Список можно начинать и не с единицы. Для нумерации важно только число, которое стоит перед первым пунктом.

27. Первый пункт
27. Второй пункт
27. Третий пункт

Между двумя нумерованными списками, идущими подряд, нужно отбить две пустые строки. Если отбить только одну, то Markdown воспримет два списка как один. Некоторые редакторы в таком случае увеличивают интервал между пунктами.

1. Первый пункт
  2. Второй пункт
  3. Третий пункт
  
  1. Четвёртый пункт
  2. Пятый пункт
  3. Шестой пункт

Маркированный список (Unordered List)

Для создания маркированного списка нужно поставить перед каждым пунктом звёздочку *, дефис - или плюс +.

* Первый пункт
* Второй пункт
* Третий пункт
- Первый пункт
- Второй пункт
- Третий пункт
+ Первый пункт
+ Второй пункт
+ Третий пункт

Вложенный список (Nested List)

Чтобы создать вложенный список, нужно поставить перед его пунктами табуляцию или несколько пробелов. В Markdown одна табуляция соответствует четырём пробелам.

Список одного вида можно вкладывать в любой другой.

1. Пункт
	1. Подпункт
		1. Подподпункт
- Пункт
	- Подпункт
		- Подподпункт

Чекбоксы (checkboxes)

Чтобы сделать чекбоксы, нужно использовать маркированный список, но между маркером и текстом поставить [x] для отмеченного пункта и [] — для неотмеченного.

- [x] Отмеченный пункт
- [ ] Неотмеченный пункт

Цитаты (Blockquotes)

Чтобы параграф отобразился как цитата, нужно поставить перед ним закрывающую угловую скобку >

> Оформление цитатой
последовательных строк
внутри одного параграфа

Внутрь одного блока цитаты можно поместить сразу несколько параграфов. Чтобы сделать это, нужно поместить закрывающую угловую скобку перед началом каждой строки.

> # Заголовок
> Первый параграф
>
> Второй параграф
>
> > Вложенная цитата

Вставка кода (Code)

Исходный код пишут с помощью обратных кавычек. Инлайн код выделяют одиночными обратными кавычками. Блок кода - тройными

\`let a = 5;\`
\`\`\`let a = 5;
let b = a +5;\`\`\`

Таблицы (Tables)

Таблица. Для таблиц используют вертикальную черту и дефис. Столбцы разделяются вертикальными линиями |, а строка с шапкой отделяется от остальных дефисами -, которых можно ставить сколько угодно.

цвет | rgb | hex
:----|:----:|------:
красный | 255,0,0 | ff0000
зеленый | 0,255,0 | 00ff00
синий   | 0,0,255 | 0000ff

Сноски (Reference)

Сноска состоит из метки и непосредственно текста сноски (содержимого).

Сноска первая [^1] и сноска вторая [^2]
[^1]: Текст сноски 1
[^2]: Текст сноски 2

Экранирование (Escaping Characters)

Чтобы спецсимволы не исчезали и не влияли на оформление, нужно использовать экранирование. Как и во многих других языках программирования, этим целям служит обратная косая черта (бэкслеш).

Список символов, которые нужно экранировать \`*_{}[]<>()#+-.! |
