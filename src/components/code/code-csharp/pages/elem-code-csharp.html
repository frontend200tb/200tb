install

Нужно установить среду разработки, где будем писать код и для получения результата. Рассмотрим два варианта.

Вариант 1. Полноценная среда разработки Visual Studio Community 2022.

Скачиваем установщик с оффициального сайта visualstudio.microsoft.com/ru/downloads . Запускаем его, установщик докачает все необходимое. Далее мы попадаем на экран где нужно выбрать какие модули надо установить. Выбираем Разработка классических приложений .NET, ASP.NET и разработка веб-приложений. После этого нажимаем Установить (или Изменить).

И при инсталляции Visual Studio на ваш компьютер будут установлены все необходимые инструменты для разработки программ, в том числе фреймворк .NET 7.

Для создания первой программы, откроем Visual Studio. На стартовом экране выберем Create a new project (Создать новый проект). В окне Создание проекта выбираем Консольное приложение .NET Core и жмем далее

Далее на следующем этапе нам будет предложено указать имя проекта и каталог, где будет располагаться проект.

На следующем окне Visual Studio предложит нам выбрать версию .NET, которая будет использоваться для проекта. Выберем последнюю на данный момент верси. - .NET 7.0

Нажмем на кнопку Create (Создать) для создания проекта, и после этого Visual Studio создаст и откроет нам проект

Вариант 2. Редактор кода Visual Studio Code.

В этом случае необходимо еще докачать платформу .NET dotnet.microsoft.com/en-us/download. После этого можно использовать нужные команды, которые будут доступны из терминала. С помощью этих команд можно создать или запустить проект. В VSCode надо установить расширение C# от ms-dotnettools.csharp от Microsoft C# for Visual Studio Code (powered by OmniSharp)

Чтобы создать новый проект нужно открыть пустую папку. Открыть терминал. Команда clear - очистить терминал

Проверим установленную версию .NET dotnet --version
7.0.401

Создадим новое консольное приложение dotnet new console -o sample

Наберем текст программы using System;
namespace sample {
class Program {
static void Main(string[] args) {
// Урок 1. Вывод в консоль
Console.WriteLine("Hello, World!");

// Урок 2. Типы данных (Ссылочные типы и типы значения)
// Переменные типов значения (value types) хранятся в стеке и
// каждая занимает определенный объем памяти.
// Всего 11 типов значений. Из них 8 численных типов
// byte название типа в C#
// Systm.Byte название типа в .NET

// 8 численных типов
// System.Byte 1 байт
byte bmin = byte.MinValue;
byte bmax = byte.MaxValue;
Console.WriteLine("Тип данных byte. byte.MinValue = {0}, byte.MaxValue = {1}", bmin, bmax);

// sbyte 1 байт
sbyte sbmin = sbyte.MinValue;
sbyte sbmax = sbyte.MaxValue;
Console.WriteLine("Тип данных sbyte. sbyte.MinValue = {0}, sbyte.MaxValue = {1}", sbmin, sbmax);

// System.Int16 2 байта
short shmin = short.MinValue;
short shmax = short.MaxValue;
Console.WriteLine("Тип данных short. short.MinValue = {0}, short.MaxValue = {1}", shmin, shmax);

// ushort 2 байта
ushort ushmin = ushort.MinValue;
ushort ushmax = ushort.MaxValue;
Console.WriteLine("Тип данных ushort. ushort.MinValue = {0}, ushort.MaxValue = {1}", ushmin, ushmax);

// System.Int32 4 байта
int imin = int.MinValue;
int imax = int.MaxValue;
Console.WriteLine("Тип данных int. int.MinValue = {0}, int.MaxValue = {1}", imin, imax);

// uint 4 байта
uint uimin = uint.MinValue;
uint uimax = uint.MaxValue;
Console.WriteLine("Тип данных uint. uint.MinValue = {0}, uint.MaxValue = {1}", uimin, uimax);

// System.Int64 8 байт
long lmin = long.MinValue;
long lmax = long.MaxValue;
Console.WriteLine("Тип данных long. long.MinValue = {0}, long.MaxValue = {1}", lmin, lmax);

// ulong 8 байт
ulong ulmin = ulong.MinValue;
ulong ulmax = ulong.MaxValue;
Console.WriteLine("Тип данных ulong. ulong.MinValue = {0}, ulong.MaxValue = {1}", ulmin, ulmax);

// System.Single 4 байта
float fmin = float.MinValue;
float fmax = float.MaxValue;
float f = 12.9f;
Console.WriteLine("Тип данных float. float.MinValue = {0}, float.MaxValue = {1}", fmin, fmax);

// System.Double 8 байт
double dmin = double.MinValue;
double dmax = double.MaxValue;
double d = 123.45;
Console.WriteLine("Тип данных double. double.MinValue = {0}, double.MaxValue = {1}", dmin, dmax);

// System.Decimal 16 байт
decimal mmin = decimal.MinValue;
decimal mmax = decimal.MaxValue;
decimal m = 16.77m;
Console.WriteLine("Тип данных decimal. decimal.MinValue = {0}, decimal.MaxValue = {1}", mmin, mmax);
Console.WriteLine("float f = {0}, double d = {1}, decimal m = {2}", f, d, m);

// System.Boolean 1 байт
bool yes = true;
bool no = false;
Console.WriteLine("Тип данных bool. Имеет только два значения {0} и {1}", yes, no);

// 3 нечисленных типа
// System.Char 2 байта
char ch = 'a';
char ch2 = 'b';
Console.WriteLine("Тип данных char. {0} {1}", ch, ch2);

// string
string s = "frontend";
Console.WriteLine(s);

//Console.ReadKey();
}
}
}

Запустим созданный проект командой dotnet run

.Net

.Net (дот нет) Framework - платформа программирования под Windows

.Net Core - открытый кросс-платформенный фреймворк под Windows, Linux, macOS

C# (си шарп) - основной язык программирования платформы .Net

.Net Framework и .Net Core состоят из исполняющей среды и базовой библиотеки классов

CLR (Common Language Runtime) - исполняющая среда под платформой .Net Framework

Core CLR - исполняющая среда под платформой .Net Core

BCL (Base Class Library) = FCL (Foundation Class Library) + Frameworks
