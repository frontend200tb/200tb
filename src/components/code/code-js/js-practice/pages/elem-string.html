Тема 1. Определить что подстрока является частью строки

Как можно проверить что подстрока является частью строки. Есть три способа.

const s = 'crossroad';
const sub = 'road';

1 способ. Метод includes() из es6.

Вернется true если подстрока является частью строки.

const c = s.includes(sub);

2 способ. Метод indexOf() из es5.

Вернется индекс первого вхождения если подстрока является частью строки.

const d = s.indexOf(sub);

2 способ. Метод test() регулярного выражения.

Создадим переменную regExp в которую запишем регулярное выражение, которое представляет собой подстроку. Вызовем на этой переменной метод test() и передадим в него строку. Это более гибкий метод, где можно настроить регистр символов, начало или конец строки проверять и т.д.

const regExp = /road/;
const e = regExp.test(s);
