1. Определить что подстрока является частью строки

Как можно проверить что подстрока является частью строки. Есть три способа.

const s = 'crossroad';
const sub = 'road';

1 способ. Метод includes() из es6.

Вернется true если подстрока является частью строки.

const c = s.includes(sub);

2 способ. Метод indexOf() из es5.

Вернется индекс первого вхождения если подстрока является частью строки.

const d = s.indexOf(sub);

2 способ. Метод test() регулярного выражения.

Создадим переменную regExp в которую запишем регулярное выражение, которое представляет собой подстроку. Вызовем на этой переменной метод test() и передадим в него строку. Это более гибкий метод, где можно настроить регистр символов, начало или конец строки проверять и т.д.

const regExp = /road/;
const e = regExp.test(s);

2. Переставить буквы в строке

Написать функцию, которая принимает строку и массив целых положительных чисел из всех индексов этой строки. Возвращает новую строку, в которой буквы стоят в той последовательности, какая указана в массиве.

Примеры

домик, [4,1,0,3,2] // кодим украл, [1,4,3,2,0] // клару
Решение через for 
function restore(str, arr) {
  const result = [];

  for (let i = 0; i < str.length; i++) {
    result.push(str[arr[i]]);
  }

  return result.join('');
}
Решение через reduce 
function restoreReduce(str, arr) {
  return arr.reduce((acc, next) => acc + str[next], '');
}

3. Объединить две строки

Имеется несколько способов объединения строк

1 способ. Оператор + str = 'a' + 'b';
2 способ. Оператор += str = 'a'; str += 'b';
3 способ. Метод строки concat() str = 'a'; str = str.concat('b');
4 способ. Метод массива join() str = ['a', 'b'].join('');
