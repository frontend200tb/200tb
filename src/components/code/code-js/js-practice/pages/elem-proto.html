Задачa 2

const a = { x: 1 };

  console.log(a.__proto__ === Object.prototype); // ?
  console.log(a.hasOwnProperty('x')); // ?
  
  console.log(a.hasOwnProperty('__proto__')); // ?
  console.log(a.__proto__.hasOwnProperty('__proto__')); // ?
Ответ

Свойство __proto__ это сеттер-геттер. Оно находится в прототипе и само на себя показывает. А в каждом экземпляре объекта есть скрытое свойство [[Prototype]], которое указывает на __proto__
