Объект window

Объект window содержит коллекцию frames всех фреймов, заданных в HTML документе с помощью контейнернго тега frameset. Объект window имеет свойства, методы, события и дочерние объекты

onbeforeunload

Событие - перед закрытием страницы. Что позволяет сохранить данные. Событие происходит перед загрузкой новой страницы или перед закрытием окна браузера.

onblur

Событие - выход окна из фокуса. Когда пользователь переключается между открытыми страницами, переводя фокус с одной страницы на другую. Действие пользователя вызывает событие onblur.Событие onblur также можно вызвать программным способом, используя метод blur.

onerror

Событие - ошибка при передаче. Если происходит ошибка при загрузке страницы или ее элемента, происходит событие onerror. Можно использовать это событие в программе, чтобы попытаться еще раз загрузить страницу или изменить дальнейшие действия.

onfocus

Событие - окно становится активным. Когда пользователь переключается между открытыми страницами, переводя фокус с одной страницы на другую. Действие пользователя вызывает событие onfocus.Событие onfocus также можно вызвать программным способом, используя метод focus.

onload

Событие - страница полностью загружена

onresize

Событие - изменение размеров окна

onscroll

Событие - прокрутка окна

onunload

Событие - непосредственно перед закрытием страницы

Worker

Для фоновых вычислений нужно создать отдельный файл worker.js.

// worker.js
onmessage = function(event) {
  postMessage('Принял ' + event.data)
}

После этого мы можем создать объект.

let worker = new Worker('worker.js')

Передадим число 100, как сообщение, в worker.

worker.postMessage(100)

И выведем полученный ответ в консоль

let worker = new Worker('worker.js');
worker.postMessage(100);
worker.onmessage = function(event) {
  console.log(event.data);
}
