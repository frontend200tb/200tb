document (window.document)

Объект document

Все объекты сгруппированы в коллекции, которые содержат массивы объектов.

  1. all

    Коллекция всех объектов документа document.all[0] // html

  2. anchors

    Коллекция всех якорей (закладок) в документе document.anchors // HTMLCollection[]

  3. applets

    Коллекция всех объектов в документе, включая встроенные элементы управления, графические элементы, апплеты, внедренные и другие объекты document.applets // HTMLCollection[]

  4. embeds

    Коллекция всех внедренных объектов в документе document.embeds // HTMLCollection[]

  5. forms

    Коллекция всех форм на странице document.forms // HTMLCollection[form]

  6. frames

    Коллекция всех фреймов, определенных в теге frameset document.frames // undefined

  7. images

    Коллекция всех изображений на странице document.images // HTMLCollection[]

  8. links

    Коллекция всех ссылок и блоков area на странице document.links[0] // a

  9. plugins

    Еще одно название для коллекции внедренных объектов документа document.plugins // HTMLCollection[]

  10. scripts

    Коллекция всех разделов scripts на странице document.scripts // HTMLCollection[]

  11. styleSheets

    Коллекция всех конкретных свойств стиля, определенных в документе document.scripts // HTMLCollection[]

Коллекция имеет свойство length - количество всех ее элементов.
