Javarush University

Курс Frontend Full Stack разработчик

Конспект курса на русском языке

https://javarush.com/university/frontend

За 12 месяцев освойте современную веб-разработку: HTML, CSS, JavaScript, TypeScript, React, Node.js и Angular. Создавайте сложные веб-приложения с нуля, работайте с REST API и базами данных. Ментор поможет с кодом, а карьерный центр — с поиском работы. Станьте универсальным веб-разработчиком.

Готовим Middle Frontend разработчиков, за которых работодатели будут драться! Освой полный цикл веб‑разработки с ментором и создавай потрясающие проекты.

Обучение на Фронтенд‑разработчика с нуля включает в себя 5 модулей с лекциями, практикой и итоговыми проектами (от новичка до уровня Middle-разработчика), а также финальный проект.
