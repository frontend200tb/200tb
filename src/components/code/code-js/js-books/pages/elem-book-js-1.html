1.1 Упражнение. Используем консоль веб-обозревателя (17)

Код JavaScript можно набирать в консоле браузера и сразу видеть результат его выполнения.

Откроем браузер Chrome. Открыть консоль можно двумя способами:

  1. Нажать F12 и перейти на вкладку Console
  2. Нажать правой кнопкой мыши внутри страницы, выбрать пункт меню Inspect и перейти на вкладку Console

Сложим два числа

27 + 14 41

Объединим две строки в одну

'Java' + 'Script' 'JavaScript'

Сравним два числа и узнаем, действительно ли число 10 меньше числа 20

10 < 20 true

Сравним два числа и узнаем, действительно ли число 15 больше числа 20

15 > 20 false

Очистка консоли браузера

Очистить консоль можно тремя способами:

  1. Нажать на иконку Clear console
  2. Нажать комбинацию клавиш Ctrl + L
  3. Нажать правой кнопкой мыши внутри консоли и выбрать пункт меню Clear console

1.2 Типы данных и операторы (20)

Любое значение в JavaScript относится к определенному типу данных.

1.3 Числа, арифметические операторы и приоритет операторов (20)

К типу данных number относятся целые и вещественные числа. Кроме десятичных, числа могут быть восьмеричные или шестнадцатеричные. Вначале восьмеричного числа ставится 0, вначале шестнадцатеричного числа ставится 0x.

Если в одном выражении присутствуют несколько операторов, то вначале выполняются операторы с большим приоритетом, затем с меньшим.

К числам также относятся три специальных значения NaN, Infinity, -Infinity.

1.4 Строки. Конкатенация строк (22)

К типу данных string относятся строки. Строки пишутся в одинарных или двойных кавычках. Оператор + выполняет конкатенацию двух строк.

'Java' + 'Script' 'JavaScript'

1.5 Условия и операторы сравнения (23)

Операторы сравнения возвращают результат типа boolean. В языке JavaScript есть восемь операторов сравнения <, <=, >, >=, ==, !=, ===, !== .

1.6 Логические операторы (24)

Логические операторы также возвращают результат типа boolean. В языке JavaScript есть три логических оператора &&, ||, ! .

1.7 Переменные (24)

Переменная - это ячейка памяти для хранения значения. Перед испольвозванием переменной ее нужно объявить. При объявлении переменная сразу получает значение undefined. Объявить переменную можно с помощью оператора let. Оператор let всегда возвращает значение undefined.

let sum undefined

В объявленную переменную можно сохранить значение. Значение в переменную записывается с помощью оператора присваивания. Оператор присваивания всегда возвращает значение, записанное в переменную.

sum = 10 10

Можно в одном выражении объявить переменную и присвоить ей значение

let a = 1

1.7.1 Три оператора объявления переменных (27)

В JavaScript есть три оператора объявления переменных let, const, var. Только оператор var позволяет повторно объявить одну и ту же переменную, но в этом случае он ничего не делает.

1.7.2 Еще об объявлении переменных (27)

Если перед использованием переменная не была объявлена, то браузер ее сам объявит неявно

sum = 10

Однако лучше объявлять переменные явно. В строгом режиме браузер будет требовать явного объявления переменных.

1.8 Арифметические операторы, применимые лишь к переменным (28)

В JavaScript есть только три арифметических оператора, которые применимы только к переменным, их нельзя применять к значениям. Это унарные операторы -, ++, --.

1.9 Условный оператор (28)

Тернарным называется оператор, который принимает три операнда. В JavaScript есть только один тернанрный оператор. Это условный оператор

операнд1 ? операнд2 : операнд3

Если условие в операнде 1 истинно, то возвращается операнд 2, если ложно - то операнд 3.

1.10 Получение типа данных (29)

Оператор typeof возвращает тип данных в виде строки

typeof 123 'number' let str = 'a' typeof str 'string' typeof (24 > 90) 'boolean' let n typeof n 'undefined'

1.11 Преобразование типов (30)

Если в одной операции есть данные различных типов, то сначала JavaScript неявно преобразует данные к одному типу, потом вычисляется результат операции. В одном выражении может быть несколько операций, для каждой операции происходит свое неявное преобразование типов.

  1. При сложении строки с любым другим типом данных, другой тип неявно приводится к строке и происходит конкатенация строк

    Конкатенация строки и числа

    20 + '20' '2020'

    Конкатенация строки и числового значения NaN, Infinity, -Infinity

    'a' + NaN aNaN 'a' + Infinity 'aInfinity' 'a' + -Infinity 'a-Infinity'

    Конкатенация строки и логического значения true, false

    'a' + true 'atrue' 'a' + false 'afalse'

    Конкатенация строки и undefined

    'a' + undefined 'aundefined'

  2. При сложении числа с любым типом данных, кроме строки, другой тип неявно приводится к числу и происходит сложение чисел

    Сложение числа с логическим значением. true неявно преобразовывается в 1, false неявно преобразовывается в 0

    20 + true 21 20 + false 20

  3. При выполнении любого арифметического действия, кроме сложения, все операнды приводятся к числу

    Строки содержащие только цифры и десятичную точку будут преобразованы в число

    20 - '10' 10 50 * '1.5' 75

    Все остальные строки будут преобразованы в числовое значение NaN

    20 - 'abc' NaN

    Значение undefined преобразовывается в NaN

    20 - undefined NaN

  4. Если в качестве условия используется значение не логического типа, то оно приводится к значению логического типа

    В значение false преобразуются только 6 значений

    1. false
    2. 0
    3. NaN
    4. undefined
    5. null
    6. ""

    Все остальные значения преобразуются в true

  5. Операторы сравнения ==, != преобразуют значения к одному типу, потом их сравнивают

    6. Операторы строгого сравнения ===, !== сначала сравнивают типы и если они не равны, то возвращают false

Для явного преобразования строки в число можно использовать унарный оператор +

+'123' 123

Для преобразования строки в целое число можно использовать функцию parseInt. Строка может содержать пробелы в начале и в конце, а также другие символы после числа.

parseInt(' 100str ') 100

Для преобразования строки в вещественное число можно использовать функцию parseFloat. Строка может содержать пробелы в начале и в конце, а также другие символы после числа.

parseFloat(' 123.456str ') 123.456

Урок 2. Выражения и управляющие конструкции (35)

2.1 Упражнение. Пишем первый веб-сценарий (35)

Напишем html файл, в который будет встроен веб-сценарий.

Создадим файл 2.1.html и запишем в него:

<h1>Пересчет величин</h1>
<script>
  const inch = window.prompt('Величина в дюймах', 0);
  const cm = inch * 2.54;
  window.document.write('<p>', cm, '</p>');
</script>

2.2 Упражнение. Применяем условные выражения (38)

Если в программе 2.1 в окне ввода величины нажать отмену, то функция window.prompt вернет null. JavaScript неявно преобразует null в 0 и выведет ответ 0. Давайте проверим, что если пользователь нажал отмену, то выведем сообщение : "Величина не была введена".

Создадим файл 2.2.html и запишем в него:

<h1>Пересчет величин</h1>
<script>
  const inch = window.prompt('Величина в дюймах', 0);
  let cm;
  if (inch == null) {
    cm = 'Величина не была введена';
  } else {
    cm = inch * 2.54;
  }
  window.document.write('<p>', cm, '</p>');
</script>

2.2.1 Сокращенная форма условного выражение (39)

Условное выражение можно записать в сокращенной форме.

Создадим файл 2.2.1.html и запишем в него:

<h1>Пересчет величин</h1>
<script>
  const inch = window.prompt('Величина в дюймах', 0);
  let cm = 'Величина не была введена';
  if (inch != null) {
    cm = inch * 2.54;
  }
  window.document.write('<p>', cm, '</p>');
</script>

2.2.2 Множественное условное выражение (39)

Множественное условное выражение проверяет несколько условий.

Создадим файл 2.2.2.html и запишем в него:

<h1>Пересчет величин</h1>
<script>
  const inch = window.prompt('Величина в дюймах', 0);
  let cm;
  if (inch == null) {
    cm = 'Величина не была введена';
  } else if (inch < 0) {
    cm = 'Величина не может быть отрицательной';
  } else {
    cm = inch * 2.54;
  }
  window.document.write('<p>', cm, '</p>');
</script>

2.3 Упражнение. Используем блоки (40)

Блок позволяет записывать несколько выражений внутри фигурных скобок {}. Объявим переменную cm только когда нужно вывести результат.

Создадим файл 2.3.html и запишем в него:

<h1>Пересчет величин</h1>
<script>
  const inch = window.prompt('Величина в дюймах', 0);
  if (inch == null) {
    window.document.write('<p>', 'Величина не была введена', '</p>');
  } else if (inch < 0) {
    window.document.write('<p>', 'Величина не может быть отрицательной', '</p>');
  } else {
    let cm = inch * 2.54;
    window.document.write('<p>', cm, '</p>');
  }
</script>

2.3.1 Переменные в блоках (41)

Переменные объявленные в блоке операторами const или let существуют только внутри блока. При выходе из блока эти переменные удаляются из памяти.

Переменные объявленные в блоке операторами var существуют и внутри и за пределами блока. Если блок находится внутри функции, то при выходе из функции эти переменные удаляются из памяти.

2.4 Выражение выбора (41)

Выражение выбора switch case последовательно сравнивает значение заданное в блоке switch со значениями указанными в блоках case. При сравнении используется оператор строгого равенства ===, то есть преобразование типов при сравнении не производится. Обязательно использовать оператор break, чтобы не выполненять код последующих сравнений.

let screenSize = 4;
let s;
switch (screenSize) {
  case 3:
    s = 'маленький экран';
    break;
  case 4:
    s = 'средний экран';
    break;
  case 5:
    s = 'большой экран';
    break;
  default:
    s = 'неизвестный размер';
}

2.5 Упражнение. Используем цикл со счетчиком (43)

Выведем на экран таблицу с числами от 1 до 10 и квадратными корнями из них.

Создадим файл 2.5.html и запишем в него:

<h1>Квадратные корни от 1 до 10</h1>
<table>
  <script>
  for (let i = 1; i <= 10; i++) {
    window.document.write('<tr><td>', i, '</td>');
    window.document.write('<td>', Math.sqrt(i), '</td></tr>');
  }
  </script>
</table>

2.5.1 Объявление счетчика непосредственно в цикле (46)

При объявлении счетчика в условии цикла, он будет существовать только внутри цикла.

for (let i = 1; i <= 10; i++)

2.5.2 В приращении можно записать несколько выражений (46)

В условии цикла можно писать несколько выражений через запятую.

for (let i = 0, j = 0; i <= 10; i++, j += 2)

2.6 Циклы с предусловием и постусловием (46)

В цикле с предусловием whileусловие проверяется перед каждым проходом цикла.

В цикле с постусловием do whileусловие проверяется после каждого прохода цикла.

2.7 Операторы break и continue (47)

Оператор break прерывает цикл. Оператор continue прерывает один проход цикла.

2.8 Комментарии (48)

Комментарии JavaScript не обрабатывает и пропускает.

2.9 Строгий режим (48)

Чтобы включить строки режим, первым выражением в сценарии должна быть строчка

'use strict'

В строгом режиме все переменные должны быть предварительно объявлены. Иначе сценарий завершится с ошибкой.
