Регулярные выражения

Регулярное выражение можно создать двумя способами:

  1. Через литерал let reg = /[a-z]i/;

    Компиляция регулярного выражения, созданного через литерал, проиходит при анализе скрипа. Чтобы увеличисть производительность пользуйтесь этим способом если регулярное выражение постоянно.

  2. Через функцию конструктор объекта RegExp let reg = new RegExp('ab');

    Компиляция регулярного выражения, созданного через конструктор, проиходит во время выполнения скрипта. Пользуйтесь этим способом, только если шаблон заранее неизвестен или регулярное выражение может изменяться.
