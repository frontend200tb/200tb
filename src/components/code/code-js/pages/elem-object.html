about

Объект имеет ссылочный тип данных. Это набор свойств, раделенных запятой. Каждое свойство состоит из ключа и значения. После ключа ставится двоеточие. Ключ может иметь тип данных строка или символ.

Способы создания объекта:

  1. С помощью литерала объекта {}
  2. С помощью конструктора new Object()
  3. С помощью метода Object.create({})
// Создадим пустой объект
let a = {};
let b = new Object();
let c = Object.create({});

// Создадим объект с одним свойством
let a = {language: 'javascript'};
let b = new Object({language: 'javascript'});
let c = Object.create({}, {language: {value: 'javascript'}})

Методы объектов

Методы объекта Object, записанные в его прототипе Object.prototype.

  1. Object.assign() Принимает целевой объект и исходный объект (объекты).
    Изменяет целевой объект.
    Возвращает целевой объект.
    Копирует значений всех собственных перечисляемых свойств из одного или более исходных объектов в целевой объект.
  2. Object.create() Принимает объект, который станет прототипом созданного объекта и опционально свойства.
    Возвращает новый объект с заданным прототипом и свойствами.
  3. Object.defineProperties() Принимает два параметра - исходный объект и свойства
    Изменяет исходный объект.
    Возвращает исходный объект с измененными (или новыми) свойствами.
  4. Object.defineProperty() Принимает три параметра - исходный объект, свойство и дескриптор свойства
    Изменяет исходный объект.
    Возвращает исходный объект с измененным (или новым) свойством.
  5. Object.entries() Принимает объект, чьи перечислимые свойства будут возвращены в виде массива [key, value].
    Возвращает массив собственных перечисляемых свойств указанного объекта в формате [key, value].
  6. Object.freeze() Принимает исходный объект
    Изменяет исходный объект.
    Возвращает исходный объект, заморозив добавление, изменение и удаление свойств в объекте
  7. Object.fromEntries() Принимает итерируемый объект (Array, Map и другие)
    Возвращает новый объект с свойствами в указанными в принятом объекте
  8. Object.getOwnPropertyDescriptor() Принимает два параметра - объект, в котором ищется свойство и имя свойства, чье описание будет возвращено
    Возвращает дескриптор собственного свойства указанного объекта
  9. Object.getOwnPropertyDescriptors() Принимает объект, для которого нужно получить все собственные дескрипторы свойств
    Возвращает объект, содержащий все собственные дескрипторы свойств исходного объекта
  10. Object.getOwnPropertyNames() Принимает исходный объект, чьи перечисляемые и неперечисляемые собственные свойства будут возвращены
    Возвращает массив строк, который соответствует свойствам, найденным непосредственно в данном объекте
  11. Object.getOwnPropertySymbols() Принимает исходный объект, чьи символьные свойства будут возвращены
    Возвращает массив всех свойств символа, найденных прямо на данном объекте
  12. Object.getPrototypeOf() Принимает исходный объект, чей прототип будет возвращён
    Возвращает прототип (то есть, внутреннее свойство [[Prototype]]) указанного объекта
  13. Object.groupBy() Принимает два параметра - итерируемые элементы и callback-функцию brВозвращает новый объект
  14. Object.hasOwn() Принимает исходный объект и свойство
    Возвращает true если свойство принадлежит объекту, иначе возвращает false если такого свойства нет или оно унаследовано
  15. hasOwnProperty() Принимает свойство, которое будет проверяться у объекта
    Возвращает true если исходный объект соделжит указанное свойство, иначе возвращает false
  16. Object.is() Принимает два значения
    Возвращает true если указанные два значения одинаковые
  17. Object.isExtensible() Принимает исходный объект
    Возвращает true если исходный объект расширяемый (то есть, можно ли к нему добавлять новые свойства).
  18. Object.isFrozen() Принимает исходный объект
    Возвращает true если исходный объект заморожен
  19. isPrototypeOf() Принимает объект, в чьей цепочке прототипов производится поиск
    Возвращает true если исходный объект входит в цепочку прототипов указанного в параметре объекта

  21. Object.keys() Принимает исходный объект, чьи собственные перечисляемые свойства будут возвращены
    Возвращает массив из собственных перечисляемых свойств исходного объекта
  22. Object.preventExtensions() Принимает исходный объект, который нужно сделать нерасширяемым
    Возвращает исходный объект, который делается нерасширяемым
  23. propertyIsEnumerable() Принимает имя проверяемого свойства
    Возвращает true если указанное свойство является перечисляемым
  24. Object.seal() Принимает исходный объект который надо запечатать
    Возвращает запечатанный исходный объект, предотвращая добавление новых свойств к объекту и делая все существующие свойства не настраиваемыми
  25. Object.setPrototypeOf() Принимает два параметра - исходный объект, которому устанавливается прототип и новый прототип объекта
    Возвращает исходный объект с установленным прототипом
  26. toLocaleString() Не принимает параметры

  27. toString() Не принимает параметры

    Возвращает массив значений перечисляемых свойств объекта

1. Object.assign(target, source)

Принимает целевой объект и исходный объект (объекты)
Изменяет целевой объект.
Возвращает целевой объект.

Копирует значений всех собственных перечисляемых свойств из одного или более исходных объектов в целевой объект. После копирования он возвращает целевой объект.

const target = { a: 1, b: 2 };
const source = { b: 4, c: 5 };

const returnedTarget = Object.assign(target, source);

console.log(target); // { a: 1, b: 4, c: 5 }

console.log(returnedTarget === target); // true

2. Object.create(proto[, properties])

Появился в es5
Принимает объект, который станет прототипом созданного объекта и опционально свойства.
Возвращает новый объект с заданным прототипом и свойствами.

Параметры

proto Объект, который станет прототипом вновь созданного объекта.

properties Объект (необязательный параметр), чьи свойства представляют собой дескрипторы value, writable, enumerable, configurable попадут в новый объект.

Создадим объект с нулевым прототипом

Object.create(null); // {}

То же самое

Object.create(Object.prototype); // {}

Создадим новый объект со свойством language: javascript

Object.create({}, {language: {value: 'javascript'}}); // {language: 'javascript'}

3. Object.defineProperties(obj, props)

Принимает два параметра
  1. obj - исходный объект, на котором определяются новые или изменяются существующие свойства.
  2. props объект, чьи собственные перечисляемые свойства представляют собой дескрипторы для создаваемых или изменяемых свойств.

Возвращает исходный объект с измененными свойствами.

Метод Object.defineProperties() определяет новые или изменяет существующие свойства, непосредственно на объекте, возвращая этот объект.

Дескрипторы свойств обладают следующими дополнительными ключами:

  1. configurable Равен true только в том случае, если тип этого дескриптора свойства может быть изменён и если свойство может быть удалено из содержащего его объекта. Значение по умолчанию установлено в false.
  2. enumerable Равен true только в том случае, если это свойство можно увидеть через перечисление свойств содержащего его объекта. Значение по умолчанию установлено в false.
  3. value Значение, ассоциированное со свойством. Может быть любым допустимым значением JavaScript (числом, объектом, функцией и т.д.). Значение по умолчанию установлено в undefined.
  4. writable Равен true только в том случае, если значение, ассоциированное со свойством, может быть изменено с помощью оператора присваивания. Значение по умолчанию установлено в false.
  5. get Функция, используемая как геттер свойства, либо undefined, если свойство не имеет геттера. Возвращаемое значение функции будет использоваться как значение свойства. Значение по умолчанию установлено в undefined.
  6. set Функция, используемая как сеттер свойства, либо undefined, если свойство не имеет сеттера. Функция принимает единственным аргументом новое значение, присваиваемое свойству. Значение по умолчанию установлено в undefined.

Создадим объект

let d = {};
console.log(d); // {}

Определим в нем новое свойство

Object.defineProperties(d, {language: {value: 'typescript'}});
console.log(d); // {language: 'typescript'}

4. Object.defineProperty(obj, prop, descriptor)

Принимает три параметра:

  1. obj Объект, на котором определяется свойство.
  2. prop Ключ нового или изменяемого свойства.
  3. descriptor Дескриптор нового или изменяемого свойства.

Обычное добавление свойств через присваивание создаёт свойства, которые можно увидеть через перечисление свойств (с помощью цикла for...in или метода Object.keys), чьи значения могут быть изменены и которые могут быть удалены. Этот же метод позволяет настроить эти дополнительные детали свойства.

Создадим объект

let a = {};
console.log(a); // {}

Определим в нем новое свойство

Object.defineProperty(a, 'language', {
  value: "javascript",
  enumerable: true,
  configurable: true,
  writable: true,
  get: function () {
    return this.value;
  },
  set: function (newValue) {
    this.value = newValue;
  },
})
console.log(a); // {language: 'typescript'}

5. Object.entries(obj)

Принимает объект, чьи перечислимые свойства будут возвращены в виде массива [key, value].
Возвращает массив перечислений собственных свойств объекта с парами [key, value], в том же порядке, что и в цикле for...in. 
let obj = {a:1,b:2,c:3};
Object.entries(obj); // [['a',1],['b',2],['c',3]]

6. Object.freeze(obj)

Принимает исходный объект
Изменяет исходный объект
Возвращает исходный объект, заморозив добавление, изменение и удаление свойств в объекте

Метод Object.freeze() замораживает объект: это значит, что он предотвращает добавление новых свойств к объекту, удаление старых свойств из объекта и изменение существующих свойств или значения их атрибутов перечисляемости, настраиваемости и записываемости. В сущности, объект становится эффективно неизменным. Метод возвращает замороженный объект.

let a = {
  b: 10,
  c: true,
}
Object.freeze(a);
a.d = 20 // свойство d не добавится
a.b = 20 // свойство b не изменится
delete a.c // свойство c не удалится

7. Object.fromEntries(iterable)

Принимает итерируемый объект (Array, Map и другие)
Не изменяет исходный объект
Возвращает новый объект с свойствами в указанными в принятом объекте Object.fromEntries([['0','a'], ['1','b'],['2','c']]); // {0: 'a', 1: 'b', 2: 'c'} Object.fromEntries([['a','0'], ['b','1'],['c','1']]); // {a: '0', b: '1', c: '1'} 
const map = new Map([
  ["foo", "bar"],
  ["baz", 42],
]);
Object.fromEntries(map); // { foo: "bar", baz: 42 }

8. Object.getOwnPropertyDescriptor(obj, prop)

Принимает два параметра - объект, в котором ищется свойство и имя свойства, чье описание будет возвращено
Не изменяет исходный объект
Возвращает дескриптор собственного свойства указанного объекта 
let a = {foo: 17};
Object.getOwnPropertyDescriptor(a, 'foo'); // {value: 17, writable: true, enumerable: true, configurable: true}
let b = { get bar() { return 17 }, };
Object.getOwnPropertyDescriptor(b, 'bar'); // {set: undefined, enumerable: true, configurable: true, get: ƒ}
