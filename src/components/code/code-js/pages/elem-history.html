about

History API позволяет работать с историей браузера в пределах одной сессии. Браузер создаёт новую сессию, когда пользователь открывает новую вкладку или новое окно браузера.

С помощью History API можно переходить по истории вперёд, назад и управлять содержимым истории. Доступ к API осуществляется с помощью объекта window.history.

Свойства window.history

  1. window.history.length

    Количество записей в текущей сессии истории

  2. window.history.state

    Возвращает текущий объект истории

Методы window.history

  1. window.history.back()

    back() перемещает пользователя по истории на страницу назад. Метод, идентичный вызову go(-1)

  2. window.history.forward()

    forward() перемещает пользователя по истории на страницу вперёд. Метод, идентичный вызову go(1)

  3. window.history.go(n)

    go() универсальный метод для перемещения по истории вперёд или назад. Метод, позволяющий гулять по истории. В качестве аргумента передается смещение, относительно текущей позиции. Если передан 0, то будет обновлена текущая страница. Если индекс выходит за пределы истории, то ничего не произойдет.

  4. window.history.pushState(data, title [, url])

    pushState() добавляет новую запись в истории сессии. Добавляет элемент истории.

  5. window.history.replaceState(data, title [, url])

    replaceState() изменяет текущую запись в истории сессии. Обновляет текущий элемент истории
