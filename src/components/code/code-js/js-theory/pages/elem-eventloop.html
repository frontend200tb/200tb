Порядок работы Event loop

  1. JS это синхронный язык программирования и выполняется в один поток, поэтому он берет из кода последовательно каждую команду и смотрит
  2. Если это синхронная команда, то она передается в Call Stack и сразу же выполняется
  3. Если встречается microtask, то он кладется в очередь микрозадач Microtask queue
  4. Если встречается macrotask, то он кладется в очередь макрозадач Macrotask queue
  5. Когда весь синхронный код выполнится, выполняется очередь микрозадач. Берется первая команда из Microtask queue, передается в Call Stack и сразу же выполняется
  6. Когда очередь микрозадач вся выполнится, то выполняется одна задача из очереди макрозадач

Микрозадачи:

  1. process.nextTick()
  2. Promise callback
  3. async functions
  4. queueMicrotask
  5. MutationObserver

Макрозадачи:

  1. setTimeout()
  2. setInterval()
  3. setImmediate()
  4. requestAnimationFrame()
