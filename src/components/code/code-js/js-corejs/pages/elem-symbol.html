Symbol

Если у объекта есть какой-то признак, то это итерируемый объект и его можно подставлять в цикл for of. Для этого нужен уникальный идентификатор, который сам в итерации не участвует и не пересекается со стандартными ключами. Поскольку в итерации не участвуют объекты класса Symbol, то завели Symbol.iterator. Это как глобально созданный символ. Любой объект у которого мы добавим Symbol.iterator становится перебираемым.

Например, пусть есть объект

let o = {to: 10}

Если попробовать такой объект перебрать циклом for of, то выскочит ошибка

TypeError: o[Symbol.iterator] is not a function

Добавим в объект свойство Symbol.iterator, которому присвоим функцию, которая определяет метод перебора

let o = {
  to: 10,
  [Symbol.iterator]: function() {
    return {
      next() {
        return {
          done:true
        }
      }
    }
  }
}
