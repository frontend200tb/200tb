Свойства объекта Number

  1. Number.EPSILON
    Примерно равно 2 в минус 52 степени
    Свойство Number.EPSILON представляет собой разницу между единицей и наименьшим значением, большим единицы, которое может быть представлено типом Number
  2. Number.MAX_SAFE_INTEGER
    Равно 9007199254740991 или 253 - 1
    Константа, которая представляет максимальное безопасное целое число
  3. Number.MAX_VALUE
    Примерно равно 1.79E+308
    Свойство, которое представляет максимальное числовое значение, представимое в JavaScript. Значения, большие MAX_VALUE представляются как «бесконечность».
  4. Number.MIN_SAFE_INTEGER
    Равно -9007199254740991 или -253 - 1
    Константа, которая представляет минимальное безопасное целое число
  5. Number.MIN_VALUE
    Примерно равно 5e-324
    Свойство, которое представляет минимальное положительное числовое значение, представимое в JavaScript. Значения, меньшие MIN_VALUE преобразуются в 0
  6. Number.NaN
    Эквивалентно глобальному объекту NaN
  7. Number.NEGATIVE_INFINITY
    Отрицательная бесконечность
  8. Number.POSITIVE_INFINITY
    Положительная бесконечность

Методы объекта Number

  1. Number.isFinite()
    Принимает значение, и проверяет является ли это значение коненым числом
    Возвращает true если передано конечное число, иначе false
  2. Number.isInteger()
    Принимает значение, и проверяет является ли это значение целым числом
    Возвращает true если передано целое число, иначе false
  3. Number.isNaN()
    Принимает значение, и проверяет является ли это значение NaN
    Возвращает true если передано число со значением NaN, иначе false
  4. Number.isSafeInteger()
    Принимает значение, и проверяет является ли это значение безопасным целым числом
    Возвращает true если передано безопасное целое число, иначе false
  5. Number.parseFloat()
    Принимает строку
    Возвращает дробное число, иначе NaN если нельзя преобразовать в число
  6. Number.parseInt()
    Принимает строку и опционально целое число от 2 до 36, представляющее основание системы счисления
    Возвращает целое число, иначе NaN если нельзя преобразовать в число
  7. toExponential()
    Принимает опционально целое число, определяющее количество цифр после запятой
    Возвращает число - объект Number в экспоненциальной записи с одной цифрой перед запятой.
  8. toFixed()
    Принимает опционально целое число от 0 до 20, определяющее количество цифр после запятой. По умолчанию равно 0
    Возвращает число - объект Number без экспоненциальной записи с указанным количеством цифр после запятой.
  9. toLocaleString()
    Принимает опционально два параметра locales и options, которые определяют язык и правила форматирования результата
    Возвращает строку с языкозависимым представлением числа
  10. toPrecision()
    Принимает опционально целое число от 1 до 100, определяющее количество значащих цифр
    Возвращает строковое представление объекта Number с фиксированной запятой округленный до заданного количества значащих цифр
  11. toString()
    Принимает опционально целое число от 2 до 36, определяющее основание системы счисления
    Возвращает строковое представление объекта Number
  12. valueOf()
    Не принимает параметры
    Возвращает примитивное значение объекта Number

1. Number.isFinite()


Появился в ES6
Принимает значение, и проверяет является ли это значение коненым числом
Возвращает true если передано конечное число, иначе возвращает false
Здесь не происходит преобразования типов. Если пришло не число, то сразу же возвращается false. Этим отличается от глобальной функции isFinite(), которая сначала преобразует значение в число, а затем проверяет

Задачи на метод Number.isFinite()

Number.isFinite(25); // true Number.isFinite(0); // true Number.isFinite(-1.22); // true Number.isFinite(Infinity); // false Number.isFinite(NaN); // false Number.isFinite('10'); // false

2. Number.isInteger()


Появился в ES6
Принимает значение, и проверяет является ли это значение целым числом
Возвращает true если передано целое число, иначе возвращает false
Здесь не происходит преобразования типов. Если пришло не число, то сразу же возвращается false

Задачи на метод Number.isInteger()

Number.isInteger(25); // true Number.isInteger(0); // true Number.isInteger(-50); // true Number.isInteger(Infinity); // false Number.isInteger(NaN); // false Number.isInteger('10'); // false

Вещественное число с точкой

Number.isInteger(5.0); // true Number.isInteger(5.01); // false

3. Number.isNaN()


Появился в ES6
Принимает значение, и проверяет является ли это значение NaN
Возвращает true если передано число со значением NaN, иначе возвращает false
Здесь не происходит преобразования типов. Если пришло не число, то сразу же возвращается false. Этим отличается от глобальной функции isNaN(), которая сначала преобразует значение в число, а затем проверяет

Задачи на метод Number.isNaN()

Эти значения будут true и для проверки isNaN()

Number.isNaN(NaN); // true Number.isNaN(Number.NaN); // true Number.isNaN(0 / 0); // true

Эти значения будут false и для проверки isNaN()

Number.isNaN(true); // false Number.isNaN(null); // false Number.isNaN(37); // false Number.isNaN(0); // false Number.isNaN(-50); // false

Метод Number.isNaN(). Особенности

Это более надежная версия оригинальной глобальной функции isNaN()

При использовании глобальной функции isNaN() это всё будет true

Number.isNaN("NaN"); // false Number.isNaN(undefined); // false Number.isNaN({}); // false Number.isNaN("blabla"); // false

4. Number.isSafeInteger()


Появился в ES6
Принимает значение, и проверяет является ли это значение безопасным целым числом
Возвращает true если передано безопасное целое число, иначе возвращает false

Безопасное целое число может быть представлено как число двойной точности (от 2^53 - 1 до -2^53 - 1)

Задачи на метод Number.isSafeInteger()

Number.isSafeInteger(25); // true Number.isSafeInteger(0); // true Number.isSafeInteger(-100); // true Number.isSafeInteger(0.25); // false Number.isSafeInteger('55'); // false Number.isSafeInteger(Math.pow(2, 53)); // false Number.isSafeInteger(Math.pow(2, 53) - 1); // true Number.isSafeInteger(false); // false Number.isSafeInteger(Infinity); // false Number.isSafeInteger(25 / 0); // false

5. Number.parseFloat()


Появился в ES6
Принимает строку
Возвращает дробное число, иначе NaN если нельзя преобразовать в число

6. Number.parseInt()


Появился в ES6
Принимает строку и опционально целое число от 2 до 36, представляющее основание системы счисления
Возвращает целое число, иначе NaN если нельзя преобразовать в число

7. toExponential()


Появился в ES3
Принимает опционально целое число от 0 до 20, определяющее количество цифр после запятой. По умолчанию, показывает столько цифр, сколько необходимо для представления
Возвращает число - объект Number в экспоненциальной записи с одной цифрой перед запятой.

Задачи на метод toExponential()

let num = 2.26519; num.toExponential(3); // 2.265e+0

8. toFixed()


Появился в ES3
Принимает опционально целое число от 0 до 20, определяющее количество цифр после запятой. По умолчанию равно 0
Возвращает число - объект Number без экспоненциальной записи с указанным количеством цифр после запятой.

Задачи на метод toFixed()

let num = 2.26519; num.toFixed(); // 2 num.toFixed(8); // 2.26519000

9. toLocaleString()


Принимает опционально два параметра locales и options, которые определяют язык и правила форматирования результата
Возвращает строку с языкозависимым представлением числа

10. toPrecision()


Появился в ES3
Принимает опционально целое число от 1 до 100, определяющее количество значащих цифр
Возвращает в виде строки объект Number с фиксированной запятой округленный до заданного количества значащих цифр. Если необходимо, добавляется запятая и нули до указанной длины

Задачи на метод toPrecision()

Без параметров возвращает число без форматирования

let num = 15.3525; num.toPrecision(); // 15.3525 num.toPrecision(2); // 15 num.toPrecision(7); // 15.35250 let num = 0.002839; num.toPrecision(2); // 0.0028

11. toString()


Появился в ES1
Принимает опционально целое число от 2 до 36, определяющее основание системы счисления
Возвращает строковое представление объекта Number

Задачи на метод toString()

let num = 25; num.toString(); // 25 num.toString(2); // 11001 num.toString(8); // 31 num.toString(16); // 19

12. valueOf()


Появился в ES1
Не принимает параметры
Возвращает примитивное значение объекта Number

Задачи на метод valueOf()

let num = 25; num.valueOf(); // 25
