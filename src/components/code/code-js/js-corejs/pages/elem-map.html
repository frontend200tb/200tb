about

Map – это коллекция ключ/значение, как и Object. Но основное отличие в том, что Map позволяет использовать ключи любого типа.

Создание объекта Map

Создадим новую пустую коллекцию nums, в который потом можно будет добавлять элементы в виде пар ключ-значение.

const nums = new Map();

Создадим новую коллекцию month

const month = new Map([[1, 'januar'], [2, 'februar']]); // Map(2) {1 => 'januar', 2 => 'februar'}

Перебор объекта Map

Для перебора коллекции Map есть 3 метода:

  1. map.keys() – возвращает итерируемый объект по ключам,
  2. map.values() – возвращает итерируемый объект по значениям,
  3. map.entries() – возвращает итерируемый объект по парам вида [ключ, значение], этот вариант используется по умолчанию в for..of.
  4. map.forEach() - выполняет callback-функцию для каждого элемента коллекции

Свойства Map

После создания любая коллекция имеет два свойства

  1. constructor возвращает функцию конструктор коллекции function Map() { [native code] }
  2. size возвращает количество элементов в коллекции

Еще одно свойство есть у объекта Map:

  1. Map.prototype возвращает ссылку на объект Map.prototype. Позволяет добавлять новые свойства и методы к объекту Map()

Свойство size

Коллекции Map имеют только одно свойство - size.

Значение size является целым числом, которое показывает как много записей имеет объект Map. Функция для установки size равна undefined; вы не можете изменить это свойство.

const month = new Map([[1, 'januar'], [2, 'februar']]); // Map(2) {1 => 'januar', 2 => 'februar'} month.size; // 2

Методы Map

Методы объекта Map, записанные в его прототипе Map.prototype.

  1. clear()
    Изменяет исходную коллекцию
    Не принимает параметры
    Возвращает undefined. Удаляет все элементы из коллекции
  2. delete()
    Изменяет исходную коллекцию
    Принимает значение ключа, по которому удаляет элемент из коллекции
    Возвращает true, если элемент был успешно удален, иначе false
  3. entries()
    Не изменяет исходную коллекцию
    Не принимает параметры
    Возвращает новый Итератор, который содержит пары [ключ, значение] для каждого элемента в коллекции в порядке их добавления
  4. forEach()
    Не изменяет исходную коллекцию
    Принимает callback-функцию, которая будет выполнена для каждого элемента
    Возвращает undefined
  5. get()
    Не изменяет исходную коллекцию
    Принимает ключ, по которому ищется значение
    Возвращает значение по принятому ключу или undefined если такого ключа нет
  6. Map.groupBy()
    Не изменяет исходную коллекцию
    Принимает итерируемый объект и callback-функцию
    Возвращает новую коллекцию
  7. has()
    Не изменяет исходную коллекцию
    Принимает ключ, которое нужно проверить на наличие в коллекции
    Возвращает true, если по указанному ключу значение есть в коллекции, иначе false
  8. keys()
    Не изменяет исходную коллекцию
    Не принимает параметры
    Возвращает новый итератор со значениями всех ключей коллекции
  9. set()
    Изменяет исходную коллекцию
    Принимает пару ключ-значение, которую нужно добавить или обновить в коллекции
    Возвращает исходную коллекцию с добавленным новым значением
  10. values()
    Не изменяет исходную коллекцию
    Не принимает параметры
    Возвращает новый итератор, который содержит значения из исходной коллекции в порядке их добавления в коллекцию
