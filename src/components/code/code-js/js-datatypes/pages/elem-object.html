about

Объект имеет ссылочный тип данных. Это набор свойств, раделенных запятой. Каждое свойство состоит из ключа и значения. После ключа ставится двоеточие. Ключ может иметь тип данных строка или символ.

Способы создания объекта:

  1. С помощью литерала объекта {}
  2. С помощью конструктора new Object()
  3. С помощью метода Object.create({})
// Создадим пустой объект
let a = {};
let b = new Object();
let c = Object.create({});

// Создадим объект с одним свойством
let a = {language: 'javascript'};
let b = new Object({language: 'javascript'});
let c = Object.create({}, {language: {value: 'javascript'}})
