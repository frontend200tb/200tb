about

Тип данных number представляет собой число. Числа в js хранятся в 64-битном формате, который называют "числа с плавающей точкой двойной точности" (double precision floating point numbers). В рязрядах (0-51) хранится значение числа, (52-62) - положение десятичной точки, (63) - знак числа. Диапазон таких чисел от -(253-1) до +(253-1)

Кроме чисел, к типу данных number относятся также Infinity, -Infinity и NaN.

Большие числа можно писать слитно

let billion = 1000000000;

Можно использовать символ нижнего подчеркивания _. Это синтаксический сахар, который делает число более читабельным. Движок js попросту игнорирует _ между цифрами.

let billion = 1_000_000_000;

Записанные разным способом числа равны.

1000000000 === 1_000_000_000; // true

Если число большое и у него в конце есть много нулей, то можно записать такое число через символ e. Буква e умножает число на 1 с указанным количеством нулей

1000 === 1e3; // true 7300000 === 7.3e6; // true

Если число маленькое и у него в начале есть много нулей, то также это число можно записать через символ e, указав справа от него отрицательное число. Буква e делит число на 1 с указанным количеством нулей.

Например, запишем 1 микросекунду

0.000001 === 1e-6; // true

Особенности чисел 0 и -0.

Ноль представлен двумя числами 0 и -0.

Сравнение знаком ===. Такое сравнение показывает что в js это одно и то же число.

0 === -0; // true

Сравнение через Object.is. Такое сравнение показывает что в js это разные объекты.

Object.is(0, -0); // false

0 можно получить при делении на Infinity

1 / Infinity; // 0 1 / -Infinity; // -0

Особенности NaN

NaN (не число) означает "что угодно, только не число"

Сравнение знаком ===. Такое сравнение показывает, что два значения не равны, потому что это "что угодно, только не число"

NaN === NaN; // false

Сравнение через Object.is. Такое сравнение показывает что в js это один объект.

Object.is(NaN, NaN); // true

Особенности Infinity и -Infinity

Infinity означает бесконечность, -Infinity означает минус бесконечность.

Сравнение знаком ===.

Infinity === Infinity; // true

Сравнение через Object.is.

Object.is(Infinity, Infinity); // true

Infinity можно получить при делении на 0

1 / 0; // Infinity 1 / -0; // -Infinity

Создание числа

Создать число можно тремя способами, используя

  1. Литерал числа

    Этоот способ рекомендуется всегда, где это возможно

    let a = 10;

  2. Преобразование к числу

    Когда приходят данные, например, в виде строки, а нужно получить число

    +'10'; // 10 parseInt('10'); // 10 parseFloat('10'); // 10 Number('10'); // 10

  3. Конструктор объекта new Number()

    Этот способ не рекомендуется. При этом создаются объекты, а это ссылочный тип данных, поэтому каждый экземпляр объекта будет разным

    new Number(10) === new Number(10); // false new Number(10) == new Number(10); // false Object.is(new Number(10), new Number(10)); // false

Записать литерал числа можно тремя способами. Покажем на примере числа миллиард.

  1. Обычный

    1000000000

  2. С символом нижнего подчеркивания _

    1_000_000_000

  3. С символом e

    1e9

Есть четыре системы счисления, в которых можно записать литерал числа. Покажем на примере чилса 32

  1. Десятичная

    32 // 32

  2. Двоичная

    0b100000 // 32

  3. Восьмеричная

    0o40 // 32

    Если первая цифра 0, а следующие меньше 8, то js считает это число восьмеричным

    0777 === 511; // true 0777 // 511 0778 // 778

  4. Шестнадцатиричная

    0x20 // 32

Преобразование к числу

Есть два вида преобразования - явное и неявное.

Явное это когда мы сами специально преобразовываем нечисловой тип данных к числу.

Неявное преобразование к числу происходит в математических выражениях и везде, где js ожидает увидеть число.

Явное преобразование к числу. Функция Number(). Оператор унарный +

Есть два способа явно преобразовать значение к числу. Работают они одинаково

  1. Функция Number()
  2. Оператор унарный +

Функция Number() и оператор унарный + преобразуют к числу по своим правилам для каждого типа данных.

  1. Преобразование числа (тип данных number)

    Возвращается это же число

    Number(10); // 10

  2. Преобразование строки (тип данных string)

    Если строку возможно преобразовать в число - то возвращается это число

    Number('10'); // 10

    Пробельные символы по краям обрезаются

    Number(' 10 '); // 10

    Пустая строка возвращает 0

    Number(''); // 0

    Если строку невозможно преобразовать в число - то возвращается NaN

    Number('10a'); // NaN Number('10 5'); // NaN Number('true'); // NaN

  3. Преобразование true и false (тип данных boolean)

    Number(true); // 1 Number(false); // 0

  4. Преобразование null (тип данных null)

    Number(null); // 0

  5. Преобразование undefined (тип данных undefined)

    Number(undefined); // NaN

  6. Преобразование типа данных bigint

    Namber(1001n); // 1001

  7. Преобразование типа данных symbol

    При попытки преобразовать тип данных symbol получим ошибку

    Number(Symbol()); // TypeError: Cannot convert a Symbol value to a number

  8. Преобразование объекта

    Любой объект преобразовывается к числу NaN

    Number({}); // NaN

    Особенности преобразования объекта

    Есть две особенности преобразования объекта

    1. Преобразование массива

      Пустой массив вернет 0

      Number([]); //

      Если в массиве один элемент, то будет преобразован этот элемент

      Number([' 10 ']); // 10

      Если в массиве больше одного элемента, то вернет NaN

      Number([1, 2]); // NaN

    2. Преобразование объекта, у которого метод valueOf() возвращает значение

      Если в объекте определен метод valueOf(), который возвращает значение, то происходит преобразование к числу этого значения

      let a = {};
a.valueOf = function() {return '  10  '};
Number(a); // 10

Неявное преобразование к числу

При неявном преобразовании значение сначала преобразуется в число с помощью функции Number, потом уже число подставляется в выражение.

Знаки математических операций преобразуют операнды к числу

'10' - '2'; // 8 '10' / '2'; // 5 '10' * '2'; // 20 '10' ** '2'; // 100

Объект Math ожидает в качестве параметра число

Math.round('22.4'); // 22 Math.round(''); // 0 Math.round(true); // 1

Свойства чисел

  1. Number.EPSILON
    Примерно равно 2 в минус 52 степени
    Свойство Number.EPSILON представляет собой разницу между единицей и наименьшим значением, большим единицы, которое может быть представлено типом Number
  2. Number.MAX_SAFE_INTEGER
    Равно 9007199254740991 или 253 - 1
    Константа, которая представляет максимальное безопасное целое число
  3. Number.MAX_VALUE
    Примерно равно 1.79E+308
    Свойство, которое представляет максимальное числовое значение, представимое в JavaScript. Значения, большие MAX_VALUE представляются как «бесконечность».
  4. Number.MIN_SAFE_INTEGER
    Равно -9007199254740991 или -253 - 1
    Константа, которая представляет минимальное безопасное целое число
  5. Number.MIN_VALUE
    Примерно равно 5e-324
    Свойство, которое представляет минимальное положительное числовое значение, представимое в JavaScript. Значения, меньшие MIN_VALUE преобразуются в 0
  6. Number.NaN
    Эквивалентно глобальному объекту NaN
  7. Number.NEGATIVE_INFINITY
    Отрицательная бесконечность
  8. Number.POSITIVE_INFINITY
    Положительная бесконечность

1. Number.isFinite()


Принимает значение, и проверяет является ли это значение коненым числом
Возвращает true если передано конечное число, иначе возвращает false
Здесь не происходит преобразования типов. Если пришло не число, то сразу же возвращается false

Задачи на метод Number.isFinite()

Number.isFinite(0); // true Number.isFinite(Infinity); // false Number.isFinite(NaN); // false Number.isFinite('10'); // false

2. Number.isInteger()


Принимает значение, и проверяет является ли это значение целым числом
Возвращает true если передано целое число, иначе возвращает false
Здесь не происходит преобразования типов. Если пришло не число, то сразу же возвращается false

Задачи на метод Number.isInteger()

Number.isInteger(0); // true Number.isInteger(Infinity); // false Number.isInteger(NaN); // false Number.isInteger('10'); // false

Вещественное число с точкой

Number.isInteger(5.0); // true Number.isInteger(5.01); // false

3. Number.isNaN()


Принимает значение, и проверяет является ли это значение NaN
Возвращает true если передано число со значением NaN, иначе возвращает false
Здесь не происходит преобразования типов. Если пришло не число, то сразу же возвращается false

Задачи на метод Number.isNaN()

Эти значения будут true и для проверки isNaN()

Number.isNaN(NaN); // true Number.isNaN(Number.NaN); // true Number.isNaN(0 / 0); // true

Эти значения будут false и для проверки isNaN()

Number.isNaN(true); // false Number.isNaN(null); // false Number.isNaN(37); // false

Метод Number.isNaN(). Особенности

Это более надежная версия оригинальной глобальной функции isNaN()

При использовании глобальной функции isNaN() это всё будет true

Number.isNaN("NaN"); // false Number.isNaN(undefined); // false Number.isNaN({}); // false Number.isNaN("blabla"); // false

4. Number.isSafeInteger()


Принимает значение, и проверяет является ли это значение безопасным целым числом
Возвращает true если передано безопасное целое число, иначе возвращает false
