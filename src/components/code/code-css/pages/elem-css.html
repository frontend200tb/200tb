info

CSS (Cascading Style Sheets) - каскадные таблицы стилей. CSS ипользуется для стилизации веб-страниц, созданных с помощью HTML. CSS позволяет изменять размер шрифта, цвет или размер различных элементов на веб-странице. CSS предоставляет широкий набор возможностей для стилизации, что позволяет веб-сайтам работать на разных устройствах с разными размерами экранов.

HTML и CSS используются вместе для создания удобных и красивых веб-сайтов. HTML определяет структуру и содержание веб-страницы. CSS определяет стиль отображаемого на веб-странице содержания. Следуя принципу разделения задач, HTML заботится о содержании, а CSS заботится о стиле этого содержания.

В случае отсутствия CSS, для HTML применяются стили по-умолчанию.

CSS поддерживается группой внутри W3C (организации по стандартизации) под названием CSS Working Group. Группа определяет новые функции CSS в документе спецификаций CSS. Новые функции добавляются так, чтобы обеспечить обратную совместимость. Создатели веб-браузеров внедряют поддержку новых функций CSS на основании спецификаций CSS.

Подключение CSS на страницу

Способы добавления CSS на страницу

  1. Встроенный CSS

    Код CSS пишется в HTML документе в HTML тегах используя атрибут style. Только для этого элемента h1 применится указанный стиль и цвет текста станет красный.

    <h1 style="color: red">Hello</h1>

    Встроенный CSS делает невозможным повторное использование стилей ни на этой странице, ни на других страницах, потому что стили привязаны к определенному HTML элементу.

    Встроенный CSS можно использовать когда необходимо быстро изменить стили одного определенного элемента на веб-странице.

  2. Внутренний CSS

    Код CSS пишется в HTML документе внутри тега style, который добавляют внутрь тега head. Для всег элементов h1 на этой странице применится указанный стиль и цвет текста станет красный.

    <head>
  <style>
    h1 {
      color:red;
    }
  </style>
</head>
<body>
  <h1>Hello</h1>
</body>

    Внутренний CSS делает невозможным повторное использование стилей на других страницах, потому что стили привязаны к определенной странице.

  3. Внешний CSS

    Код CSS пишется в отдельном файле. Например, напишем стиль в файле style.css

    h1 {
  color: red;
}

    К HTML файлу подключают файл стилей с помощью тега link, который добавляют внутрь тега head.

    <head>
  <link rel="stylesheet" href="./style.css">
</head>
<body>
  <h1>Hello</h1>
</body>

    Внешний CSS позволяет повторно использовать стили и является рекомендуемым способом написания CSS.

Подключение шрифтов

При отображении страницы браузер выбирает шрифт по-умолчанию. Разные браузеры выбирают разные шрифты. Чтобы страницы выглядели одинаково, можно самому подключить нужный шрифт. Для этого нужно добавить ссылки на шрифты в HTML файл. После этого можно изменять шрифт используя свойство font-family. Обычно указывают несколько шрифтов в одной строчке один за одним. Тогда если первый шрифт недоступен, то будет использоваться следующий и так далее. Последним указывают семейство шрифтов из которого браузер сам выбирает доступный шрифт.

К HTML файлу подключают шрифты с помощью тега link, который добавляют внутрь тега head.

<head>
  <link rel="stylesheet" href="https://fonts.googleapis.com/">
</head>

Синтаксис CSS

В синтаксе CSS есть 6 понятий

  1. Правило

    h1 { color: red; }

    В одном файле CSS может быть много CSS правил. Файлы CSS состоят из CSS правил.

  2. Селектор

    h1

    Каждое правило начинается с селектора. Селектор определяет элемент, к которому применяются правила. В одном правиле можно использовать несколько селекторов, разделив их запятой

  3. Объявление

    { color: red; }

    После селектора ставятся фигурные скобки. Между фигурными скобками записывают объявления. Это пары свойство: значение. В одном правиле можно использовать несколько объявлений

  4. Свойство

    color:

    Первым в объявлении пишется свойство, после которого ставится двоеточие.

  5. Значение

    red;

    Вторым в объявлении пишется значение, после которого ставится точка с запятой.

  6. Комментарий

    /* комментарий */

    Можно использовать однострочный и многострочный комментарий, заключив его между /* */. Комментарии игнорируются браузерами и используются для объяснения кода или скрытия кода от браузера.

Специфичность

Когда к одному элементу на странице прописываются стили в разных строчках css файла, то браузер принимает решение на основе специфичности селектора.

У каждого селектора есть вес. К элементу применяются стили того селектора, вес которого больше.

Если вес селектора одинаковый, то применяется тот, который записан последним.

Вес селектора складывается из трех цифр

  • 1000 Селекторы по идентификатору (#id)
  • 100 Селекторы по классу (.class), по псевдоклассу (:pseudoclass), по аттрибуту ([attribute] или [attribute="value"])
  • 10 Селекторы по элементу (tagName), по псевдоэлементу (::pseudoelement)

Псевдоклассы :not и :is не влияют на специфичность

При этом, учитываются селекторы внутри них (внутри скобок)

Встроенные стили

Встроенные стили (inline) всегда важнее стилей, записанных в селекторах - в теге style или во внешем файле css. Поэтому применяются всегда встроенные стили, если они есть.

Но есть один способ переопределить встроенные стили и стили любого другого селектора. Нужно добавить ключевое слово !important в конце объявления.

h1 { color: red !important; }

Использование этого метода считается плохой практикой, так как он отменяет правила специфичности

Блочная модель

Каждый элемент, отображаемый на странице, можно рассматривать как блок (box). Блок имеет 4 характеристики, которые можно контролировать с помощью CSS. Эти характеристики влияют на отображение блока на странице.

  1. Content. Содержимое элемента

    Свойства width и height задают ширину и высоту элемента. По-умолчанию это ширина и высота контента.

    div {
  width: 200px;
  height: 200px;
  background-color: black;
}

  2. Padding. Внутренний отступ между контентом и границей

  3. Border. Граница элемента

    Свойство border определяет задает границу элемента и состоит из 3 частей, которые задают толщину, цвет и стиль линии границы.

    div {
  width: 200px;
  height: 200px;
  border: 5px solid black;
}

  4. Margin. Внешний отступ между границами соседних элементов

Псевдоэлементы

Псевдоэлементы - это ключевые слова, которые позволяют стилизовать части выбранного элемента.

Стилизация первой буквы. Для всех элементов p первая буква станет зеленая.

p::first-letter { color: green; }

Стилизация маркеров в списке. Во всех списках маркеры станут зеленые и примут вид стрелки =>.

li::marker { color: green; content: '\21D2' }

Стилизация плейсхолдера в текстовом поле ввода. Во всех текстовых полях ввода плейсхолдер станет зеленый и курсивный.

input::placeholder { color: green; font-style: italic }

Псевдоклассы

Псевдоклассы - это ключевые слова, которые позволяют стилизовать элементы в зависимости от их состояния.

Они также позволяют стилизовать элементы в зависимости от их расположения на HTML странице.

Они добавляются в конце селектора, чтобы придать элементу стиль, основанный на его состоянии или местоположении.

Применение стилей при наведении указателя мыши на элемент

h1:hover { color: red; }

Применение стилей для кнопки в отключенном состоянии

button:disabled { border: none };

Применение разных стилей зависимо от валидности введенных в них данных

input:invalid { color: red; } input:valid { color: green; }

Первый дочерний элемент списка

li:first-child { color: green; }

Последний дочерний элемент списка

li:last-child { color: blue; }

Второй дочерний элемент списка

li:nth-child(2) { color: red; }

Все четные строки таблицы

tr:nth-child(even) { background-color: gray; }

Все нечетные строки таблицы

tr:nth-child(odd) { background-color: white; }

Псевокласс отрицания. Выбирем элементы h1, у которых нет класса с именем default

h1:not(.default) { color: darkmagenta; }

Наследование

  • Некоторые свойства наследуются от родительского элемента.
  • Наследование позволяет повторно использовать стили, установленные для родительского элемента или любого из элементов-предков.
  • Не все свойства наслдуются. Как правило, наследуется большинство свойств, связанных с типографикой.
  • Именно потому установка такого свойства, как font-family, для элемента body или html изменяет шрифт для всей веб-страницы.

Принудительное наследование

Ключевое свойство inherit позволяет наследовать от родителя свойства, которые не наследуются по умолчанию

span { border: inherit; }

Отмена наследования

Ключевое свойство initial устанавливает значение по умолчанию

span { color: initial; font-size: initial; }

Наследование всех свойств

Свойство all со значением inherit устанавливает наследование для всех свойств

all: inherit;

Отмена наследования всех свойств

Свойство all со значением initial устанавливает значение по умолчанию для всех свойств

all: initial;

Переходы

HTML элементы могут иметь различные состояния при смене которых может изменяться, например, цвет элемента

Переходы позволяют определить, как элемент переходит из одного состояния в другое

Создание перехода

Для создания перехода нужно задать 2 свойства
  1. Свойство transition-property со значением того свойства элемента на котором будет создан переход transition-property: background-color; Можно создать переход на всех свойствах сразу transition-property: all; Как правило, все свойства, которые могут иметь промежуточное состояние, могут подвергаться переходу. Например, свойство font-family не может подвергнуться переходу, потому что у него нет промежуточного состояния. Для цвета есть промежуточное состояние и мы можем изменять его плавно. Размеры тоже можно изменять плавно.
  2. Свойство transition-duration со значением длительности перехода transition-duration: 1s;

Свойство transition-timing-function

Свойство transition-timing-function определяет временную функцию для скорости перехода в разный момент времени. По-умолчанию скорость линейная, свойство равно linear. Но мы можем задать любую другую.

transition-timing-function: ease;

Свойство transition-delay

По-умолчанию смена состояний происходит мгновенно. Но мы можем задать задержку перед началом перехода.

transition-delay: 2s;

Свойство transition это сокращение для 4 свойств

transition: all 0.8s linear 2s;

  1. transition-property

  2. transition-duration

  3. transition-timing-function

  4. transition-delay

Анимация

CSS позволяет анимировать различные HTML элементы путем анимации различных CSS свойств.

Анимация позволяет выделить интерактивные элементы

Анимация позволяет улучшить дизайн веб-страницы и повысить уровень восприятия контента пользователями

Создание анимации

  • @keyframes - ключевое слово
  • move - имя анимации, выбирается произвольно
  • from - стили в начале анимации
  • to - стили в конце анимации

Переместим элемент по оси X (первое значение) и по оси Y (второе значение). (0,0) это координаты левого верхнего угла.

@keyframes move {
  from {
    translate: 0px 0px;
  }
  to {
    translate: 200px 200px;
  }
}

Применение анимации

animation-name: move animation-duration: 3s animation-timing-function: ease-in-out

Повторение анимации

По-умолчанию анимация повторяется 1 раз. Но можно задать число повторений или бесконечную анимацию.

animation-iteration-count: 5; animation-iteration-count: infinite;

Задержка анимации

По-умолчанию анимация начинается сразу, без задержки. Но можно задать время задержки перед началом анимации.

animation-delay: 2s;

Направление анимации

По-умолчанию анимация воспроизводится в обычном направлении и свойство animaion-direction равно normal. Пожно выбрать другое направление

animaion-direction: reverse; animaion-direction: alternate;

Сокращенное свойство animation

animation: move 3s ease 2s infinite alternate;

Плавное изменение прозрачности

@keyframes fadeIn {
  from {
    opacity: 0;
  }
  to {
    opacity: 1;
  }
}

.element {
  animation: fadeIn 1s ease-in;
}

Вращение

Бесконечное вращение с одинаковой скоростью (линейной).

@keyframes rotate {
  from {
    transform: rotate(0deg);
  }
  to {
    transform: rotate(360deg);
  }
}

.element {
  animation: rotate 2s infinite linear;
}

Увеличение размеров

@keyframes pulse {
  0% {
    transform: scale(1);
  }
  50% {
    transform: scale(1.1);
  }
  100% {
    transform: scale(1);
  }
}

button:hover {
  animation: pulse 1s ease;
}

Позиционирование

По-умолчанию элементы размещаются на веб-странице один за другим согласно их местам в html документе.

Используя CSS свойство position можно изменить расположение любого элемента на веб странице

Свойство position может принимать одно из 5 значений

  1. static (по-умолчанию)

    position: static;

    Элементы распологаются в соответствии со структурой HTML документа

  2. relative

    .element {
  position: relative;
  top: 30px;
  left: 50px;
}

    Относительная позиция используется вместе с top, left, right или bottom для перемещения элемента относительно его стандартной позиции. Остальные элементы ведут себя так, как будто элемент остался на своем месте и занимает его.

  3. absolute

    position: absolute;

    Элемент может перекрывать другие элементы. Остальные элементы ведут себя так, как будто элемента нет на странице. Они о нем ничего не знают.

    Элемент с абсолютной позицией можно перемещать используя top, left, right или bottom, относительно ближайшего родительского элемента, имеющего не статическую позицию.

  4. fixed

    position: fixed;

    В фиксированной позиции элемент остается неподвижным на веб-странице даже при прокрутке страницы. Потому что он позиционируется относительно окна браузера.

    Элемент с фиксированной позицией можно перемещать используя top, left, right или bottom, относительно области просмотра (видимой области веб-страницы).

  5. sticky

    positionL sticky;

    Липкая позиция - это гибрид между относительной и фиксированной позицией.

    Элемент будет иметь относительную позицию, пока веб-страница не будет прокручена до определенного момента, после чего он уже будет иметь фиксированную позицию.

    Порог, после которого элемент становится фиксированным, задается с помощью свойств top, right, left или bottom.

Переполнение

Содержимое элемента может выйти за его границы, если его размеры меньше, чем размер его содержимого.

По-умолчанию содержимое отображается за рамками элемента-контейнера.

Свойство overflow

Свойство overflow управляет переполнением контента и может принимать одно из двух значений

  1. hidden

    overflow: hidden; Скрывает переполненный контент.

  2. auto

    overflow: auto; Показывает полосы прокрутки в элементе контейнера, если содержимое его переполняет.

Медиа-запросы

Медиа запросы (media queries) позволяют применять разные стили для разных размеров экрана и разных устройств пользователя.

Медиа запросы являются неотъемлемой частью адаптивного веб-дизайна, поскольку позволяют изменять макет веб страницы в зависимости от устройства пользователя.

Создание медиа запроса

Должно выполняться условие при котором применяются стили, указанные в медиа запросе.

Цвет текста всех параграфов станет красным, если ширина меньше или равна 500px

@media (max-width: 500px) {
  p {
    color: red;
  }
}

Цвет текста всех параграфов станет зеленый, если ширина больше или равна 700px

@media (mшт-width: 700px) {
  p {
    color: green;
  }
}

Можно объединять несколько медиа запросов в одном.

Для всех устройств имеющих портретную ориентацию и максимальную ширину 480px

@media (orientation: portrait) and (max-width: 480px) {
  .content {
    width: 100%;
  }
}
