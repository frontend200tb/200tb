display

Каждый элемент является прямоугольным блоком. Свойство display контролирует как этот блок отображается на странице. Все элементы делятся на три группы.

  1. Блочные элементы (h1, h2, div, section, ...) имеют значение

    display: block;

    Блочные элементы начинаются с новой строки, занимают всю доступную ширину внутри родительского элемента. Им можно задать свойства width, height, margin.

  2. Строчные элементы (span, strong, em, img, ... ) имеют значение

    display: inline;

    Строчные элементы появляются в одном ряду при наличии свободного места и имеют тот же разммер, что и их контент. Они игнорируют свойства width, height, верхний и нижний margin.

    Примечание: можно задавать ширину и высоту изображений для тега img, при этом размер строчного элемента img будет принимать размер его содержимого, то есть будет изменяться вместе с размером изображения. Если задать только ширину или высоту, то второе свойство будет рассчитано автоматически, чтобы сохранить пропорцию изображения.

  3. Блочно-строчные элементы (button, input, select) имеют значение

    display: inline-block;

    Блочно-строчные (inline-block) элементы появляются в одном ряду при наличии свободного места. Им можно задать свойства width, height, margin.

Свойство display может принимать одно из 5 значений

  1. block

  2. inline

  3. inline-block

  4. flex

  5. grid

box-sizing

Свойство box-sizing задает блочную модель элементу и может принимать одно из двух значений.

  1. content-box

  2. border-box

Значение по-умолчанию

box-sizing: content-box;

В модели content-box свойства width и height это ширина и высота контента. Таким образом элемент на странице может занимать больше места, чем указано в свойствах width и height.

Ширина элемента = левая граница + левый внутренний отступ + контент + правый внутренний отступ + правая граница.

Высота элемента = верхняя граница + верхний внутренний отступ + контент + нижний внутренний отступ + нижняя граница.

Второе значение

box-sizing: border-box;

В модели border-box свойства width и height это ширина и высота элемента. Таким образом элемент на странице занимает ровно столько места, сколько указано в свойствах width и height.

margin

Свойство margin это сокращение для 4 свойств

  1. margin-top

  2. margin-right

  3. margin-bottom

  4. margin-left

По-умолчанию некоторые элементы имеют внешний отступ, но не все. Чтобы его задать или убрать надо использовать свойство margin.

margin это сокращенное свойство для внешних отступов всех четырех сторон

margin: 25px;

margin сверху и снизу 10px, справа и слева 20px

margin: 10px 20px;

margin сверху 10px, справа и слева 20px, снизу 30px

margin: 10px 20px 30px;

margin разный для четырех сторон по часовой стрелке, начиная с верхней стороны

margin: 10px 20px 30px 40px;

margin для каждой стороны можно задать отдельным свойством

margin-top: 10px; margin-right: 20px; margin-bottom: 30px; margin-left: 40px;

padding сверху 10px, справа и слева 20px, снизу 30px

padding: 10px 20px 30px;

Граница (Border)

К границе относятся 2 свойства - border и border-radius

border

Свойство border это сокращение для 7 свойств

  1. border

    1. border-width

    2. border-style

    3. border-color

  2. border-top

    1. border-top-width

    2. border-top-style

    3. border-top-color

  3. border-right

    1. border-right-width

    2. border-right-style

    3. border-right-color

  4. border-bottom

    1. border-bottom-width

    2. border-bottom-style

    3. border-bottom-color

  5. border-left

    1. border-left-width

    2. border-left-style

    3. border-left-color

По-умолчанию у элемента нет границы. Чтобы задать границу надо использовать свойство border. Зададим границу толщиной 2px, сплошной линией, черного цвета

border: 2px solid black;

border это сокращенное свойство для трех свойств

border-width: 2px; border-style: solid; border-color: black;

border-style (стиль границы) может принимать одно из 8 значений:

  1. solid
  2. dotted
  3. dashed
  4. double
  5. ridge
  6. groove
  7. inset
  8. outset

Для каждой из 4 сторон границы можно задать отдельное свойство, которое будет сокращением из трех свойств.

border-radius

Свойство border-radius это сокращение для 4 свойств

  1. border-top-left-radius

  2. border-top-right-radius

  3. border-bottom-right-radius

  4. border-bottom-left-radius

По-умолчанию граница имеет прямые углы. Чтобы их скруглить нужно использовать свойство border-radius

border-radius: 25px;

У элемента будут закругленные углы с радиусом 25px даже если у элемента нет границы (не задано свойство border). Обычно без границы применяют на изображениях или на элементах с заданным цветом фона.

border-radius это сокращенное свойство для всех четырех углов

Все углы закруглены на 25px

border-radius: 25px;

Разные радиусы углов по часовой стрелке, начиная с верхнего левого угла

border-radius: 10px 15px 20px 25px;

Можно задать радиус для каждого угла в отдельном свойстве

border-top-left-radius: 10px; border-top-right-radius: 15px; border-bottom-right-radius: 20px; border-bottom-left-radius: 25px;

Фон (Background)

К фону относится 2 свойства

  1. background
  2. background-blend-mode

В блочной модели элементы имеют фон, который занимает контент элемента и внутренний отступ.

background

Свойство background это сокращение для 8 свойств

  1. background-attachment
  2. background-clip
  3. background-color
  4. background-image
  5. background-origin
  6. background-position
  7. background-repeat
  8. background-size

background-attachment

Свойство background-attachment определяет, будет ли фоновое изображение зафиксировано, или оно будет прокручиваться вместе с остальным содержимым страницы

background-clip

Свойство background-clip определяет область прорисовки фонового изображения

background-color

Свойство background-color определяет цвет фона

background-color: green;

background-image

Свойство background-image определяет путь к картинке

background-image: url('https://picsum/id/46/200');

background-origin

Свойство background-origin определяет область позиционирования фонового изображения

background-position

Свойство background-position определяет начальное положение фонового изображения в пределах содержащего его элемента

background-repeat

Свойство background-repeat определяет, как будет повторяться картинка, заданная свойством background-image, если ее размер меньше размера элемента.

Свойство background-repeat может принимать одно из 4 значений

  1. repeat

    Картинка повторяется так чтобы занять все свободное пространство, по караям обрезается

  2. no-repeat

    Картинка не повторяется, а свободное пространство заполняется цветом фона, если он задан

  3. round

    Картинка масштабируется и повторяется без обрезки, чтобы заполнить свободное пространство

  4. space

    Картинка повторяется столько раз, сколько возможно без обрезки с добавлением отступов чтобы заполнить свободное пространство

background-size

Размер картинкок зависит от их фактической ширины и высоты.

Свойство background-size переопределяет размер, который займет картинка. Картинки могут быть растянуты, чтобы заполнить весь фон элемента или ограничены, чтобы вписаться в границы элемента без обрезки.

Свойство background-size может принимать одно из 4 значений

  1. contain

    Картинка вписывается в свободное пространство с сохранением соотношения сторон. Картинка повторяется с обрезкой, если остается свободное место

  2. cover

    Картинка заполняет все свободное пространство и для этого она может быть обрезана или растянута

  3. background-size: 50% 200px;

    Можно отдельно задать два значения - первое для ширины, второе для высоты картинки. Картинка может потерять пропорции. Проценты берутся от размера свободного пространства.

  4. background-size: 200px;

    5. Можно задать одно значение для ширины. Высота будет рассчитана автоматически с сохранением пропорции

background-blend-mode

Свойство background-blend-mode определяет режим наложения фонового изображения элемента на фоны других элементов. Это режим смешивания фоновых слоев. Фон одного элемента может иметь несколько слоев. Количество слоев опрделяется количеством значений, разделенных запятыми в свойстве background-image

background-image: linear-gradient(45deg, #f8a261, #6e9fc9), url('./image.png');

Первый фон из списка будет применяться к верхнему слою. В этом примере мы увидим только градиентную заливку.

Свойство background-blend-mode делает верхний слой полупрозрачным, что позволяет увидеть сквозь него нижние слои. Обязательное условие - фоновых слоев должно быть минимум два.

background-image: linear-gradient(45deg, #f8a261, #6e9fc9), url('./image.png'); background-blend-mode: overlay;

Режимы смешивания пишутся через запятую. Всего может быть 6 режимов смешивания

  1. Нормальный режим - normal background-blend-mode: normal; // значение по умолчанию Отсутствует смешивание и верхний слой остается непрозрачным
  2. Режим контраста - overlay, soft-light, hard-light
  3. Режим затемнения - darken, multiply, color-burn
  4. Режим осветления - lighten, screen, color-dodge
  5. Режим сравнения - difference, exclusion
  6. Компонентный режим - hue, saturation, color, luminosity

@font-face

Позволяет определить настройки шрифтов, а также загрузить специфичный шрифт на компьютер пользователя

font

Свойство font это сокращение для 6 свойств

  1. font-style
  2. font-variant
  3. font-weight
  4. font-size
  5. line-height
  6. font-family

В свойстве font 6 свойств идут в следующем порядке

font: [font-style||font-variant||font-weight] font-size [/line-height] font-family | inherit

В качестве обязательных значений свойства font указывается размер шрифта и его семейство. Остальные значения являются опциональными и задаются при желании. Любые не указанные значения сбрасываются до дефолтных. Хотя значения для свойств font-size-adjust и font-kerning не могут быть указаны внутри шортката font, они тоже сбрасываются до дефолтных.

Пример краткой записи:

p { font: 12px sans-serif; }

Пример полной записи

p { font: italic small-caps bold 12px/18px serif; }

Системные шрифты задаются обычным свойством font всего с одним параметром — тип шрифта. Разумеется, тип начертания, размер и прочие атрибуты уже содержатся внутри этого параметра.

.menu { font: menu; }

Допускается в качестве значения использовать ключевые слова, определяющие шрифт различных элементов операционной системы пользователя:

caption Шрифт на управляющих элементах (кнопки, радио, выпадающие списки).

icon Шрифт для текста под иконками.

menu Шрифт, используемый в системных меню.

message-box Шрифт в диалоговых окнах.

small-caption Шрифт для подписей к небольшим элементам управления.

status-bar Шрифт в строке состояния.

font-style

Свойство font-style определяет стиль начертания шрифта и принимает одно из 2 значений

  1. normal
  2. italic

font-variant

Before CSS3, this property accepted one of two possible values: normal (how text is rendered by default) and small-caps. The font-variant CSS shorthand property allows you to set all the font variants for a font. Краткая запись для рендеринга шрифтов. Определяет, как следует представлять строчные буквы - делать их все прописными уменьшенного размера или оставить без изменений

font-weight

Свойство font-weight определяет вес (толщину) шрифта и задается цифровым значением или именем

  1. 100 thin
  2. 200 extraLight
  3. 300 light
  4. 400 regular
  5. 500 medium
  6. 600 semiBold
  7. 700 bold
  8. 800 extraBold
  9. 900 black
  10. 950 extraBlack

font-size

Свойство font-size определяет размер шрифта

font-size: 20px;

line-height

Свойство line-height определяет высоту строки текста.

line-height: 1.8;

Нужно умножить размер шрифта на значение свойства line-height чтобы получить высоту строки.

font-family

Задает семейство шрифта, которое будет использоваться для оформления текста содержимого

font-family: 'Josefin Sans', sans-serif;

font-feature-settings

Низкоуровневое управление настройками шрифтов

font-kerning

Кернинг

font-size-adjust

Относительный размер шрифта

font-stretch

Ширина шрифта. Устанавливает более широкое или узкое начертание шрифта

font-synthesis

Transition (Переходы)

К переходам относятся 5 свойств.

  1. transition
    1. transition-property
    2. transition-duration
    3. transition-timing-function
    4. transition-delay

Позволяет делать плавные переходы между двумя значениями какого-либо CSS свойства

transition

Свойство transition это сокращение для 4 свойств

  1. transition-property
  2. transition-duration
  3. transition-timing-function
  4. transition-delay

Запишем сокращенное свойство четырех свойств

transition: all 0.8s linear 2s;
  • all - для всех свойств
  • 0.8s - длительность перехода 0.8 секунд
  • linear - линейная функция
  • 2s - задержка 2 секунды перед переходом

Запишем сокращенное свойство трех свойств

transition: all 0.8s linear;
  • all - для всех свойств
  • 0.8s - длительность перехода 0.8 секунд
  • linear - линейная функция
  • по-умолчанию задержка перед переходом 0 секунд

Запишем сокращенное свойство двух свойств

transition: all 0.8s;
  • all - для всех свойств
  • 0.8s - длительность перехода 0.8 секунд
  • по-умолчанию ease функция
  • по-умолчанию задержка перед переходом 0 секунд

transition-property

Определяет CSS свойство, для которого будет применён эффект перехода. Это могут быть такие свойства как background-color, color или width

transition-property: background-color;

transition-duration

Устанавливает продолжительность анимации перехода

transition-duration: 0.8s;

transition-timing-function

Определяет кривую скорости для анимации перехода

transition-timing-function: linear;

Свойство transition-timing-function может принимать значения:

  • linear

    Одинаковая скорость от начала до конца

  • ease (по умолчанию)

    Увеличение скорости к середине и замедление к концу

  • ease-in

    Начинается медленно, ускоряется к концу

  • ease-out

    Противоположность ease-in. Начинается быстро, замедляется к концу

  • ease-in-out

    Начинается медленно с ускорением посередине и замедлением к концу

transition-delay

По умолчанию переход начинается сразу. Но можно добавить задержку до начала перехода.

transition-delay: 2s;

Flexbox (12)

Flexbox позволяет располагать элементы в рядах или колонках.

Flexbox не позволяет располагать элементы одновременно в рядах и колонках.

С помощью Flexbox можно очень легко расположить элементы в контейнере в горизонтальном или вертикальном направлении

Flex свойства применяются либо к родительскому элементу, который называется Flex контейнером, либо к прямым дочерним элементам, которые называются Flex элементы.

Сойства родительского элемента упорядочивают дочерние элементы как группу в flexbox.

Свойства дочерних элементов управляют размерами дочерних элементов, их ростом и уменьшением.

Как создать Flex контейнер

Зададим свойство display элементу который станет flex контейнером

display: flex;

Как создать Flex элементы

Все вложенные в контейнер элементы первого уровня становятся flex элементами.

flex-direction

Определяет направление главной оси во флекс-контейнере

По умолчанию flex элементы располагаются горизонтально в ряд в начале контейнера.

Главная ось (main axis)

Главная ось это направление, в котором располагаются flex элементы. У главной оси есть начало (main-start) и конец (main-end)

Поперечная ось (cross axis)

Поперечная ось проходит перпендикулярно главной оси. У поперечной оси есть начало (cross-start) и конец (cross-end)

Свойство flex-direction может принимать одно из 2 значений

  1. flex-direction: row; (по-умолчанию)

  2. flex-direction: column;

justify-content

Выравнивает флекс-элементы по главной оси

Свойство justify-content может принимать одно из 5 значений

  1. justify-content: flex-start; (по-умолчанию)

    Выравнивает flex элементы от начала главной оси

  2. justify-content: flex-end;

    Выравнивает flex элементы от концу главной оси

  3. justify-content: center;

    Выравнивает flex элементы по центру главной оси

  4. justify-content: space-between;

    Выравнивает flex элементы по всей ширине контейнера с равными отступами между элементами

  5. justify-content: space-around;

    Одинаковые отступы между элементами и в два раза меньше отступы от начала и конца контейнера

  6. justify-content: space-evenly;

    Одинаковые отступы между элементами и от начала и конца контейнера

align-items

Выравнивает flex элементы по поперечной оси

Свойство align-items может принимать одно из 5 значений

  1. align-items: stretch; (по-умолчанию)

    Элементы растягиваются по поперечной оси и занимают всю высоту flex контейнера.

  2. align-items: flex-start;

    Элементы выравниваются по началу поперечной оси.

  3. align-items: flex-end;

    Элементы выравниваются по концу поперечной оси.

  4. align-items: center;

    Элементы выравниваются по центру поперечной оси.

flex-wrap

По-умолчанию flex элементы выходят за рамки flex контейнера, если они не помещаются в нем. Свойство flex-wrap разрешает или нет перенос в следующий ряд.

Свойство flex-wrap может принимать одно из 2 значений

  1. flex-wrap: no-wrap; (по-умолчанию)

  2. flex-wrap: wrap;

gap

Для Flex контейнера свойство gap задает минимальный промежуток между элементами. Для Grid контейнера свойство gap задает минимальный промежуток между рядами и колонками.

Можно записать одно значение

gap: 10px;

Для flex контейнера значение применится между рядами и между элементами. Для Grid контейнера значение применится между рядами и между колонками

Можно записать два значения

gap: 10px 20px;

Для flex контейнера первое значение применится между рядами, а второе между элементами. Для Grid контейнера первое значение применится между рядами, а второе между колонками

align-content

Выравнивает строки флекс-элементов по вертикали внутри флекс-контейнера

flex-grow

Flex элементы могут увеличиваться в ширину или высоту, если на главной оси есть дополнительное пространство.

Свойство flex-grow применяется к конкретным flex элементам и определяет, сколько дополнительного пространства они могут занимать.

Значение flex-grow это число, которое определяет коэффициент увеличения ширины флекс-элемента относительно других флекс-элементов

Flex элементы растут в ширину, если flex направление это ряд, и в высоту, если flex направление это столбец.

flex-grow: 0; (по-умолчанию)

Если есть свободное место, то элементы не увеличиваются в ширину.

flex-grow: 3;

Элемент будет расти в 3 раза быстрее, чем элементы с flex-grow: 1; при появлении свободного места.

flex-shrink

Flex элементы могут уменьшаться по ширине или высоте, если размер flex контейнера меньше, чам суммарный размер элементов на главной оси.

Свойство flex-shrink применяется к конкретным flex элементам и определяет, насколько каждый элемент может сжаться, чтобы вписаться в контейнер.

Значение flex-shrink это число, которое определяет коэффициент уменьшения ширины флекс-элемента относительно других флекс-элементов

Flex элементы сжимаются по ширине, если flex направление это ряд, и по высоте, если flex направление это столбец.

flex-shrink: 1; (по-умолчанию)

Если уменьшается контейнер, то элементы одинаково уменьшаются в ширину.

flex-shrink: 3;

Элемент будет сжиматься в 3 раза быстрее, чем элементы с flex-shrink: 1; если уменьшается контейнер.

align-self

Устанавливает выравнивание по высоте отдельно взятого флекс-элемента

Переопределяет свойство align-items контейнера

align-self: flex-start;

flex-basis

Начальный размер flex элемента до его увеличения и сжатия.

flex-basis: 150px;

Свойство flex-basis задает начальную ширину, если flex направление это ряд, и высоту, если flex направление это столбец.

Элементы могут становиться больше первоначального размера, занимая свободное место на главной оси, или уменьшаться, чтобы поместиться в flex контейнер.

Свойство flex-basis переопределяет свойство width или height в зависимости от направления flex-direction

flex

Универсальное свойство, которое позволяет указать параметры элемента, чтобы он эффективно заполнял доступное пространство.

flex-flow

Позволяет за одну декларацию задать направление главной оси и возможность переноса флекс-элементов на новую строку

order

Определяет порядок вывода флекс-элементов внутри флекс-контейнера

Grid

Grid это двумерная структура для организации элементов ваб страницы в ряды и колонки

Grid может содержать определенное количество рядов и колонок

Элементы сетки могут занимать несколько рядов и колонок

Grid позволяет создавать сложные макеты

Как и в flexbox, есть свойства сетки которые применяются только к родительскому элементу, а есть свойства, которые применяются только к дочерним элементам.

Свойства, которые применяются к родительскому элементу (контейнер сетки), определяют макет сетки (ряды и колонки).

Свойства, которвые применмяются к дочерним элементам (элементам сетки), контролируют, как определенные элементы размещаются в сетке.

Как создать Grid контейнер

display: grid;

Как создать Grid элементы

Все вложенные в контейнер элементы первого уровня становятся grid элементами.

Линии сетки - это горизонтальные и вертикальные линии, которые образуют сетку (как линии в таблице между рядами и столбцами).

Ячейки сетки создаются пересечением линий сетки. Элементы помещаются в ячейки.

Row track - это ряд с ячейкми

Column track - это столбец с ячейкми

Grid track - это пространство между двумя соседними линиями сетки

grid-template-rows

Определяет количество и размер рядов в сетке. Количество и размеры можно задать разными способами.

Создадим два ряда c фиксированными размерами

grid-template-rows: 100px 200px;

Создадим 2 ряда по 200px

grid-template-rows: repeat(2, 200px);

Создадим 3 ряда одинакового размера

grid-template-rows: 1fr 1fr 1fr;

grid-auto-rows

Определяет рамер автоматически генерируемых рядов. Ряды генерируются автоматически, если элементов сетки больше, чем ячеек в сетке

grid-auto-rows: 100px;

grid-template-columns

По-умолчанию у сетки только одна колонка. Свойство grid-template-columns определяет количество и размер колонок в сетке. Количество и размеры можно задать разными способами.

Создадим три колонки с фиксированными размерами

grid-template-columns: 150px 300px 150px;

Создадим 3 колонки по 150px

grid-template-columns: repeat(3, 150px);

Создадим 3 колонки с одинаковыми размерами

grid-template-columns: 1fr 1fr 1fr;

Создадим 3 колонки с комбинированными размерами

grid-template-columns: 150px 1fr 1fr;

grid-auto-columns

Определяет рамер автоматически генерируемых колонок. Колонки генерируются автоматически, если элементов сетки больше, чем ячеек в сетке

grid-auto-columns: 100px;

grid-template-areas

  "sidebar main main"

grid-row

По-умолчанию элементы занимают одну ячейку

Можно точно задать между какими линиями нужно расположить элемент.

Элемент займет два ряда между 1 и 3 линией

grid-row: 1 / 3;

Элемент займет три ряда после 1 линии

grid-column

По-умолчанию элементы занимают одну ячейку

Можно точно задать между какими линиями нужно расположить элемент.

Элемент займет 2 столбца между 2 и 4 линией

grid-column: 2 / 4;

Элемент займет 2 столбца после 2 линии

grid-column: 2 / span 2;

grid-area

Размещение элементов в зоне сетки, по ее метке

grid-area: header; grid-area: sidebar; grid-area: main; grid-area: footer;

justify-self
