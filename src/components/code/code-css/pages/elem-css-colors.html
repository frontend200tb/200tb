Формат цвета

HEX

Формат HEX записывается тремя шестнадцатиричными числами от 00 до FF, которые пишутся после знака #. Каждое число определяет красный, зеленый и синий цвета. Можно использовать сокращенную запись - по одной цифре каждого цвета если они одинаковые

Красный цвет

#FF0000 #F00

Зеленый цвет

#00FF00 #0F0

Синий цвет

#0000FF #00F

Черный цвет

#000000 #000

Белый цвет

#FFFFFF #FFF

Значения цветов HEX легко преобразуются в значения RGB с помощью преобразования трех шестнадцатеричных чисел в десятичные

RGB, RGBA

Формат RGB определяется тремя значениями для красного, зеленого и синего цвета в диапазоне от 0 до 255 (8 бит).

Красный цвет

rgb(255, 0, 0)

Зеленый цвет

rgb(0, 255, 0)

Синий цвет

rgb(0, 0, 255)

Черный цвет

rgb(0, 0, 0)

Белый цвет

rgb(255, 255, 255)

Значения цветов RGB легко преобразуются в значения HEX с помощью преобразования трех десятичных чисел в шестнадцатеричные

Формат RGBA это формат RGB плюс еще одно значение - прозрачность от 0 до 1

Полная прозрачность

rgb(255, 0, 0, 0)

Полупрозрачность

rgb(255, 0, 0, 0.5)

Полная непрозрачность

rgb(255, 0, 0, 1)
