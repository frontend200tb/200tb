Специфичность селекторов

Вес селектора складывается из трех цифр

  • 1000 Селекторы по идентификатору (id)
  • 100 Селекторы по классу (class), по псевдоклассу, по аттрибуту
  • 10 Селекторы по элементу, по псевдоэлементу

Список всех CSS комбинаторов

Комбинаторы позволяют объединить несколько селекторов для выбора элементов на основе их положения на веб-странице или отношения к другим элементам. Два селектора могут быть соединены путем добавления комбинатора между ними. Комбинаторы позволяют выбирать элементы более гибко.

  1. Комбинатор потомков

    div p Выбирает все элементы p, которые являются потомками любого уровня родительского элемента div

  2. Комбинатор прямых потомков

    div > p Выбирает все элементы p, которые являются прямыми потомками элемента div

  3. Комбинатор соседнего родства

    h1 + div Выбирает один элемент div который находится непосредственно после элемента h1 на одном уровне иерархии

  4. Комбинатор общего родства

    h1 ~ div Выбирает все элементы div которые находятся после элемента h1 и у которых общий родитель

Можно использовать несколько комбинаторов в одном правиле CSS

Сложные селекторы

Как читать и записывать сложные селекторы? Лучше всего читать справа налево.

Выберем все параграфы в первом элементе div который идет сразу после h1 на одном с ним уровне вложенности

h1 + div p

Запишем все ссылки a, которые находятся после элемента с атрибутом [data-menu], который идет непосредственно после тега h2, который является прямым потомком элемента с class="header", который вложен в тег с id="page"

#page .header > h2 + [data-menu] ~ a 
<div class="page">
  <header class="header">
    <h2>My page</h2>
    <p data-menu="">List of links</p>
    <a href="">link 1</a>
    <a href="">link 2</a>
    <a href="">link 3</a>
  </header>
</div>

Селектор по id

Селектор начинается с #, за которой идет id.

Увеличим шрифт элементу с id="heading" и сделаем его жирным

#heading {
  font-size: 20px;
  font-weight: 700;
}

Выбирает элемент с указанным атрибутом id, который может быть только один на странице. Элементы с id должны быть уникальные. Они могут быть на разных страницах, но не могут быть на одной.

Селектор по классу

Селектор начинается с точки, за которой идет имя класса. Выбирает все элементы одного класса class

Уберем со страницы все элементы с классом none и всем элементам с классом red зададим цвет фона красный

.none {
  display: none;
}

.red {
  background-color: red;
}

Селектор по псевдоклассу (:)

  1. :active

    Кнопка в нажатом состоянии

    button:active { border: 2px solid black };

  2. :any-link

    Любые элементы, которые могут иметь аттрибут href (например, a, area, link)

    :any-link { color: green };

  3. :checked

    Применяется к элементам, состояние которых меняется аттрибутом checked

    :checked { color: green };

  4. :default

    Применяется к элементам, у которых можно задать начальное состояние (например, input type="radio", input type="checkbox", option button)

    :default { border: none };

  5. :disabled

    Находит элементы формы по состоянию их аттрибута disabled. Кнопка в отключенном состоянии

    button:disabled { border: none };

  6. :dir

    Находит элементы по направлению текста в них

    span:dir { color: green };

  7. :empty

    Элементы, внутри которых нет других элементов или текста

    :empty { border: 1px solid black };

  8. :enabled

    Находит элементы формы по состоянию их аттрибута disabled. Кнопка в нормальном состоянии

    button:enabled { border: 1px solid black };

  9. :first-child

    Первый дочерний элемент своего родительского элемента

    li:first-child { color: green; }

  10. :focus

    Элемент в фокусе

    input:focus { color: lightblue; }

  11. :focus-within

    Элемент, внутри которого есть элемент в фокусе

    form:focus-within { background-color: lightblue; }

  12. :fullscreen

    Документ развернули на все окно с помощью javascript

    .player:fullscreen .controls { display: none; }

  13. :has

    Уточнает основной селектор дополнительным. Ссылки, которые содержат изображения

    a:has(img) { color: red; }

  14. :hover

    Наведение указателя мыши на элемент

    h1:hover { color: red; }

  15. :in-range

    Применяется для input у которого определены аттрибуты min и max и введенное значение соответствует этому диапазону

    input:in-range { color: red; }

  16. :invalid

    Когда данные в поле ввода не валидны

    input:invalid { color: red; }

    17. :is

    При группировке селекторов можно сократить и упростить написание

    :is(h1, h2, h3, h4) a { color: red; }

    Выберает дочерние элементы элемента с классом product, которые имеют классы title, price или sum

    .product :is(.title, .price, .sum) { color: red; }

  17. :lang

    Селектор по языку содержимого

    :lang(Array) { color: blue; }

  18. :last-child

    Последний дочерний элемент своего родительского элемента

    li:last-child { color: blue; }

  19. :link

    Непосещенные ссылки

    :link { color: blue; }

  20. :not(.default)

    Псевокласс отрицания. Выбирем элементы h1, у которых нет класса с именем default

    h1:not(.default) { color: darkmagenta; }

  21. :nth-child(2)

    Второй дочерний элемент своего родительского элемента

    li:nth-child(2) { color: red; }

  22. :nth-child(even)

    Все четные элементы своего родительского элемента

    tr:nth-child(even) { background-color: gray; }

  23. :nth-child(odd)

    Все нечетные элементы своего родительского элемента

    tr:nth-child(odd) { background-color: white; }

  24. :optional

    Находит любой input у которого не установлен аттрибут required

    input:optional { color: blue; }

  25. :out-of-range

    Применяется для input у которого определены аттрибуты min и max и введенное значение не соответствует этому диапазону

    input:out-of-range { color: blue; }

  26. :read-only

    Элементы недоступные для редактирования

  27. :read-write

    Элементы доступные для редактирования

  28. :required

    Находит любой input у которого есть аттрибут required

    input:required { color: blue; }

  29. :root

    Корневой тег документа. Для HTML это html, для SVG это svg.

  30. :target

    При переходе по ссылке внутри страницы

  31. :valid

    Когда данные в поле ввода валидны

    input:valid { color: green; }

  32. :visited

    Посещенные ссылки

    :visited { color: orange; }

Селектор атрибутов

Селектор заключается в квадратные скобки. Выбирает все элементы с одинаковым атрибутом

Изменим цвет фона всем элементам с атрибутом data-category на зеленый

[data-category] {
  background-color: green;  
}

Селектор атрибутов по значению

Селектор заключается в квадратные скобки. Выбирает все элементы с одинаковым значением указанного атрибута

Изменим цвет фона всем элементам с атрибутом data-category='main' на зеленый

[data-category='main'] {
  background-color: green;  
}

Селектор атрибутов чувствителен к регистру. Но можно указать параметр i, что сделает значение атрибута нечувсвительным к регистру

[data-category='main' i] {
  background-color: green;  
}

Селектор по элементу

Селектор с именем тега. Выбирает все элементы одного типа.

Обычно используют для стилизации всех ссылок. Уберем со всех ссылок подчеркивание и поставим заголовки h1 по центру.

a {
  text-decoration: none;
}

h1 {
  text-align: center;
}

Селектор по псевдоэлементу (::)

  1. ::after

    Располагается после элемента. Добавим текст 'пока' после текста в span

    span::after { content: 'пока' }

  2. ::before

    Располагается перед элементом. Добавим текст 'привет' перед текстом в span

    span::before { content: 'привет' }

  3. ::first-letter

    Стилизация первой буквы. Для всех элементов p первая буква станет зеленая.

    p::first-letter { color: green; }

  4. ::first-line

    Первая строка. Для всех элементов p первая строка станет зеленая.

    p::first-line { color: green; }

  5. ::marker

    Стилизация маркеров в списке. Во всех списках маркеры станут зеленые и примут вид стрелки =>.

    li::marker { color: green; content: '\21D2' }

  6. ::placeholder

    Стилизация плейсхолдера в текстовом поле ввода. Во всех текстовых полях ввода плейсхолдер станет зеленый и курсивный.

    input::placeholder { color: green; font-style: italic }

  7. ::selection

    Стилизация выделенного текста. При выделении мышью, текст станет зеленый и курсивный.

    input::selection { color: green; font-style: italic }

Универсальный селектор

Селектор *. Выбирает все элементы на странице.

Обычно используют для обнуления отступов и изменения box модели на всей странице.

* {}
  margin: 0;
  padding: 0;
  box-sizong: border-box;

Групповой селектор

В одном правиле CSS можно использовать несколько селекторов, разделяя их запятыми

Увеличим шрифт элементам с id="heading", заголовкам h2 и параграфам и сделаем их жирными

#heading,
h2,
p {
  font-size: 20px;
  font-weight: 700;
}

Комбинатор потомков

Пробел между селекторами - это комбинатор потомков. Два селектора с пробелом между ними. div p Выбирает все элементы p, которые являются потомками любого уровня родительского элемента div

Изменим цвет текста на красный всем параграфам, которые вложены в div на любом уровне

div p {
  color: red;
}

Комбинатор прямых потомков

Знак > между селекторами - это комбинатор прямых потомков. Два селектора со знаком > между ними. div > p Выбирает все элементы p, которые являются прямыми потомками элемента div

Изменим фон на зеленый всем параграфам, которые вложены в div на первом уровне

div > p {
  background-color: green;
}

Комбинатор соседнего родства

Знак + между селекторами - это комбинатор соседнего родства. Два селектора со знаком + между ними. h1 + div Выбирает элемент div который находится непосредственно после элемента h1 на одном уровне иерархии

Изменим размер шрифта в блоке div который идет сразу после заголовка h1

h1 + div {
  font-size: 20px;
}

Комбинатор общего родства

Знак ~ между селекторами - это комбинатор общего родства. Два селектора со знаком ~ между ними. h1 ~ div Выбирает все элементы div которые находятся после элемента h1 и у которых общий родитель

Изменим толщину шрифта во всех блоках div которые идут после заголовка h1 на одном уровне вложенности

h1 ~ div {
  font-weight: 700;
}
