Паскаль. Урок 0. Настройка среды разработки

Установим среду разработки (IDE) для языка Паскаль.

Идем на страницу lazarus-ide.org и скачиваем программу по ссылке Windows (32 and 64 Bits) Direct download

Устанавливаем Lazarus из скаченного файла, соглашаемся со всеми пунктами по-умолчанию.

При первом запуске Lazarus работает в многооконном режиме и не понимает русских букв. Исправим это.

Переключение Lazarus в однооконный режим

Откроем меню Пакет - Установить/удаленить пакеты.

В окошке Доступные для установки выбираем пакет anchordockingdsgn 1.0. Нажимаем кнопку Установить выбранное.

Нажимаем кнопку Сохранить и перезапустить IDE.

В открывшемся окне Подтвердите новый набор пакетов для IDE нажимаем кнопку Продолжить.

Начнется компиляция пакета AnchorDockingDsgn 1.0, потом Сборка IDE.

После чего Lazarus перезапустится в однооконном режиме.

Как в Lazarus писать на русском

В окне Редактор исходного кода нажимаем правой кнопкой мыши. Выбираем Параметры файла - Кодировка - UTF-8 с BOM

Проверка запуска тестовой программы

Запустим Lazarus и создадим консольное приложение из главного меню.

Создание консольного приложения

Открываем Файл - Создать - Проект - Программа

Необходимо убрать лишний код и заменить его на

program hello;
begin
  writeln('hello world!');
  readln();
end.

Запуск программы в Lazarus

Нажимаем зеленый треугольник (похож на кнопку плей, рядом с кнопками пауза и стоп) или нажимаем клавишу F9.

Откроется консоль с надписью hello world!. Чтобы закрыть окно консоли надо нажать Enter или нажать крестик на окне консоли, или нажать кнопку стоп в Lazarus или нажать CTRL + F2.

Паскаль. Урок 1. Минимальная программа. Тело программы и операторные скобки. Комментарии в коде

Минимальный код программы

Убираем из программы лишний код. Оставляем только минимальный код программы

program project1;
begin
end.

В первой строчке после слова program пишется имя программы.

Затем пишем операторные скобки
begin - открывающая операторная скобка
end. - закрывающая операторная скобка

Тело программы

Тело программы пишут между операторными скобками

program project1;
begin // открывающая операторная скобка
// тело программы
end. // закрывающая операторная скобка

Первая программа на Паскаль

Создадим нашу первую программу program hello;
begin
  writeln('hello world!');
  readln();
end.

Комментарии

Комментарии нужны чтобы пояснять для чего нужны кусочки кода. А также для того чтобы не выполнять косочки кода. Комментарии бывают однострочные и многострочные.

Однострочные комментарии

Однострочные комментарии пишутся в любой строке после знака //. Начиная со знака // и до конца строки это комментарий.

a:= 5; // переменная изменила значение

Многострочные комментарии

Многострочные комментарии пишутся в фигурных скобках. При этом они могут занимать несколько строк, одну строку или часть одной строки.

{ комментарий на одной строке }
{
комментарий
на
несольких
строках
}
a:= 5; {комментарий занимает часть строки}

Паскаль. Урок 2. Литералы и Вывод данных на экран (в консоль). Первое знакомство с Write и Writeln

Литералы - это данные записанные непосредственно в программном коде. Каждый литерал соответствует какому-то типу данных. Рассмотрим строковые, символьные и числовые литералы.

Строковые литералы

Строковые литералы это любые символы записанные в одинарных кавычках. Строковый литерал всегда пишется на одной строке кода. Перенос на другую строку внутри литерала приведет к ошибке при запуске программы. Пробел это тоже символ, поэтому из пробелов тоже можно сделать строковый литерал.

'text'
'12234'
'qw!@#$1567'
' '

Символьные литералы

Символьные литералы любой один символ записанный в одинарные кавычки. Пробел в одинарных кавычках это тоже символьный литерал.

'a'
'A'
'5'
'#'
' '

Числовые литералы

Числовые литералы можно разделить на целочисленые и вещественные. Целочисленные - это положительные и отрицательные целые числа, ограниченные максимальным и минимальным числом и ноль. Вещественные числа это числа с точкой, отделяющей целую часть числа от десятичной.

0
-11
125
55.678
-123.456

Процедуры write() и writeln()

Процедуры write() и writeln() выводят данные, переданные им как аргументы в консоль.

writeln() после вывода данных переносит курсор на новую строку.

writeln() вызванная без аргументов ничего не выводит, но все равно переносит курсор на новую строку.

readln()

readln() ждет ввода любых данных в консоль от пользователя и завершает свою работу при нажатии ENTER

Паскаль. Урок 3. Типы данных в языке Pascal. Объявление переменных. Целые числа integer. Оператор присваивания. Инициализация в Паскаль

Представления данных

Данные в программе могут быть представлены в 4 видах:

  • литерал
  • переменная
  • результат выражения
  • вызов функции (значение, которое возвращает функция)

У каждого данного всегда есть только один тип данных.

Типы данных

В Паскале выделяют 5 типов данных:

  • integer -- целые числа.
  • real -- вещественные числа.
  • boolean -- логический тип данных
  • char -- символьный тип.
  • string -- строковый тип

Объявление переменных

Объявление переменных - это уведомление компилятора Паскаль о том, что в вашей программе будут использоваться переменные с указанными именами, каждая из которых имеет определённый (указанный в секции объявления) тип.

Объявление переменных происходит после ключевого слова var, до тела программы

var i,j: integer;
r: real;
b: boolean;
s: string;
c: char;

Оператор присваивания

Оператор присваивания := используется чтобы переменной присвоить значение.

a := 5; // переменной a присваивается значение литерала
b := a; // переменной b присваивается значение другой переменной
c := a + b; // переменной c присваивается результат выражения

Инициализация переменной

Инициализация переменной - это первое присваивание какого-то значения переменной.
