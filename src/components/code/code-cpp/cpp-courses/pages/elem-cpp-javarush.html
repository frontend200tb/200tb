C++ — язык системного уровня и реальных технологий

C++ — это один из самых мощных и востребованных языков программирования в мире. Он используется десятилетиями и лежит в основе огромного количества технологий, которыми мы пользуемся каждый день.

Версия C++23 — это современный стандарт, вобравший в себя всё лучшее из предыдущих поколений языка. Это уже не тот «страшный C++ из 90-х». Это язык с мощной стандартной библиотекой, удобными абстракциями, поддержкой функционального и объектно-ориентированного стиля, а также с современными возможностями, которые делают код чище и безопаснее.

C++23 добавил новые инструменты, упрощающие работу с диапазонами (ranges), обработкой ошибок (std::expected), улучшил constexpr, добавил новые адаптеры и сделал стандартную библиотеку ещё более выразительной. Это язык, который постоянно развивается, оставаясь при этом фундаментом индустрии.

Команды и первая программа на C++

1. Знакомство с командами

Если когда-то вы представляли работу программиста как что-то непостижимое и таинственное, пришла пора развенчать этот миф! Программирование — это не секретное искусство, а шикарная и очень интересная работа. И сейчас вы в этом убедитесь.

Компьютерные программы пишут при помощи языков программирования — особых правил и слов, понятных компьютеру. Сегодня вы познакомитесь с языком C++: напишете свою первую программу, разберете, что такое команды, и заставите компьютер слушаться вас (насколько он захочет, конечно).

Программа — это набор (список) команд. Сначала выполняется первая команда, затем вторая, третья, и так далее. Когда все команды выполнены, программа завершается.

Какие именно команды могут быть в списке команд, зависит от того, кто их выполняет, а точнее, от того, какие команды знает (и понимает) исполнитель. Собаке можно дать команду «Сидеть», «Голос», кошке — «Брысь», человеку — «Стой!», ну а роботу — «Работай!».

Команда — это простое указание компьютеру: сделай то-то! Самое главное — компьютер выполняет именно то, что вы ему сказали, а не то, что вы хотите. Пока искусственный интеллект не подрастет, конечно.

Набор команд языка C++ довольно обширен. Например, этой командой можно вывести на экран надпись "Робот — друг человека":

std::cout << "Робот — друг человека" << '\n';

Вывод текста в консоль с помощью std::cout

Но мы начнем не с команд, а с пары простых принципов. Знание нескольких принципов заменяет знание многих фактов.

Принцип первый: в языке программирования C++ каждую команду принято писать с новой строки. В конце команды ставится точка с запятой ;.

Допустим вы хотите, чтобы компьютер вывел вашу фразу три раза, дайте ему команду несколько раз.

Программа из трех команд:

std::cout << "Робот — друг человека" << '\n'; std::cout << "Робот — друг человека" << '\n'; std::cout << "Робот — друг человека" << '\n';

Три команды подряд (выполняются сверху вниз)

В результате компьютер выведет на экран фразу 3 раза. Все просто.

Принцип второй: программа не может просто состоять из команд. Команды языка C++ должны находиться внутри функций, а функции — внутри файлов. Это очень интересно - давайте подробнее.

2. Структура типичной программы

Любая программа — это набор команд, но компьютеру нужно знать, с какой команды начать. В C++ точка входа почти всегда одна: функция main. Это как кнопка «Старт» у микроволновки: можно долго выбирать режимы, но пока не нажал — ничего не происходит.

Минимальная программа - это один файл, который содержит одну фукнцию - main() и выглядит вот так:

int main() {
  return 0;
}

Здесь есть три важных элемента: слово int, имя main и блок { ... }. Давайте разберём каждый — спокойно и без магии.

Слово int — это тип результата функции. Пока не пугайтесь слова «тип»: сейчас нам важно только одно — main должен вернуть какое-то число, и это число называют кодом завершения. Мы будет всегда возвращать 0.

Круглые скобки () означают, что это функция. Они обязательно нужны после слова main, иначе C++ не догадается что это та самая функция.

Фигурные скобки { ... } задают блок кода — контейнер для команд, которые выполняются последовательно. Для main блок обязателен: это буквально «тело программы».

Полезная привычка на старте — всегда оформлять main так, чтобы скобки стояли аккуратно и читались без напряга:

int main() {
  команда1;
  команда2;
  команда3;
  return 0;
}

Шаблон: команды внутри тела функции main()

Смысл такой: всё, что внутри {} — выполняется

3. Подключаем функции

Если вы еще не умеете писать свои функции, то можете пользоваться чужими. В C++ есть большая библиотека функций на все случаи жизни - она называется STD (от слова standard).

Допустим вы хотите чтобы ваша программа что-то вывела на экран, тогда вам нужно подключить в вашу программу библиотеку iostream, и вы сможете пользоваться ее функциями.

Выглядит это в коде программы примерно так:

#include <iostream>

int main() {
  return 0;
}

Подключение заголовка iostream

Тут все очень просто: #include означает включить в программу функции, которые есть в библиотеке. <iostream> - это имя библиотеки в угловых скобках <...>.

Никакой магии. Магия появится потом.
