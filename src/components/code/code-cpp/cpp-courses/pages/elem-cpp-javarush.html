Лекция 1. Команды и первая программа на C++

1. Знакомство с командами

Если когда-то вы представляли работу программиста как что-то непостижимое и таинственное, пришла пора развенчать этот миф! Программирование — это не секретное искусство, а шикарная и очень интересная работа. И сейчас вы в этом убедитесь.

Компьютерные программы пишут при помощи языков программирования — особых правил и слов, понятных компьютеру. Сегодня вы познакомитесь с языком C++: напишете свою первую программу, разберете, что такое команды, и заставите компьютер слушаться вас (насколько он захочет, конечно).

Программа — это набор (список) команд. Сначала выполняется первая команда, затем вторая, третья, и так далее. Когда все команды выполнены, программа завершается.

Какие именно команды могут быть в списке команд, зависит от того, кто их выполняет, а точнее, от того, какие команды знает (и понимает) исполнитель. Собаке можно дать команду «Сидеть», «Голос», кошке — «Брысь», человеку — «Стой!», ну а роботу — «Работай!».

Команда — это простое указание компьютеру: сделай то-то! Самое главное — компьютер выполняет именно то, что вы ему сказали, а не то, что вы хотите. Пока искусственный интеллект не подрастет, конечно.

Набор команд языка C++ довольно обширен. Например, этой командой можно вывести на экран надпись "Робот — друг человека":

std::cout << "Робот — друг человека" << '\n';

Вывод текста в консоль с помощью std::cout

Но мы начнем не с команд, а с пары простых принципов. Знание нескольких принципов заменяет знание многих фактов.

Принцип первый: в языке программирования C++ каждую команду принято писать с новой строки. В конце команды ставится точка с запятой ;.

Допустим вы хотите, чтобы компьютер вывел вашу фразу три раза, дайте ему команду несколько раз.

Программа из трех команд:

std::cout << "Робот — друг человека" << '\n'; std::cout << "Робот — друг человека" << '\n'; std::cout << "Робот — друг человека" << '\n';

Три команды подряд (выполняются сверху вниз)

В результате компьютер выведет на экран фразу 3 раза. Все просто.

Принцип второй: программа не может просто состоять из команд. Команды языка C++ должны находиться внутри функций, а функции — внутри файлов. Это очень интересно - давайте подробнее.

2. Структура типичной программы

Любая программа — это набор команд, но компьютеру нужно знать, с какой команды начать. В C++ точка входа почти всегда одна: функция main. Это как кнопка «Старт» у микроволновки: можно долго выбирать режимы, но пока не нажал — ничего не происходит.

Минимальная программа - это один файл, который содержит одну фукнцию - main() и выглядит вот так:

int main() {
  return 0;
}

Здесь есть три важных элемента: слово int, имя main и блок { ... }. Давайте разберём каждый — спокойно и без магии.

Слово int — это тип результата функции. Пока не пугайтесь слова «тип»: сейчас нам важно только одно — main должен вернуть какое-то число, и это число называют кодом завершения. Мы будет всегда возвращать 0.

Круглые скобки () означают, что это функция. Они обязательно нужны после слова main, иначе C++ не догадается что это та самая функция.

Фигурные скобки { ... } задают блок кода — контейнер для команд, которые выполняются последовательно. Для main блок обязателен: это буквально «тело программы».

Полезная привычка на старте — всегда оформлять main так, чтобы скобки стояли аккуратно и читались без напряга:

int main() {
  команда1;
  команда2;
  команда3;
  return 0;
}

Шаблон: команды внутри тела функции main()

Смысл такой: всё, что внутри {} — выполняется

3. Подключаем функции

Если вы еще не умеете писать свои функции, то можете пользоваться чужими. В C++ есть большая библиотека функций на все случаи жизни - она называется STD (от слова standard).

Допустим вы хотите чтобы ваша программа что-то вывела на экран, тогда вам нужно подключить в вашу программу библиотеку iostream, и вы сможете пользоваться ее функциями.

Выглядит это в коде программы примерно так:

#include <iostream>

int main() {
  return 0;
}

Подключение заголовка iostream

Тут все очень просто: #include означает включить в программу функции, которые есть в библиотеке. <iostream> - это имя библиотеки в угловых скобках <...>.

Никакой магии. Магия появится потом.

Лекция 2. Вывод на экран в C++: std::cout и перевод строки

1. Знакомство с командой std::cout

Вы только что написали свою первую программу на C++ и вывели на экран первое сообщение. Поздравляю вас с этим. Давайте же не останавливаться...

Любая программа на C++ состоит из команд (инструкций). Команды бывают разные. В языке C++ есть команды на все случаи жизни. Каждая команда описывает какое-то определенное действие. В конце каждой команды ставится точка с запятой.

Для вывода на экран в C++ обычно используют поток вывода std::cout.

Примеры команд:

std::cout << 1 << '\n'; // Выводит на экран число: 1 std::cout << "Amigo" << '\n'; // Выводит на экран надпись: Amigo std::cout << "Risha & Amigo" << '\n'; // Выводит на экран надпись: Risha & Amigo

Здесь используется объект std::cout и оператор << (читается как «вывести в поток»). Чтобы завершить строку, обычно добавляют перевод строки: '\n'.

Размер букв имеет значение

В C++ размер имеет значение: важно, большими или маленькими буквами написаны имена. std::cout работать будет, а std::Cout — нет. Если вы напишете что-то не теми буквами, компилятор C++ это не распознает, и следовательно выполнять не будет. Так что будьте внимательными.

Двойные кавычки

Если вы хотите вывести на экран текст, его нужно с двух сторон обозначить двойными кавычками.

Одинарная кавычка выглядит вот так ', а двойная — вот так ". Двойная кавычка — это не две одинарных: просьба не путать.

Двойная — это та, которая рядом с кнопкой Enter. На ней еще обычно находится кириллическая буква "Э".

2. Знакомство с выводом без перевода строки

Если вам нужно вывести текст без перехода на новую строку, в C++ просто не добавляйте '\n' в конце.

Команда с '\n' выводит текст и после него добавляет перевод курсора на следующую строку, как бы нажимая Enter. Таким образом следующая команда вывода начнёт писать уже на новой строке.

В отличие от этого, вывод без '\n' не добавляет Enter после вывода текста. Поэтому текст, выведенный несколькими такими командами, будет выводиться в одну строчку.

Пример 1:

std::cout << "Amigo";
std::cout << "The" << '\n';
std::cout << "Best";

Смешанный вывод: часть без перевода строки, часть — с переводом

Выведет на экран:

AmigoThe
Best

Пример 2:

std::cout << "Amigo";
std::cout << "The";
std::cout << "Best";

Вывод без перевода строки — всё «склеится» в одну линию

Выведет на экран:

AmigoTheBest

Если хотите вывести текст на одной строке — не добавляйте '\n', если на нескольких — добавляйте. Все просто.

3. Разбираем вывод через std::cout

Если вы хотя бы немного знаете английский, то изучать программирование вам будет намного проще. Название многих вещей в C++ напрямую связанное с тем, что они делают.

Например cout — это сокращение от console output — «вывод в консоль».

std::cout — это один из самых распространённых способов взаимодействия программы и человека. Когда пользователь пишет программе команды на клавиатуре, а она в ответ выводит ему текст на экран, то программисты говорят, что программа работает с консолью.

Другими словами, работа с консолью — это вывод текста на экран и ввод данных с помощью клавиатуры. Поэтому если вы в будущем услышите, что программа что-то вывела в консоль, то это просто значит, что она вывела текст на экран.

А переход на новую строку в таких примерах мы делаем явно, добавляя '\n' (символ перевода строки).

Мини‑шпаргалка по выводу

Когда вы только начинаете, полезно держать в голове несколько “микро‑правил”, которые постоянно всплывают в задачах. Вместо списка сложим это в маленькую таблицу‑напоминалку.

Вывести текст. строка — в двойных кавычках

std::cout << "Hi";

Вывести число. число — без кавычек

std::cout << 123;

Вывести один символ. символ — в одинарных кавычках

std::cout << 'A';

Перейти на новую строку. '\n' — один символ перевода строки

std::cout << '\n';

Вывести “сложную” строку из частей. цепочка << печатает подряд

std::cout << "X=" << 10 << '\n';

Это всё кажется примитивным, но именно на таких “примитивах” чаще всего и спотыкаются: забыли кавычку, перепутали ' и ", забыли перевод строки, не добавили пробел.

1. Переменные и коробки

Переменные — это такие специальные штуки для хранения данных. Любых данных. Почти все данные в C++ так или иначе оказываются в переменных. Больше всего переменная по смыслу напоминает коробку: самую обычную коробку.

Допустим, вы написали на листке число 13 и положили этот лист в коробку. Теперь мы можем говорить, что «коробка хранит значение 13».

Каждая переменная в C++ имеет три важных свойства: тип, имя и значение.

Имя используется для того, чтобы отличать одну переменную от другой. Это как надпись на коробке.

Тип переменной определяет тип значений/данных, которые в ней можно хранить. В коробке для торта храним торт, в коробке для обуви — туфли, и т.д.

Значение — это некий объект, данные или информация, которая хранится в переменной.

Каждый объект в языке C++ имеет свой тип. Например, могут быть такие типы данных: целое число, дробное число, текст, тип Cat, тип House и т.д.

У каждой переменной (коробки) тоже есть свой тип. Переменная может хранить значения только того же типа, что и она сама. Разные коробки используются для хранения различных вещей: коробка конфет, коробка для десятка яиц и т.п. Все как в жизни.

2. Создание переменной

В языке C++ для того, чтобы создать переменную, используется команда вида:

тип имя;

Где тип — это тип переменной (совпадает с типом значений, которые переменная сможет хранить). А имя — это имя переменной.

Примеры:

int a;

Создаем переменную по имени a типа int.

std::string s;

Создаем переменную по имени s типа std::string.

double c;

Создаем переменную по имени c типа double.

Два самых часто используемых типа — это целые числа (обозначаются словом int) и текст (в C++ обычно используют std::string). Также популярен тип double — это дробные (вещественные) числа.

3. Присваивание

Как уже говорилось выше, у переменной есть имя, тип и значение. Имя и тип мы уже разобрали, а что насчет значения? Как занести значение в переменную?

Чтобы занести значение в переменную, существует специальная операция — операция присваивания. Она копирует значение из одной переменной в другую. Не переносит, а именно копирует. Как файл на диске. Выглядит это присваивание так:

имя = значение;

Где имя — это имя переменной, а значение — это значение, которое будет занесено в переменную. В качестве значения может фигурировать конкретное значение, имя другой переменной или даже какое-нибудь выражение с использованием переменных.

Примеры:

int main() {
  int i;         // Создается переменная i
  int a, b;      // Создаются переменные a, b
  int x;         // Создается переменная x

  i = 3;         // В переменную i заносится значение 3.

  a = 1;         // В переменную a заносится значение 1.
  b = a + 1;     // В переменную b заносится значение 2.

  x = 3;         // В переменную x заносится значение 3.
  x = x + 1;     // В этой строке значение x увеличивается на 1, x теперь равен 4.
}

Символ равно

Для операции присваивания используется символ равно =. Это не сравнение, не равенство. Это именно команда копирования значения справа от знака равно в переменную, которая слева. Для сравнения в языке C++ используется двойное равно ==.

4. Имя переменной: что можно, а что нельзя

Вот какие факты вам полезно будет знать при создании переменных:

  • Имя переменной может быть любым (например, temperature, score, userAge).
  • Но есть ограничения — оно не может начинаться с цифры, и не может совпадать со служебными словами C++ (например, нельзя назвать переменную int или if).
  • Не может содержать специальные символы, кроме _.
  • В имени важна разница между строчными и прописными буквами (Age и age — разные переменные).
  • Обычно используют стиль camelCase: первая буква маленькая, дальше — каждое слово с заглавной (userAge, maxScore).

Примеры разрешенных имен:

#include 

int main() {
  int x123 = 1;
  std::string koteyka;
  double PI = 3.14;
  std::string MAIN_PATH = "c:/";
}

Примеры запрещенных имен:

int 1first = 1;              // имя не может начинаться с цифры
int number# = 25;            // символ # нельзя использовать в имени
std::string name" = "John";  // кавычки в имени недопустимы
double pi+e = 5.123';        // имя не может содержать +

6. Краткая история C++

Чтобы понять некоторые вещи в C++, нужно немного вернуться назад — к языку C.

Язык C появился в начале 1970-х годов в Bell Labs. Его создал Деннис Ритчи. Это был компактный, быстрый и очень практичный язык, который позволял писать системные программы и даже операционные системы. Он давал программисту много контроля над памятью и устройством программы. Именно поэтому C стал фундаментом для огромного количества технологий.

В начале 1980-х Бьёрн Страуструп решил расширить C и добавить в него новые возможности, прежде всего поддержку классов. Сначала язык назывался “C with Classes”, а позже получил имя C++. Идея была простой: сохранить эффективность C, но сделать язык удобнее для создания больших и сложных программ.

С тех пор C++ развивался почти 50 лет. Он регулярно обновляется, и каждая новая версия стандарта добавляет новые возможности. Сегодня существуют стандарты C++11, C++14, C++17, C++20, C++23 — и работа над следующими версиями уже идёт. В результате язык стал очень мощным и многогранным.

Однако в C++ нередко можно встретить несколько способов сделать одно и то же. Например, даже инициализировать переменную можно по-разному:

int a = 5;
int b(5);
int c{5};

Все три варианта корректны — просто появились они в разные периоды развития языка. Похожая история и со строками: есть разные формы создания, разные способы объединения, форматирования и преобразования значений. Это следствие долгой эволюции языка.

Важно понимать: C++ не переписывался заново каждые 10 лет. Он аккуратно расширялся, сохраняя совместимость со старым кодом. Именно поэтому в нём накопилось так много возможностей.

Но это не значит, что нам нужно изучать всё подряд. Мы будем использовать самые современные и актуальные стандарты — C++20 и C++23. Мы будем писать код в современном стиле, выбирать более безопасные и понятные конструкции и постепенно разбираться в языке без перегруза историческими деталями.

1. Объявление переменных

Давайте еще раз вспомним, как создавать переменные. Чтобы создать переменную, нужно написать такую команду: тип имя;.

И опять напоминаю, что в языке C++ играет роль, какие буквы вы пишете — большие или маленькие. int a — это не то же самое, что и int A.

Кстати, в C++ можно одновременно создавать переменную и присваивать ей значение. Это экономит время и место:

2. Тип int

В переменной типа int можно хранить целые числа: 5, -10, 100500. Также над типом int можно выполнять различные арифметические операции: сложение, вычитание, умножение, деление и другие.

Примеры:

int main() {
  {
    int x = 1;                  // x будет равен 1
    int y = x * 2;              // y будет равен 2
    int z = 5 * y * y + 2 * y + 3; // z будет равен 20+4+3, будет равен 27
  }

  {
    int a = 5;                  // a будет равно 5
    int b = 1;                  // b будет равно 1
    int c = (a - b) * (a + b);  // c будет равно 4*6, будет равно 24
  }

  {
    int a = 64;                 // a будет равно 64
    int b = a / 8;              // b будет равно 8
    int c = b / 4;              // c будет равно 2
    int d = c * 3;              // d будет равно 6
  }
}

Если вы в школе не прогуливали математику, тот тут вам все должно быть понятно

А если прогуливали, то вот вам список разрешенных операций:

+ — сложение
- — вычитание
* — умножение
/ — деление
% — остаток от деления

3. Тип double

Тип int хранит только целые числа. А что если нам нужны числа с дробной частью? Для этого существует тип double.

Он хранит вещественные числа:

double x = 3.14;
double y = -0.5;
double z = 2.0;

Особенность double

Числа типа double хранятся в памяти приближённо. Иногда могут появляться неожиданные эффекты:

double x = 0.1 + 0.2; // 0.300000004

Вы можете ожидать ровно 0.3, но внутри хранится приближённое значение. Это нормально. Так работает представление дробных чисел в компьютере.

4. Тип std::string

Тип std::string позволяет хранить текстовые строки. А чтобы задать какую-то текстовую строку, в C++ надо написать текст строки, а с обеих сторон поставить двойные кавычки.

std::string s = "Amigo";

s будет содержать текст Amigo

std::string s = "123";

s будет содержать текст 123

std::string s = "Bond 007";

s будет содержать текст Bond 007

Выглядит несложно, правда? Ну если так, тогда вот еще один интересный факт.

Строки в C++ можно склеивать с помощью знака плюс — +. Пример:

#include <string>

int main() {
  std::string s1 = std::string("Amigo") + " the best"; // s будет содержать Amigo the best

  std::string s2 = ""; // s будет содержать пустую строку — вообще без символов.

  int x = 333;
  std::string s3 = std::string("Amigo") + std::to_string(x); // s будет содержать Amigo333
}

Обратите внимание на последний пример: мы сложили строку и число. Тут тоже все просто: число будет преобразовано в строку, затем две строки будут склеены. При сложении строк и чисел всегда получается строка.

5. Вывод переменной на экран

Вроде все так очевидно и элементарно. Ну тогда, может, вы сразу догадаетесь, какой командой можно вывести переменную на экран?

На самом деле все просто. Чтобы вывести что-то на экран, мы используем std::cout, куда передаем то, что мы выводим. Для перевода строки удобно добавить '\n'.

#include <iostream>
#include <string>

int main() {
  std::cout << "Amigo\n";              // Amigo

  std::cout << "Ami" << "go" << '\n';  // Amigo

  std::string s1 = "Amigo";
  std::cout << s1 << '\n';          	// Amigo

  std::string s2 = "Am";
  std::cout << s2 << "igo" << '\n'; 	// Amigo
}

Надеюсь, теперь стало немного понятнее. А правильно вы все поняли или нет, мы сейчас проверим. Практика — критерий истины: только с помощью практики можно проверить, хорошо ли вы во всем разобрались.

6. Сокращённые операции присваивания (+=, -=, *=, /=)

В C++ есть удобная форма записи операций вида:
