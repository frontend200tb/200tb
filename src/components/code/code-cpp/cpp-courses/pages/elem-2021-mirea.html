02.10.2021 Занятие 1. Основы C++ 1

Мы будем программировать в программе CodeBlocks 20.03. Любая программа на C++ всегда начинается с функции main(). Напишем первую программу - вывод приветствия.

#include 

using namespace std;

int main() {
  cout << "Hello world";
  return 0;
}

Выведем приветствие в две строки

#include 

using namespace std;

int main() {
  cout << "Hello world" << endl;
  cout << "Hello again";
  return 0;
}

Выведем текст, число и математическое выражение

#include 

using namespace std;

int main() {
  cout << "2 + 2" << endl;
  cout << 2 << endl;
  cout << 2 + 2 << endl;
  return 0;
}

В спортивном программировании для перевода строки используется символ '\n', а не endl. Потому что endl кроме перевода строки еще сбрасывает буфер, а это лишние операции, на которые тратится время.

Попробуем подключить русский язык. Сам текст программы, содержащий русский язык, должен быть сохранен в кодировке windows-1251.

#include 

using namespace std;

int main() {
  setlocale(LC_ALL, "Russian");
  cout << "Привет";
  return 0;
}
