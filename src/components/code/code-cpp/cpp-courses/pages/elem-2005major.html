1 Installation

Для прохождения курса будем использовать Microsoft Visual C++ 2005 Express Edition. Это IDE, которая позволяет создавать проекты.

Откроем Visual C++ и создадим проект

  1. Выберем пункт меню

    File - New - Project Ctrl+Shift+N

  2. В открывшемся окне New Project на вкладке Project types выберем General, а на вкладке Templates выберем Empty Project

    В поле Name напишем имя проекта first

    В поле Location можно выбрать расположение проекта.

    Нажимаем кнопку OK

  3. В окне Solution Explorer появится наш проект first

  4. В нашем проекте first появились три папки Header Files, Resource Files, Source Files. Header Files для заголовочных файлов. Source Files для исходных файлов на языке C++. Resource Files для файлов ресурсов, которые относятся только к Windows платформе и нас в данном курсе не интересует.

Создадим наш первый файл

  1. Нажмем на папке Source Files и выберем пункт меню

    Project - Add New Item Ctrl+Shift+A

  2. В открывшемся окне Add New Item на вкладке Categories выберем Code, а на вкладке Templates выберем C++ File (.cpp)

    В поле Name напишем имя файла main

    В поле Location можно выбрать расположение файла.

    Нажимаем кнопку ADD

  3. В папке Source Files появится название файла main.cpp

  4. Справа от окна Solution Explorer в новом окне откроется файл main.cpp

Каждая программа на языке C++ начинает выполняться с функции main(). Напишем ее

#include <iostream>

using namespace std;

int main() {
  cout << "Hello World!";
  cin.get();
}

Соберем наш проект. Выберем пункт меню

Build - Build Solution F7

Чтобы запустить процесс отладки нажимаем кнопку Start Debugging. Откроется консоль и мы увидим результат

Hello World!

Консоль не закрывается и ожидает от нас нажатия на любую клавишу. Это благодаря команде cin.get().

2 Variables

Напишем программу, которая вводит два числа и вычисляет их сумму. Прежде чем ввести два числа, нужно выделить для них память. Чтобы сохранить результат, нужно выделить память для хранения результата. Память в выделяется при помощи объявления переменных. Для объявления переменной сначала пишут ее тип, потом ее имя. Тип int означает целочисленную переменную.

#include <iostream>

using namespace std;

int main() {
  int x;
  int y;
  int z;

  x = 12;
  y = 23;
  z = x + y;

  cout << z;
}

3 Variables. if, if else

Ллогический тип данных может принимать одно из двух значений true или false. Условный оператор if в круглых скобках содержит условие, а в фигурных скобках - тело. Сначала оценивается условие, если оно равно true, то выполняется тело оператора, если равно false, то тело не выполняется

Создадим переменную со значением true. Проверим, что переменная равна значению true

#include <iostream>

using namespace std;

int main() {
  bool x = true;

  if (x == true) {
    cout << "true";
  }
}

Создадим переменную со значением false. Проверим, что переменная не равна значению true

#include <iostream>

using namespace std;

int main() {
  bool x = false;

  if (x != true) {
    cout << "true";
  }
}

Создадим целочисленную переменную x и введем ее значение с клавиатуры. Проверим, что переменная меньше 12.

#include <iostream>

using namespace std;

int main() {
  int x;
  cin >> x;

  if (x < 12) {
    cout << "true";
  } else {
    cout << "false";
  }
}

4 Loops

Напишем цикл с помощью оператора for.

#include <iostream>

using namespace std;

int main() {
  for (int i = 0; i < 5; i++) {
    cout << "line " << i << endl;
  }
}

Напишем цикл с помощью оператора while.

#include <iostream>

using namespace std;

int main() {
  int i = 0;
  while (i < 5) {
    cout << "line " << i << endl;
    i++;
  }
}

Напишем цикл с помощью оператора do while.

#include <iostream>

using namespace std;

int main() {
  int i = 0;
  do {
    cout << "line " << i << endl;
    i++;
  } while (i < 5);
}

6 Arrays

Создадим массив и сразу проинициализируем его пятью элементами.

#include <iostream>

using namespace std;

int main() {
  int a = {12, 23, 34, 45, 56};
  for (int i = 0; i < 20; i++) {
    cout << a[i] << ' ';
  }
}

Если в массиве a хранится база данных возрастов сотрудников, которые имеют ID. По номеру ID можно получить возраст из базы данных.

#include <iostream>

using namespace std;

int main() {
  int a = {12, 23, 34, 45, 56};
  while (true) {
    cout << "Enter your ID ";
    int ID = 0;
    cin >> ID;
    if (ID < 0 || ID > 4) {
      break;
    }

    cout << "Age is " << a[ID] << endl;
  }
}

7 Arrays

Напишем алгоритм поиска максимального возраста id человека с таким возрастом.

#include <iostream>

using namespace std;

int main() {
  int a = {12, 23, 34, 45, 56};
  int age = a[0];
  int id = 0;
  for (int i = 1; i < 5; i++) {
    if (age < a[i]) {
      age = a[i];
      id = i;
    }
  }

  cout << "Age " << age << "has id = " << id;
}

Найдем средний возраст работников.

#include <iostream>

using namespace std;

int main() {
  int a = {12, 23, 34, 45, 56};
  int sum = 0;
  for (int i = 0; i < 5; i++) {
    sum += a[i];
  }
  int average = sum / 5;

  cout << average;
}

8 Functions

Напишем функцию суммирования двух целых чисел.

#include <iostream>

using namespace std;

int sum(int x, int y) {
  return x + y;
}

int main() {
  int a = 5;
  int b = 10;

  cout << sum(a, b);
}

Напишем функцию суммирования целых чисел массива.

#include <iostream>

using namespace std;

int sum(int arr[], size) {
  int res = 0;
  for (int i = 0; i < size; i++) {
    res += arr[i];
  }
  return res;
}

int main() {
  int a = {12, 23, 34. 45, 56};

  cout << sum(a, 5);
}

9 Classes

Напишем класс Person, который содержин одно поле и один метод, который печатает значение поля.

#include <iostream>

using namespace std;

class Person {
public:
  int age;
  char* name;

  void printAge() {
    cout << age << endl;
  }
  void printName() {
    cout << name << endl;
  }
};

int main() {
  Person p1;
  p1.age = 25;
  p1.name = "Petr";
  p1.printAge();
  p1.printName();

  Person p2;
  p2.age = 35;
  p2.name = "Vova";
  p2.printAge();
  p2.printName();
}

10 Classes

Весь код программы мы писали в одном файле. Обычно, когда программа содержит функцию и классы, то ее разбивают на несколько файлов. Создадим новый файл, в котором будет содержаться класс Person. Для этого выберем пункт меню

Project - Add Class

В открывшемся окне Add Class на вкладке Categories выберем C++, на вкладке Templates выберем C++ Class и нажмем кнопку Add.

В открывшемся окне Generic C++ Class Wizard в поле Class name напишем имя класса Person. Для него будут созданы заголовочный файл Person.h и исходный файл Person.cpp. Это хорошая практика держать декларацию класса отдельно от его реализации. Нажимаем на кнопку Finish

В папке Header Files появится файл Person.h. В папке Source Files появится файл Person.cpp.

Файл Person.h будет уже иметь начальный код

#pragma once

class Person {
public:
  Person();
  ~Person();
};

Файл Person.cpp будет уже иметь начальный код

#include "Person.h"

Person::Person() {}
Person::~Person() {}

Теперь напишем свой класс Person в файлах Person.h и Person.cpp и уберем его из файла main.cpp.

11 Conclusion

Для класса Person напишем конструктор. Конструктор класса это функция, которая проводит начальную инициализацию объекта во время его создания.

Файл Person.h

#pragma once

class Person {
public:
  int age;
  char* name;

  Person(int _age, char* _name);
  ~Person();
  void printAge();
  void printName();
};

Файл Person.cpp

#include <iostream>
#include "Person.h"

using namespace std;

Person::Person(int _age, char* _name) {
  age = _age;
  name = _name;
}
Person::~Person() {}

void Person::printAge() {
  cout << age << endl;
}
void Person::printName() {
  cout << name << endl;
}

Файл main.cpp

#include <iostream>
#include "Person.h"

using namespace std;

int main() {
  Person p1(25, "Petr");
  p1.printAge();
  p1.printName();

  Person p2(35, "Vova");
  p2.printAge();
  p2.printName();
}

Добавим конструктор по умолчанию, для того чтобы можно было создавать объекты класса Person без параметров.

Файл Person.h

#pragma once

class Person {
public:
  int age;
  char* name;

  Person();
  Person(int _age, char* _name);
  ~Person();
  void printAge();
  void printName();
};

Файл Person.cpp

#include <iostream>
#include "Person.h"

using namespace std;

Person::Person() {}
Person::Person(int _age, char* _name) {
  age = _age;
  name = _name;
}
Person::~Person() {}

void Person::printAge() {
  cout << age << endl;
}
void Person::printName() {
  cout << name << endl;
}

Файл main.cpp

#include <iostream>
#include "Person.h"

using namespace std;

int main() {
  Person p[5];

  p[0].name = "Petr";

  p[0].printName();
}
