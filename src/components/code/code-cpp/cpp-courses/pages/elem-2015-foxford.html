1 Введение в язык С

Курс посвящен языкам программирования C и C++. Мы начнем с самого начала.

Почему нужно изучать языки C и C++ и чем они лучше других языков?

  1. Скорость. Языки C и C++ это самые быстрые высокоуровневые языки программирования.
  2. Лаконичность. В них нет ничего лишнего, нет избыточных конструкций и этим они достаточно просты. Каждое слово языка и каждый символ существенно влияет на работу программы.
  3. Популярность. Количество кода на языках C и C++ превосходит написанное на любом другом языке.
  4. Обучение. Изначально язык создавался для программистов, для написания ядра операционной системы. Позднее стал использоваться для обучения программированию.
  5. Спортивное программирование.

Язык C разработали в 1969-1973 годах сотрудники Bell Labs Кен Томпсон и Денис Ритчи. Он создавался для разработки операционной системы Unix. Это процедурный язык программирования. Это значит что в программе есть только переменные или функции и нет сложных конструкций.

Язык C++ разработал в 1983 году сотрудник Bell Labs Бьерн Страуструп.

Языки программирования делятся на:

  1. компилируемые (C, C++, Pascal)
  2. интерпретируемые (Python)

Компилятор берет файл с текстом программы на языке C и выдает файл с набором инструкций для процессора.

Интерпретатор построчно проходит по тексту программы и выполняет ее.

Выражение может содержать операции +, -, *, /, %, его можно присваивать переменным или использовать как аргумент функции. Выражение возвращает значение определенного типа данных.

Сначала поработаем с целыми числами (тип данных int).

int a = 2 + 3;
printf("%d", 2 + 3);

Во всех современных компьютерах используется двоичная система счисления. Двоичная система, как и десятичная, является позиционной. То есть вклад цифры в число зависит от ее позиции в числе.

Переведем число 38 из десятичной системы в двоичную.

3810 = 1001102

Тип данных int представляет собой двоичное число из 4 байт.

Тип int может быть со знаком или без знака. Чтобы представить число со знаком, нужно хранить знак числа. Знак числа хранится в самом старшем бите. 0 для положительных чисел и 1 для отрицательных.

Записывается отрицательное число так, чтобы при сложении с положительным числом, равным ему по модулю, в результате получался 0. Для этого достаточно инвертировать все биты положительного числа и прибавить к результату 1.

-3810 = 110110102

Проверим результат, для этого сложим два числа

38 + (-38) = 0 10'0110 + 1101'1010 = 1'0000'0000

Это был пример для числа размером 1 байт. В резултате получисля 0, потому что 1 в девятом разряде это переполнение и она вышла за границы числа.
