Урок 1

Для прохождения курса будем использовать C++Builder 6.

В верхней части окна программы расположена строка заголовка. Она обычно содержит название программы и имя открытого файла. Также на ней находятся кнопки Свернуть, Восстановить, Закрыть окно. Под заголовком находится строка меню, которая предоставляет доступ ко всем функциям и командам программы. Под главным меню находятся быстрые кнопки, объединенные в группы по назначению. Они позволяют получить быстрый доступ к наиболее часто используемым командам. Рядом с быстрыми кнопками расположена палитра компонентов. Она состоит из нескольких закладок, на которых располагаются визуальные компоненты, используемые при создании программ в среде C++Builder. При малом разрешении экрана все эти закладки не умещаются в пределах рабочей области программы и становятся не доступны. Чтобы их увидеть надо воспользоваться кнопками прокрутки, которые расположены в правой верхней части палитры компонентов.

В левой части экрана располагается Object Inspector - инспектор объектов. В этом окне производится настройка основных свойств визуальных компонентов.

Справа от окна инспектора объектов распологается окно дизайнера форм. Это окно содержит строку заголовка, в котором отображается название формы, и кнопки Свернуть, Восстановить, Закрыть окно.

Под ним располагается окно редактора кода, в котором производится набор кода программы.
