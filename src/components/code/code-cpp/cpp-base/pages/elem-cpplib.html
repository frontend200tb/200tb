algorithm

Содержит алгоритмические функции:

  1. min(arg1, arg2) возвращает минимальный элемент из arg1 и arg2;
  2. max(arg1, arg2) возвращает максимальный элемент из arg1 и arg2;
  3. swap(arg1, arg2) меняет местами значения в переменных из arg1 и arg2;
  4. sort(po1, pos2) упорядочивает элементы в указанном диапазоне по не убыванию элементов или согласно критерию упорядочивания, заданному функцией.
  5. find(pos1, pos2, arg) находит позицию первого вхождения элемента с заданным значением в диапазон. Если элемент не будет найден значение будет равно .end();
  6. count(pos1, pos2, arg) возвращает количество элементов в диапазоне равных значению arg.

iostream

vector

4 метода, которые возвращают итератор на элемент вектора:

  1. .begin() возвращает итератор, указывающий на первый элемент в векторе.
  2. .end() возвращает итератор, указывающий за последним элементом вектора или конец вектора.
  3. .rbegin() возвращает итератор, указывающий на последний элемент вектора.
  4. .rend() возвращает итератор, указывающий перед первым элементом вектора.

Остальные методы:

  1. .size() возвращает целое число размер вектора.
  2. .empty() возвращает true если вектор пустой, либо false.
  3. .clear() удаляет все элементы вектора.
  4. .back() возвращает значение последнего элемента.
  5. .front() возвращает значение первого элемента.
  6. .push_back(val) добавляет в конец вектора новый элемент со значением val.
  7. .emplace_back(val) аналогично .push_back(val).
  8. .pop_back() удаляет последний элемент вектора.

Добавление элементов на определенную позицию:

  1. .push_front(val) добавляет в начало вектора новый элемент со значением val.
  2. .pop_front() удаляет первый элемент вектора.
  3. .insert(pos, val) добавление элемента val в vector в заданную позицию итератором pos.
  4. .emplace(pos, val) аналогично .insert(pos, val).
  5. .insert(pos, vals) добавление элементов vals в vector начиная с позиции, заданной итератором pos.
  6. .insert(pos, n, val) добавлет n элементов val в vector начиная с позиции, заданной итератором pos.
  7. .insert(pos, begin, end) добавлет начиная с позиции, заданной итератором pos, элементы из другого контейнера между итераторами begin и end.

Удаление элементов на определенной позиции:

  1. .erase(pos) удаление элемент на позиции заданной итератором pos.
  2. .erase(begin, end) удаление элементов из диапазона, заданного итераторами begin и end.

Изменение размера вектора

  1. .resize(n) новый размер вектора. Если вектор содержал больше элементов, то с конца удаляются лишние элементы. Если вектор содержал меньше элементов, то в конец добавляются элементы инициализированные значением по умолчанию.
  2. .resize(n, val) новый размер вектора. Если вектор содержал больше элементов, то с конца удаляются лишние элементы. Если вектор содержал меньше элементов, то в конец добавляются элементы со значением val.

Изменение элементов вектора

  1. .assign() заменяет элементы вектора
  2. .swap() обменивает значения двух контейнеров

cmath

Содержит математические функции и константы

Константы

  1. e 2.71828 основание натурального логарифма
  2. pi 3.14159 число пи
  3. sqrt2 1.41421 корень числа 2
  4. phi 1.618 золотое сечение (число Фидия)

Математические функции

  1. abs(arg) возвращает число по абсолютной величине;
  2. sqrt(arg) возвращает корень от числа;
  3. pow(arg1, arg2) возвращает число arg1 в степени arg2;
  4. exp(arg) возвращает экспоненту в степени arg;
  5. log(arg) возвращает натуральный логарифм от arg;
  6. log10(arg) возвращает логарифм по основанию 10 от arg;

Функции округления

  1. round(arg) возвращает правильное математическое округление;
  2. ceil(arg) возвращает округление в большую сторону;
  3. floor(arg) возвращает округление в меньшую сторону.

Тригонометрические функции

  1. sin(arg) возвращает синус от аргумента;
  2. cos(arg) возвращает косинус от аргумента;
  3. tan(arg) возвращает тангенс от аргумента.
  4. asin(arg) возвращает арксинус от аргумента;
  5. acos(arg) возвращает арккосинус от аргумента;
  6. atan(arg) возвращает арктангенс от аргумента. Результат от (-pi / 2) до (pi / 2)
  7. atan2(arg) возвращает арктангенс от аргумента. Результат от -pi до pi

Arg передается как радианы, не градус!

Переведем grad градусов в arg радиан

int arg = (grad / 180) * pi;

В библиотеке нет котангенса, но его можно выразить через тангенс

int cotan = 1 / tan(arg);

Проверка на значение

  1. isinf(arg) возвращает true, если аргумент представляет +Infinity или -Infinity
  2. isnan(arg) возвращает true, если аргумент представляет NaN
