1. Предисловие

Подробно рассказано о том, как начать работать с ассемблером и как писать программы на этом языке. Для практической работы необходимо использовать ассемблер Microsoft MASM 6.15. В основном рассматривается 32-разрядный режим работы с Windows, также приведены сведения для 16-разрядного режима, например, описание работы прерываний и понятие о DOS и BIOS. Книга является введением в ассемблер.

Язык ассемблера дает представление о том, как аппаратная часть взаимодействует с операционной системой и как прикладные программы обращаются к операционной системе. Язык ассемблера - это язык низкого уровня, когда каждая команда непосредственно интерпретируется в команду процессора. Язык ассемблера существует для каждого типа процессоров. Команды на языке ассемблера должны иметь взаимно однозначное соответствие с системой машинных команд и должны быть согласованы с архитектурой компьютера.

Ассемблер это программа, преобразовывающая исходные коды языка ассемблера в машинные коды. Ассемблер можен создавать листинг программы с номерами строк, адресами памяти, исходными операторами и таблицей перекрестных ссылок символов и переменных. Вместе с программой ассемблера испозьзуется программа загрузчик, которая собирает отдельные файлы, созданные ассемблером в единую программу, и программа отладчик, позволяющая проверять и трассировать исполняемую программу и контроллировать содержимое памяти.

Для разработки в среде DOS используется ассемблер TASM от фирмы Borland. Для разработки в среде Windows используется ассемблер MASM от фирмы Microsoft.
