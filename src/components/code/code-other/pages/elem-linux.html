Командная строка linux

pwd (print working directory)
печатает путь к текущей директории
ls (list)
печатает содержимое текущей директории
cd директория (change directory)
переходит в указанную директорию. Если вызвать cd без аргумента, то будет переход в домашнюю директорию пользователя
cat файл
выводит на экран содержимое файла
chmod
изменяет права доступа к объекту
