Версии языка

Компьютеры понимают единственный язык - машинный код. Он представляет собой последовательность нулей и единиц, которые компьютер интерпретирует как команды которые нужно выполнять. Машинный код невероятно туден для людей. Поэтому стали создавать языки программирования, которые были бы понятны людям. Гвидо ван Россум (Guido van Rossum) создал язык Python. Гвидо начал работу над языком, который планировался как замена языку АВС, еще в декабре 1989 года.

Python прошел через несколько итераций и сейчас имеется два пути его развития. Путь 2.х обладает обратной совместимостью с предыдущими версиями Python, которой не обладает путь 3.х.

Типы данных

int

Целочисленный (integer) тип. Не содержит десятичную точку. Хранится в 65 битной переменной.

от -9 223 372 036 854 775 808 до 9 223 372 036 854 775 807

float

Числа с плавающей точкой (float). Содержат десятичную точку. Например 1.0 или 3.14.

максимальное значение 1.7976931348623157 * 10308 минимальное значение 2.2250738585072014 * 10-308

Комплексное число

Комплексное число состоит из пары чисел - действительной и мнимой частей. Мнимая часть указывается с помощью символа j после нее. Например запишем комплексное число с действительной частью 3 и мнимой частью 4

3 + 4j

bool

Логический тип (boolean) назван в честь Джорджа Буля (George bool). Логический тип может содержать только одно из двух значений True или False

Арифметические операторы

  1. + сложение 5 + 2 = 7
  2. - вычитание 5 - 2 = 3
  3. * умножение 5 * 2 = 10
  4. / деление 5 / 2 = 2.5
  5. % остаток от деления 5 % 2 = 1
  6. ** возведение в степень 5 ** 2 = 25
  7. // деление нацело 5 // 2 = 2

Операторы присваивания

  1. = присваивание a = 5
  2. += сложение и присваивание a += 2 сохранит 7
  3. -= вычитание и присваивание a -= 2 сохранит 3
  4. *= умножение и присваивание a *= 2 сохранит 10
  5. /= деление и присваивание a /= 2 сохранит 2.5
  6. %= остаток от деления и присваивание a %= 2 сохранит 1
  7. **= возведение в степень и присваивание a **= 2 сохранит 25
  8. //= деление нацело и присваивание a //= 2 сохранит 2

Унарные операторы

  1. + возвращает то же самое число +4 = 4
  2. - меняет знак числа на противоположный -(-4) = 4
  3. ~ инвертирует каждый бит числа ~4 (0000 0100) = -5 (1111 1011)

Побитовые операторы

  1. & и (только две 1 вернут 1) 0b1100 & 0b0110 = 0b0100
  2. || или (только два 0 вернут 0) 0b1100 || 0b0110 = 0b1110
  3. ^ исключающее или (совпадающие значения вернут 0, разные вернут 1) 0b1100 ^ 0b0110 = 0b1010
  4. ~ дополнение до 1 (возвращает значение, которое дополняет данное число до 1) ~0b1100 = -0b1101 ~0b0110 = -0b0111
  5. << сдвиг влево (новые биты справа - нулевые, лишние биты слева - теряются) 0b00110011 << 2 = 0b11001100
  6. >> сдвиг вправо (новые биты слева - нулевые, лишние биты справа - теряются) 0b00110011 >> 2 = 0b00001100

Операторы сравнения

  1. == равенство 1 == 2 вернет False
  2. != неравенство 1 != 2 вернет True
  3. > больше 1 > 2 вернет False
  4. < меньше 1 < 2 вернет True
  5. >= больше или равно 1 >= 2 вернет False
  6. <= меньше или равно 1 <= 2 вернет True

Логические операторы

  1. and и True and True == True
  2. or или True or False == True
  3. not отрицание not True == False

Операторы членства

  1. in одно значение присутствует в другом "Hello" in "Hello Goodbye" вернет True
  2. not in одно значение отсутствует в другом "Hello" not in "Hello Goodbye" вернет False

Операторы тождественности

Проверяют тип данных

  1. is тип значения является данным типом type(2) is int вернет True
  2. is not тип значения не является данным типом type(2) is not int вернет False

Приоритеты операторов

  1. () скобки для группирования выражений
  2. ** возведение в степень
  3. ~ + - унарные операторы
  4. * умножение, / деление, % остаток от деления, // деление нацело
  5. + сложение, - вычитание
  6. << >> побитовые сдвиги
  7. & побитовое И
  8. ^ побитовое исключающее ИЛИ, | побитовое ИЛИ
  9. <= < > >= операторы сравнения
  10. == != операторы равенства
  11. = += -= *= /= %= //= **= операторы присваивания
  12. is, is not операторы тождества
  13. in, not in операторы членства
  14. and, or, not логические операторы

Дата и время

Чтобы работать с датой и временем необходимо импортировать модуль datetime

import datetime

Получим текущее время

datetime.datetime.now() вернет (2016,12,20,10,37,24,460099)

Получим текущую дату

str(datetime.datetime.now().date()) вернет '2016-12-30'
