  printf("\n s = %f", s);
}

Программа 2 (55)

Тело движется прямолинейно с ускорением. Даны a м/с² ускорение, v м/с начальная скорость. Требуется определить, какой путь пройдет тело за t секунд.

Решение 
#include <stdio.h>

float a, v, t, s;

int main() {
  a = 20.2;
  v = 50.3;
  t = 65;
  s = (v * t) + (a * t * t / 2);
  printf("s = %f meters", s);
}
