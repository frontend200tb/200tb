1 Вступление (6)

Сотрудник фирмы Bell Labs Дэннис Ритчи создал язык Си в 1972 г. во время совместной работы с Кеном Томпсоном над операционной системой UNIX. Си был разработан как инструмент для программистов-практиков. Главной целью было создание удобного и полезного языка, облегчающего труд программиста.

Программы на языке Си отличаются компактностью и быстротой исполнения.

2 Введение в язык Си (18)

Напишем программу, которая переводит 2 морские сажени в футы.

main() {
  int feet, fathoms;
  fathoms = 2;
  feet = 6 * fathoms;
  printf("В %d морских саженях содержится %d футов", fathoms, feet);
}

Напишем свою функцию butler() и вызовем ее из функции main().

butler() {
  printf("Вы вызывали, сэр?\n");
}

main() {
  printf("Я вызываю функцию butler.\n");
  butler();
  printf("Да, принесите мне чашку чая и гибкие диски.\n");
}

3 Данные, язык Си и вы (31)

Целое число в двоичном виде помещается в память. Вещественное число разделяется на мантиссу и порядок и обе части отдельно помещаются в память. Даже если вещественное число равно целому, например 7.0 = 7, в памяти они хранятся по разному. Операции с дробными числами могут приводить к потере точности.

Целые константы

Число без десятичной точки и без показателя степени рассматривается как константа типа int, например 22 и -273. Но числа 22.0 и 22e3 не целые, потому что в их записи есть десятичная точка и показател степени.

Для записи константы типа long, нужно вконце числа дописать букву L или l, например 212L.

Если целове число начинается с цифры 0, то это интерпретируется как восьмеричное число. Например, 020 это удвоенная степень основания 8, то есть десятичное число 16.

Если целое начинается с символов 0x или 0X, то это интерпретируется как шестнадцатеричное число. Например, 0x20 это удвоенная степень основания 16, то есть десятичное число 32.

Шестнадцатеричные числа очень популярны в программировании, например число 65536 это шестандацеричное число 0x10000.

Если к наибольшему значению переменной целого типа применить операцию сложения, то происходит переполнение. Целочисленные переменные на компьютере работают как одометр в машине. При этом следующее число после максимального является минимально возможным для типа данных.

Для чисел без знака, например 65535 + 1, в результате мы получим значение равное 0.

Для чисел со знаком, например 32767 + 1, то в результате мы получим значение равное -32768.

Напишем программу, которая выдает номер кода введенного символа даже если на машине используется код отличный от ASCII.

main() {
  char ch;
  printf("Введите символ.\n");
  scanf ("%c", &ch);
  printf("Код символа %c равен %d.\n", ch, ch);
}

Функция printf первый раз выведет величину ch как символ, второй раз - как целое число.

Типы данных float и double

Для представление чисел с плавающей точкой типа float в памяти отводится 32 бита - 8 бит для порядка и знака и 24 бита для мантиссы (степень числа 10). Такой способ представляет числа с точностью 6-7 десятичных цифр в диапазоне от 10-37 до 1038.

Для представление чисел с плавающей точкой типа double в памяти отводится 64 бита.

При записи константы с плавающей точкой можно не писать точку или знак e. Также можно не писать целую часть или дробную.

3.1415 .2 100. 4e16 .8e-5
