sizeof (15)

С помощью операции sizeof можно определить размер памяти, которая соответствует идентификатору или типу

#include <stdio.h>

struct {
  char h;
  int b;
  double f;
} str;

int main() {
  int a;
  a = sizeof(str);
  printf("%d\n", a);
}
16

Пустой оператор (25)

Пустой оператор состоит только из точки с запятой. При выполнении этого оператора ничего не происходит. Он необходим в следующих случаях:

  1. В операторах do, for, while, if если по синтаксису требуется оператор, но его нет
  2. После метки перед фигурной скобкой. После метки всегда должен быть оператор, а фигурная скобка оператором не является.
#include <stdio.h>

int main() {
  int i = 0;
  for (;; i++) {
    if (i == 1) {
      goto a;
    }
    if (i == 3) {
      goto b;
    }
    printf("%d ", i);
    a:;
  }
  b:;
}
