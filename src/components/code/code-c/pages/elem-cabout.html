about

Язык Си один из самых популярных и распространенных языков программирования. Высокая скорость программ написанных на языке Си является причиной того что на нем пишут операционные системы и драйверы.

История создания языка Си

К 1969 году языков программирования было много. Господствовал процедурный стиль программирования. Когда писались функции. Был язык для инженерных расчетов Fortran. Был язык для экономических расчетов Kobol. А языка для программистов не было. Программисты работали под UNIX, писали на ассемблере, а вокруг уже работали на языках высокого уровня. И они стали пытаться сделать язык для себя. Сначала язык A, потом язык B, потом Си и на этом остановились.

1966 язык BCPL (Деннис Ритчи). Предшественник языка B.

1969 язык B (Деннис Ритчи). Предшественник языка C.

1969-1972 - на базе языка B (би) - упрощенного варианта BCPL (Basic Combined Programming Language) в компании Bell Labs программистом Деннисом Ритчи (Dennis Ritchie) разрабатывался язык Си. Второй соавтор языка Си это Кен Томпсон.

1972 язык C (Деннис Ритчи). Создан для разработки ОС UNIX на платформе PDP-7

1973 на язык Си перенесен значительный фрагмент ядра Unix для PDP-11, ранее разработанный на языке ассемблера

1978 Деннис Ритчи вместе с Брайаном Керниганом написал хорошую книгу "Язык программирования Си"

1985 книга Кернигана и Ритчи переведена на русский язык и вышла под названием "Язык программирования Си. Задачи по языку Си"

1988 выходит второе издание книги Кернигана и Ритчи "Язык программирования Си" (первое описание будущего стандарта ANSI C)

1992 второе издание книги Кернигана и Ритчи "Язык программирования Си" выходит на русском языке

Стандарты

1983Американский национальный институт стандартов ANSI сформировал комитет для создания стандарта языка Си

1989 Утвержден первый стандарт ANSI C или C89

1990 Принят стандарт ANSI C или C90, который не отличался от C89

2000 Принят стандарт C99, в него вошли:

  • Новые встроенные типы данных: long long, _Bool, _Complex, _Imaginary
  • Новые возможности языка, включая массивы переменной длины, индексы статических массивов, compound-литералы, вариативные макросы
  • Новое ключевое слово restrict
  • Новые заголовочные файлы stdint.h, tgmath.h, fenv.h, complex.h
  • Возможности, совместимые с C++: inline-функции, однострочные комментарии //, смешение декларации и кода, символы Юникода в идентификаторах
  • Удалены опасные элементы C89, например, неявное объявление функций и неявное int

2011 Принят стандарт c11, в него вошли:

  • Улучшенная поддержка Юникода
  • Новое ключевое слово _Generic
  • Кроссплатформенная поддержка многопоточности threads.h
  • Поддержка атомарных типов в языке и библиотеке stdatomic.h

Стандартные библиотеки языка Си

Стандарт языка Си определяет несколько стандартных библиотек, содержащих протестированные функции, которые могут быть использованы в любых программах на языке Си

Библиотеки содержатся в заголовочных файлах с расширением .h

  1. stdio.h

    Содержит функции ввода-вывода, типы и описания макросов

  2. ctype.h

    Содержит функции для работы с символами

  3. string.h

    Содержит функции для работы со строками

  4. math.h

    Содержит математические функции

  5. stdlib.h

    Содержит функции для преобразования чисел, выделения памяти и т.д.

  6. assert.h

    Содержит функции для диагностики программы

  7. stdarg.h

    Содержит функции для перебора аргументов функций

  8. setjmp.h

    Содержит функции чтобы нарушить обычную последовательность входа в функции и выхода из них

  9. signal.h

    Содержит функции для обработки исключительных ситуаций, которые могут возникнуть в программе

  10. time.h

    Содержит функции для работы с датой и временем

  11. limits.h

    Содержит константные определения размеров типов данных языка Си

  12. float.h

    Содержит определения констант для арифметики с плавующей точкой

limits.h

Содержит определяемые реализацией размеры каждого типа данных. Они доступны через константы. Для каждого типа данных есть свои константы минимального и максимального значения

CHAR_MIN, CHAR_MAX

SHRT_MIN, SHRT_MAX

INT_MIN, INT_MAX

LONG_MIN, LONG_MAX

LLONG_MIN, LLONG_MAX
