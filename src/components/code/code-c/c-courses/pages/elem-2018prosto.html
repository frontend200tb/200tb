1 Выбор редактора (IDE). Первая программа. Основные типы данных

Чтобы научиться программировать, нужно программировать. Просто просматривать лекции большого толку не дает. Нужно самому еще искать, читать, учиить. Программирование это обычная инженерная специальность, которая требует дополнительных знаний.

В качестве IDE будем использовать QT. Он кроссплатформенный и бесплатный. Устанавливаем программу только на английском языке.

Создадим новый проект в редакторе QT Creator

  1. Выбираем пункт меню

    File - New File or Project (Ctrl+N)

  2. В открывшемся окне New File or Project выберем

    Choose a template - Projects - Other Project - Subdirs Project

    Нажимаем кнопку Choose.

  3. В открывшемся окне Subdirs Project на вкладке Location нужно прописать Name (название проекта) и выбрать путь к нему Create in.

    Во всех названиях и путях должны быть только английские буквы без пробелов. Расположение проекта лучше выбрать на отдельном диске в отдельной папке и как можно короче, например D:\code

    Нажимаем кнопку Next

  4. В том же окне Subdirs Project на вкладке Kits поставим галочку на пункте

    Desktop QT 5.4.2 MinGW 32bit2

    Нажимаем кнопку Next

  5. В том же окне Subdirs Project на вкладке Summary нажимаем кнопку Finish & Add Subproject

  6. В открывшемся окне New Subproject на вкладке Choose a template выберем

    Non-QT Project - Plain C Application

    Нажимаем кнопку Choose.

  7. В открывшемся окне Plain C Application на вкладке Location нужно прописать Name (название проекта) и выбрать путь к нему Create in.

    Нажимаем кнопку Next

  8. В том же окне Plain C Application на вкладке Build System выбираем qmake

    Нажимаем кнопку Next

  9. В том же окне Plain C Application на вкладке Kits поставим галочку на пункте

    Desktop QT 5.4.2 MinGW 32bit2

    Нажимаем кнопку Next

  10. В том же окне Plain C Application на вкладке Summary нажимаем кнопку Finish

  11. Откроется окно main.c - QT Creator

  12. В менеджере файлов, который должен быть слева от основного окна, создадим новый файл с заметками, в котором запишем что нам нужно изучить, прочитать. Для этого нажмем правой кнопкой мыши на папку Source, выберем Add new

  13. В открывшемся окне New File выберем

    General - Empty File

    Нажимаем кнопку Choose

  14. В открывшемся окне Empty File на вкладке Location пропишем имя файла "notes.txt" в поле Name и путь к файлу в поле Path

    Нажимаем кнопку Next

  15. В том же окне Empty File на вкладке Summary нажимаем кнопку Finish

Файл main.c содержит простую программу.

#include <stdio.h>

int main(int argc, char *argv[]) {
  printf("Hello World!\n");
  return 0;
}

Программа на языке программирования Си содержит инструкции понятные человеку. Далее компилятор переводит эти инструкции в машинный код, понятный компьютеру. Компилятор собирает все файлы с кодом в один исполняемый файл, который будет запускаться на той же платформе.

Чтобы запустить программу, нужно слева внизу нажать кнопку Run

Замена табуляции на пробелы в редакторе QT

Выбираем пункт меню

Tools - Options - C++ - Edit - Tab policy - Spaces Only

Изменим число пробелов

Indent size - 2

2 Функция, пишем свою функцию. Часть 1

Любая программа начинается с функции main. При написании программ нужно писать свои функции.

Напишем функцию суммы двух чисел

#include <stdio.h>

int sum(int a, int b) {
  return a + b;
}

int main() {
  printf("5 + 3 = %d", sum(5, 3));
  return 0;
}
