Install

  • Нужно установить среду разработки, где будем писать код и для получения результата. Рассмотрим два варианта.

Вариант 1. Полноценная среда разработки Visual Studio Community 2022.

  • Скачиваем установщик с оффициального сайта visualstudio.microsoft.com/ru/downloads . Запускаем его, установщик докачает все необходимое. Далее мы попадаем на экран где нужно выбрать какие модули надо установить. Выбираем Разработка классических приложений .NET, ASP.NET и разработка веб-приложений. После этого нажимаем Установить (или Изменить).

    И при инсталляции Visual Studio на ваш компьютер будут установлены все необходимые инструменты для разработки программ, в том числе фреймворк .NET 7.

    Для создания первой программы, откроем Visual Studio. На стартовом экране выберем Create a new project (Создать новый проект). В окне Создание проекта выбираем Консольное приложение .NET Core и жмем далее

    Далее на следующем этапе нам будет предложено указать имя проекта и каталог, где будет располагаться проект.

    На следующем окне Visual Studio предложит нам выбрать версию .NET, которая будет использоваться для проекта. Выберем последнюю на данный момент верси. - .NET 7.0

    Нажмем на кнопку Create (Создать) для создания проекта, и после этого Visual Studio создаст и откроет нам проект

Вариант 2. Редактор кода Visual Studio Code.

  • В этом случае необходимо еще докачать платформу .NET dotnet.microsoft.com/en-us/download. После этого можно использовать нужные команды, которые будут доступны из терминала. С помощью этих команд можно создать или запустить проект. В VSCode надо установить расширение C# от ms-dotnettools.csharp от Microsoft C# for Visual Studio Code (powered by OmniSharp)

    Чтобы создать новый проект нужно открыть пустую папку. Открыть терминал. Команда clear - очистить терминал

    Проверим установленную версию .NET dotnet --version
    7.0.401

    Создадим новое консольное приложение dotnet new console

    Запустим созданный проект командой dotnet run

.Net

  • .Net (дот нет) Framework - платформа программирования под Windows

    .Net Core - открытый кросс-платформенный фреймворк под Windows, Linux, macOS

    C# (си шарп) - основной язык программирования платформы .Net

    .Net Framework и .Net Core состоят из исполняющей среды и базовой библиотеки классов

    CLR (Common Language Runtime) - исполняющая среда под платформой .Net Framework

    Core CLR - исполняющая среда под платформой .Net Core

    BCL (Base Class Library) = FCL (Foundation Class Library) + Frameworks
