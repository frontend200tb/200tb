Theory

  1. History of React

    React is an open-source JavaScript library that emerged from Facebook's internal workings. React uses a declarative paradigm

    • 2011. The seeds of React took root in Facebook's dynamic ecosystem.
    • 2012. After Facebook acquired Instagram in 2012, they started using React in Instagram.com
    • 2013. Public release: Versions: 0.3.0 (May 29, 2013)— 0.8.0 (December 19, 2013)
    • 2014. Versions: 0.9.0 (February 20, 2014)— 0.12.2 (December 18, 2014)
    • 2015. Versions: 0.13.0 (March 10, 2015)— 0.14.5 (December 29, 2015)
    • 2016. Versions: 0.14.6 (January 6, 2016), 15.0.0 (April 7, 2016), 15.4.1 (November 22, 2016)
    • 2017. Versions: 15.4.2 (January 6, 2017)— 16.2.0 (November 28, 2017)
    • 2018. Versions: 16.2.1 (August 1, 2018)— 16.7.0 (December 19, 2018)
    • 2019. Versions: 16.8.0 (February 6, 2019)— 16.12.0 (November 14, 2019)
    • 2020. Versions: 16.13.0 (February 26, 2020)— 17.0.1 (October 22, 2020)
    • 2021. Versions: 17.0.2 (March 22, 2021)
    • 2022. Versions: 18.0.0 (March 29, 2022)— 18.2.0 (June 14, 2022)
    • 2024. Versions: 19 expected
