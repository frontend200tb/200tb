2. Установка программ

  1. VMware Workstation Pro
  2. eve-ng
  3. qemu-img
  4. FileZilla
  5. vnc viewer

VMware Workstation Pro

  1. Заходим на официальный сайт vmware

    https://www.vmware.com/

  2. В верхнем меню выбираем Products

  3. В открывшемся меню выбираем в самом низу правую кнопку SEE DESKTOP HYPERVISORS

  4. Попадаем на страницу

    https://www.vmware.com/products/desktop-hypervisor/workstation-and-fusion

  5. В разделе Product Overview в окошке VMware Workstation Pro for PC нажимаем кнопку DOWNLOAD NOW

eve-ng

Где скачать

  1. Заходим на официальный сайт eve-ng

    https://www.eve-ng.net/

  2. В верхнем меню выбираем DOWNLOAD

  3. В окошке Free EVE Community Edition Version 6.2.0-4 нажимаем ссылку EVE-NG CE Full ISO – direct link

Как установить

  1. Про установку eve-ng можно почитать на сайте

    https://itproblog.ru/

    В верхнем меню выбираем Содержание и нажимаем на ссылку 7 EVE-NG

  2. https://mikrotiklab.ru/emulyator/ustanovka-eve-ng.html
  3. https://mikrotiklab.ru/emulyator/dobavlenie-obrazov.html

qemu-img

  1. Заходим на официальный сайт FileZilla или его русскую версию

    https://cloudbase.it/

  2. В верхнем меню выбираем

    Projects - qemu-img for Windows

  3. Попадаем на страницу

    https://cloudbase.it/qemu-img-windows/

  4. Внизу в разделе Download binaries скачиваем архив нажимая на кнопку qemu-img 2.3.0 for Windows x64

FileZilla

  1. Заходим на официальный сайт FileZilla или его русскую версию

    https://filezilla-project.org/ https://filezilla.ru/

  2. Скачиваем Download FileZilla Client for Windows

vnc viewer

  1. Заходим на официальный сайт TightVNC Software

    https://www.tightvnc.com/

  2. В левом меню выбираем Download

  3. Внизу в разделе Older Versions of TightVNC нажимаем на ссылку download TightVNC 1.3.10.

  4. Попадаем на страницу Download TightVNC (Previous Versions)

    https://www.tightvnc.com/download-old.php

  5. Скачиваем версию Windows с описанием Complete set of executables, no installer
