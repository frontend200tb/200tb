Вопрос 5

Напишите функцию, которая сравнивает студентов. Прототип функции:

#define MAX_NUMBER_OF_MARKS 4

typedef struct
{
  const char* name;
  size_t number_of_marks;
  int marks[MAX_NUMBER_OF_MARKS];
} student_t;

/**
 * Compare two students. Criteria:
 * - average mark
 * - name
 *
 * @retval 0 if the first_student and the second_student are equal;
 * @retval a negative value if first_student is less than second_student;
 * @retval a positive value if first_student is greater than second_student.
 */
int student_cmp(const student_t* first_student, const student_t*  second_student)

Критерии сравнивания:

  1. Функция должна вернуть 1 для студентов А и Б, если у студента А больше взвешенная сумма баллов или, если баллы равны, если имя студента А больше имени студента Б в лексикографическом смысле
  2. Функция должна вернуть -1 для студентов А и Б, если у студента А меньше взвешенная сумма баллов или, если баллы равны, если имя студента А меньше имени студента Б в лексикографисеском смысле
  3. Функция должна вернуть 0 в иных случаях
  4. Веса для суммы (оценка = вес):

    1 = -5

    2 = -2

    3 = 1

    4 = 2

    5 = 4

Для лексиографического сравнения можете воспользоваться функцией strcmp

/**
 * @brief The function compares the two strings s1 and s2.
 * Returns an integer indicating the result of the comparison.
 *
 * @retval 0, if the s1 and s2 are equal
 * @retval a negative value if s1 is less than s2;
 * @retval a positive value if s1 is greater than s2
 */
int strcmp(const char *s1, const char *s2);
