Установка Sublime text (портативной версии)

  1. Заходим на сайт

    https://www.sublimetext.com/

  2. Скачиваем 64битную портативную версию

    sublime_text_build_4113_x64.zip

  3. Распаковываем ее в каталог

    c:\Portable\SublimeText\

Как добавить Sublime text в меню пуск

  1. Выбираем файл (клик правой кнопкой мыши)

    c:\Portable\SublimeText\sublime_text.exe
  2. Выбираем Закрепить на начальном экране

Как добавить программу Sublime text в контекстное меню

  1. Открыть Редактор реестра, для этого выбрать Пуск - Выполнить или нажатьwin + Rи ввести команду regedit
  2. В редакторе реестра переходим к ветке HKEY_CLASSES_ROOT\*\shell
  3. Клик правой кнопкой мыши на папке shell и выберите команду Создать > Раздел
  4. Назовите его так, как должна называться команда в контекстном меню. Скажем, Открыть с помощью Sublime text
  5. Клик правой кнопкой мыши на созданной папке и снова выбираем команду Создать > Раздел
  6. Новый вложенный раздел назовите command
  7. Щелкните на значке раздела command и дважды в окне справа на ключе (По умолчанию) REG_SZ
  8. В открывшемся окне введите следующую команду "c:\Portable\SublimeText\sublime_text.exe" "%1"

Как добавить иконку Sublime text в контекстное меню

  1. Открыть Редактор реестра, для этого выбрать Пуск - Выполнить или нажатьwin + Rи ввести команду regedit
  2. В редакторе реестра переходим к ветке HKEY_CLASSES_ROOT\*\shell\Открыть с помощью Sublime text
  3. Клик правой кнопкой мыши и выбираем команду Создать > Строковый параметр
  4. в поле Имя укажите значение Icon
  5. а в поле Значение укажите путь к exe-файлу, как показано ниже (кавычки обязательны): "c:\Portable\SublimeText\sublime_text.exe"

Открывается такое окно разделенное на две части. В левой части настройки по умолчанию, их изменить нельзя, а вот в правой части ты можешь эти настройки переназначить, каждая следующая должна идти через запятую.

Установка Packege Control

  1. Перед установкой плагинов надо установить Packege Control. Для установки в автоматическом режиме (если есть выход в интернет) надо нажать ctrl + shift + P и в появившемся окне набрать Install Package Control Для установки в ручном режиме заходим на сайт packagecontrol.io/installation. Инструкция с сайта ниже
  2. Click menu Preferences > Browse Packages
  3. Browse up Installed Packages folder C:\Portable\SublimeText\Data\Packages
  4. Download Package Control.sublime-package and copy it into the Installed Packages folder
  5. Restart Sublime Text

Установка плагинов для Sublime Text

Для установки плагинов в автоматическом режиме (если есть выход в интернет) надо нажать нажать Preferences - Package Control - Install Package или ctrl + shift + P и в появившемся окне набрать Install Package и найти нужный плагин. Для установки в ручном режиме ищем плагины на сайте

packagecontrol.io

Список полезных плагинов

  1. Emmet ускоряет написание кода
  2. All Autocomplete начинаем писать и нам предлагают подставить то что уже есть
  3. AutoFileName автоподстановка пути файлов и картинок
  4. Alignment выравнивает фрагменты кода
  5. BracketHighlighter подсветка начального и закрывающего тега
  6. ColorHighlighter подсветка цвета
  7. ColorPicker выбор цвета
  8. Goto-CSS-Declaration находит класс в css
  9. LiveReload Обновление файла в браузере при редактировании
  10. LiveServer
  11. LiveStyle
  12. Sass для работы с препроцессором Sass, выделяет синтаксис
  13. SideBarEnhancements больше команд в сайд баре
  14. SublimeLinter

Настройка горячих клавиш

Preferences - Key Bindings

Код пресета для пользовательских клавиатурных сокращений (Default (Windows).sublime-keymap:

[
{ "keys": ["alt+shift+f"], "command": "reindent" },
]

Теперь при выделении и по нажатию Alt+Shift+F ваш код будет автоматически выравниваться отступами с соответствующей иерархией вложенности элементов.
