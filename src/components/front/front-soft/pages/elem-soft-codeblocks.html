Первая программа в консоли (проект)

Откроем CodeBlocks

  1. Создадим новый проект. Для этого есть два способа

    1. В окне Start here выберем

    Create a new project

    2. Выберем меню

    File - New - Project

  2. В открывшемся окне New from template выберем

    Projects - Console application

    И нажмем Go

  3. В открывшемся окне Console application выберем язык C или C++ и нажмем Next

  4. В следующем окне Console application выберем Project title и Folder to create project in и нажмем Next

  5. В следующем окне Console application выберем Compiler, остальное оставим как есть и нажмем Finish

  6. В окне Management откроем файл

    Sources - main.c

  7. Увидим в нем код первой программы

    #include <stdio.h>

int main() {
  printf("Hello, world!");
  return 0;
}

  8. Соберем и запустим наш проект. Есть два способа сделать это:

    1. Выбрать меню Build - Build and run F9
    2. Нажать иконку Build and run

  9. Откроется консоль и мы увидим результат

    Hello, world!

Первая программа в консоли (файл)

Откроем CodeBlocks. Для олимпиадного программирования создавать проект не нужно. Достаточно написать программу в одном файле.

  1. Создадим новый файл. Для этого выберем пункт меню
    2. File - New - Empty file Ctrl-Shift-N
  2. В окне Start here появится новый пустой текстовый файл с заголовком *Untitled1
  3. Сразу сохраним его. Есть три способа сохранить новый файл
    1. Выбрать меню File - Save file Ctrl-S
    2. Выбрать меню File - Save file as
    3. Нажать иконку Save
  4. В открывшемся окне Save file выберем папку и имя файла и нажмем Сохранить

  5. Напишем код первой программы

    #include <stdio.h>

int main() {
  printf("Hello, world!");
  return 0;
}

  6. Соберем и запустим нашу программу. Есть два способа сделать это:

    1. Выбрать меню Build - Build and run F9
    2. Нажать иконку Build and run

  7. Откроется консоль и мы увидим результат

    Hello, world!

Настройка

Выбрать компилятор

Выбрать пункт меню

Settings - Compiler

Откроется окно Compiler setings

Слева выбираем Global compiler settings

В разделе Selected compiler выбираем

GNU GCC Compiler

На вкладке Compiler settings выбираем вкладку Other compiler options и прописываем

-std=c++17
