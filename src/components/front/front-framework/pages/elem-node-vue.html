Установка

  • Подключение через <script>

    https://ru.vuejs.org/v2/guide/installation.html

    Просто скачайте JS-файл и подключите его тегом <script> на странице. Будет зарегистрирована глобальная переменная Vue.

    <script src="js/vue.js"></script>

  • Подключениие через cdn

    https://ru.vuejs.org/v2/guide/installation.html 
    <script src="https://cdn.jsdelivr.net/npm/vue@2.7.0/dist/vue.js"></script>
