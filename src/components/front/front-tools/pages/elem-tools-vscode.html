install

Удаление VS Code (не портативной версии)

  1. Удалить через Панель управления - Программы и компонентыMicrosoft Visual Studio Code (User)
  2. Удалить папки c:\Users\1\.vscode\
    c:\Users\1\AppData\Roaming\Code\

Установка VS Code (портативной версии)

  1. Заходим на сайт

    https://code.visualstudio.com/

  2. Скачиваем 64битную портативную версию

    VSCode-win32-x64-1.82.2.zip

  3. Распаковываем ее в каталог

    c:\Portable\VSCode\

  4. В этом каталоге создаём папку data (здесь будет храниться пользовательский профиль настроек и расширения):

    c:\Portable\VSCode\data

  5. Далее, внутри data создаём 2 обязательные папки

    c:\Portable\VSCode\data\extensions
    c:\Portable\VSCode\data\user-data

    и 1 опциональную папки

    c:\Portable\VSCode\data\tmp extensions — здесь будут храниться все установленные расширения;
    user-data — это, собственно, пользовательский профиль настроек;
    tmp — опциональная папка, если вы хотите, чтобы Visual Studio Code хранил все временные файлы локально, если эту папку не создавать, то все временные файлы будут храниться в %TEMP%.

  6. Вот и всё, можно запускать Code.exe и настраивать, работать. Если вы уже работали на ПК с Visual Studio Code и хотите скопировать или иммигрировать с обычной версии на портативную, то повторяем все шаги выше, и дополнительно для переноса расширений и настроек необходимо сделать следующее:

    data\extensions — копируем сюда всё из %USERPROFILE%\.vscode\extensions
    data\user-data — копируем сюда всё из %APPDATA%\Code

Как добавить VS Code в меню пуск

  1. Выбираем файл (клик правой кнопкой мыши)

    c:\Portable\VSCode\Code.exe
  2. Выбираем Закрепить на начальном экране

Как добавить программу VS Code в контекстное меню

  1. Открыть Редактор реестра, для этого выбрать Пуск - Выполнить или нажатьwin + Rи ввести команду regedit
  2. В редакторе реестра переходим к ветке HKEY_CLASSES_ROOT\*\shell
  3. Клик правой кнопкой мыши на папке shell и выберите команду Создать > Раздел
  4. Назовите его так, как должна называться команда в контекстном меню. Скажем, Открыть с помощью VS Code
  5. Клик правой кнопкой мыши на созданной папке и снова выбираем команду Создать > Раздел
  6. Новый вложенный раздел назовите command
  7. Щелкните на значке раздела command и дважды в окне справа на ключе (По умолчанию) REG_SZ
  8. В открывшемся окне введите следующую команду "c:\Portable\VSCode\Code.exe" "%1"

Как добавить иконку VS Code в контекстное меню

  1. Открыть Редактор реестра, для этого выбрать Пуск - Выполнить или нажатьwin + Rи ввести команду regedit
  2. В редакторе реестра переходим к ветке HKEY_CLASSES_ROOT\*\shell\Открыть с помощью VS Code
  3. Клик правой кнопкой мыши и выбираем команду Создать > Строковый параметр
  4. в поле Имя укажите значение Icon
  5. а в поле Значение укажите путь к exe-файлу, как показано ниже (кавычки обязательны): "c:\Portable\VSCode\Code.exe"

Расширения (extentions)

Visual extentions

  1. indent-rainbow (oderwat) Цветные отступы
  2. Image preview (Kiss Tamas) Просмотр превью картинки в html коде
  3. Theme (Mhammed Talhaouy) A Visual Studio Code theme for those of us who like to code late into the night
  4. Prettier - Code formatter (Prettier) Code formatter using prettier Выравнивание кода
  5. Polacode (P & P) Делает красивые скриншоты кода
  6. Debugger for Chrome (Microsoft) Отладка javascript кода

Настройка Prettier

  1. В VSCode открыть настройки Settings
  2. Выбрать Prettier: Semi для добавления ";" в конце строки
  3. Выбрать Prettier: Single Quote для замены двойных кавычек на одинарные
  4. Выбрать Prettier: Tab Width 2 для замены табуляции на 2 пробела

Functional extentions

  1. Live Server (Ritwick Dey) local Server with live reload feature
  2. Auto Rename Tag (Jun Han) При изменении открывающего тега изменяет закрывающий
  3. CSS Peek (Pranay Prakash) Переход из html в css к выбранному классу
  4. HTML CSS Support (ecmel) Дописывает название класса в html из css
  5. Live Sass Compiler (Ritwick Dey) Вместе сним уже идет Live Server
  6. Beautify (HookyQR)
  7. HTML Snippets (Mohamed Abusaid)

JavaScript extentions

  1. ESLint (Microsoft) Integrates ESLint JavaScript into VS Code.
  2. Babel JavaScript (Michael McDermott) Подсветка нового синтаксиса
  3. Quokka.js (Wallaby.js) Песочница js внутри редактора
  4. JavaScript (ES6) code snippets (charalampos karypidis)

C# extentions

  1. .NET Runtime Install Tool (Microsoft) This extension installs and manages different versions of the .NET Runtime.
