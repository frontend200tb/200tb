Установка Babel

  1. Проверим версии установленных node.js и npm

    node -v
v16.17.0
npm -v
8.15.0

    если их нет, то поставить с оффициального сайта nodejs

  2. Создадим папку для нашего проекта и перейдем в нее

    mkdir project1
cd project1
d://project1/

  3. Инициализируем проект

    npm init -y

    (сокращенно от npm init --yes, то же что и npm init, не отвечая на вопросы)

  4. В итоге в папке с проектом появится файл package.json.

    d://project1/package.json

    После этого можно устанавливать пакеты, которые будут являться зависимостями для нашего приложения.

  5. Установим @babel/cli, @babel/core, @babel/preset-env

    npm i -D @babel/cli @babel/core @babel/preset-env

    (сокращенно от npm install --save-dev @babel/cli @babel/core @babel/preset-env)

  6. Появится папка node_modules, файл package-lock.json, а в файл package.json добавятся зависимости

    "devDependencies": {
  "@babel/cli": "^7.18.10",
  "@babel/core": "^7.18.13",
  "@babel/preset-env": "^7.18.10"
}

  7. Создадим файл .babelrc в котором будут настройки для babel

    touch .babelrc

  8. Запишем в него свойство presets

    { "presets": ["@babel/preset-env"] }

  9. Теперь создадим папку src и в ней файл main.js. Это точка входа для babel

  10. В файле package.json запишем

    "scripts": {
  "build": "babel src -d dist"
},

  11. Теперь можно в исходном файле src/main.js писать код на современном js и транспилировать его в код es2015 в выходной файл dist/main.js командой

    npm run build

  12. В файле package.json запишем

    "scripts": {
  "build": "babel src -d dist",
  "watch": "babel src -d dist -w"
},

  13. Теперь babel сам будет смотреть за исходным файлом и при его сохранении транспилировать его в конечный файл

    npm run watch

  14. Чтобы выйти из режима watch нажмите

    ctrl + C
