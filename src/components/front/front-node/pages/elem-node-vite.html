Vite

  • Vite позволяет вам создавать приложения на Vue с однофайловыми компонентами без шага сборки бандлов. Импорты запрашиваются браузером как нативные модули ES. Dev-сервер перехватывает запросы к файлам .vue и компилирует их на лету. И это происходит мгновенно.

Установка

  • Создадим новое vite приложение с помощью npm командыnpm create vite@latest

    После ввода этой команды запуститься генератор нового приложения которое будет в себе использовать Vite, и впервую очередь что у нас он спросит - это то как мы хотим назвать наш проект

    По умолчанию будет предложено имя vite-project, и после этого нам нужно будет выбрать какой фреймворк мы хотим использовать - и здесь я выберу Vue

    Окей, после выбора что мы хотим использовать Vue - нас спрашивает хотим ли мы использовать TypeScript вместе с Vue

    Я выбираю обычный vue - и как раз уже после этого vite создаст папку с именем нашего проекта, в которой находится наш проект

  • Перейдем в папку проектаcd vite-project
  • Запустим npm installnpm install
  • Запустим локальный сервер и посмотрим как выглядит страница по-умолчанию созданная в новом Vue приложении с Vitenpm run dev
