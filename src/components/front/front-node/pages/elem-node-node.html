Установка

  1. Установим последнюю версию node.js с официального сайта https://nodejs.org/en/ Вместе с node.js поставится и npm
  2. Проверим версии установленных node.js и npm node -v
    v18.18.2
    npm -v
    9.8.1     (сокращенно от node --version и npm --version)
  3. Создадим папку с названием projectmkdir projectЧтобы удалить пустую папку с названием project наберитеrmdir project
  4. Перейдем в папку с названием project cd project
    d://project/     Чтобы выйти из папки наберите cd ..
    d://
  5. Инициализируем проектnpm init -y(сокращенно от npm init --yes, то же что и npm init, не отвечая на вопросы)
  6. В итоге в папке с проектом появится файл package.json. После этого можно устанавливать пакеты, которые будут являться зависимостями для нашего приложения.d://project/package.json
  7. Для примера, установим @babel/cli, @babel/core, @babel/preset-env npm i -D @babel/cli @babel/core @babel/preset-env (сокращенно от npm install --save-dev @babel/cli @babel/core @babel/preset-env)
  8. Появится папка node_modules, файл package-lock.json, а в файл package.json добавятся зависимости "devDependencies": {
      "@babel/cli": "^7.18.10",
      "@babel/core": "^7.18.13",
      "@babel/preset-env": "^7.18.10"
    }
  9. Чтобы удалить пакеты и удалить зависимости из файла package.json npm uninstall -D @babel/cli @babel/core @babel/preset-env(Флаг -D удаляет devDependencies, флаг -S удаляет Dependencies)
