Палиндром

Написать функцию, которая на вход принимает строку и возвращает true если эта строка палиндром, иначе возвращает false. Проверяются только буквы и цифры без учета регистра.

Решаем методом двух указателей. Алгоритм работы такой:

  1. Сначала первый указатель равен индексу первого символа в строке, второй указатель равень индексу последнего символа в строке.
  2. Проверяем что второй указатель больше первого указателя и заходим в цикл, иначе возвращаем true
  3. Проверяем символ по первому указателю, если он не буква или цифра, то увеличиваем первый указатель на 1 и выполняем цикл снова
  4. Проверяем символ по второму указателю, если он н буква или цифра, то уменьшаем второй указатель на 1 и выполняем цикл снова
  5. Если оба указателя буква или цифра, то сравниваем их, если не равны возвращаем false, если равны то изменяем оба указателя и выполняем цикл снова
function isPalindrome(str) {
  let start = 0;
  let end = str.length - 1;
  while (end > start) {
    if (str[start].match(/[a-z0-9]/i)) {
      if (str[end].match(/[a-z0-9]/i)){
        if (str[start].toLowerCase() !== str[end].toLowerCase()){
          return false;
        } else {
          start++;
          end--;
        }
      } else {
        end--;
      }
    } else {
      start++;
    }    
  }
  return true;
}
